# PRD v1.0 — 지역 소상공인 데이터 수집·검증 툴

> 작성: 지훈 (실행팀 PRD) | 상태: GO (조건부) | 버전: 1.0

---

## 제품 개요

지역명을 입력하면 구글맵·카카오맵·당근마켓·인스타그램·네이버블로그/카페에서 소상공인 업체명·010번호·인스타 계정을 자동 수집하고, 네이버플레이스로 실존 검증 후 정제된 엑셀을 출력하는 **리안 컴퍼니 내부 영업용 데스크톱 툴.**

---

## Must Have

1. **지역 + 업종 키워드 입력 → 다중 플랫폼 수집 실행**
   - 입력: 지역명(예: 의정부시 신곡동) + 업종 키워드(예: 카페 / 전체업종)
   - 수집 대상: 구글맵, 카카오맵, 당근마켓, 인스타그램, 네이버블로그/카페
   - 플랫폼별 독립 모듈로 동작 (1개 차단돼도 나머지 정상 작동)

2. **010 번호 필터링 자동 적용**
   - 031, 032 등 유선번호 자동 제거
   - 010 이외 휴대폰 번호(011, 016 등) 제거
   - 번호 없는 레코드는 "번호 미확인"으로 별도 분류 (삭제 아님)

3. **네이버플레이스 실존 검증**
   - 수집된 업체명 기준으로 네이버플레이스 검색 자동 실행
   - 검증 결과: `확인됨` / `미확인` / `폐업의심` 3단계 분류
   - 네이버플레이스 URL 자동 매핑

4. **중복 제거 및 데이터 정제**
   - 동일 업체명 + 동일 번호 중복 레코드 자동 병합
   - 여러 플랫폼에서 수집된 경우 출처 플랫폼 복수 표기

5. **최종 엑셀 내보내기**
   - 컬럼 고정: `업체명 / 010번호 / 네이버플레이스URL / 인스타URL / 수집출처 / 검증상태 / 수집일자`
   - 검증상태별 시트 분리 (확인됨 / 미확인 / 폐업의심)

6. **수집 진행 현황 실시간 표시**
   - 플랫폼별 수집 건수, 현재 진행 플랫폼, 예상 완료 시간 표시
   - 플랫폼별 차단/오류 발생 시 알림 표시 후 나머지 계속 진행

7. **수집 이력 관리**
   - 동일 지역 재수집 시 이전 결과와 비교 → 신규/삭제/변경 건 하이라이트
   - 최근 30회 수집 이력 로컬 저장

---

## Must NOT (범위 외)

- ❌ 외부 사용자 대상 SaaS 기능 (로그인, 결제, 플랜 관리) — 이번 버전은 **내부 툴 전용**
- ❌ 문자 발송 기능 (이 툴은 데이터 수집만, 발송은 별도 채널)
- ❌ "문자 수신 가능 보장" 표기 — 010 필터링만 제공, 수신 가능 여부 미보증
- ❌ 실시간 자동 스케줄링 수집 (수동 실행만, 자동 크론 미지원)
- ❌ 클라우드 서버 저장 (모든 데이터 로컬 저장만)
- ❌ 인스타그램 DM 자동 발송 연동
- ❌ 무제한 수집 — **1회 실행당 최대 500건, 1일 최대 1,000건 상한**
- ❌ 수집 데이터의 외부 판매·공유 기능

## User Flow

```
1단계: 툴 실행
        → 로컬 앱 실행 (로그인 없음)

2단계: 수집 조건 입력
        → 지역명 입력 (예: "의정부시 신곡동")
        → 업종 키워드 입력 (예: "카페" 또는 공란 = 전체업종)
        → 수집 플랫폼 선택 (기본값: 전체 5개 / 개별 해제 가능)
        → [수집 시작] 버튼 클릭

3단계: 수집 진행 (자동)
        → 플랫폼별 순차 수집 실행
        → 화면에 실시간 진행 현황 표시
           (예: "카카오맵 47건 완료 / 구글맵 수집 중... / 인스타그램 대기")
        → 오류 발생 플랫폼은 알림 후 스킵, 나머지 계속

4단계: 자동 정제 (자동)
        → 유선번호(031/032 등) 자동 제거
        → 010 이외 번호 제거
        → 중복 업체 병합
        → 번호 없는 업체 "번호 미확인" 분류

5단계: 네이버플레이스 검증 (자동)
        → 업체명 기준 네이버플레이스 검색 순차 실행
        → 확인됨 / 미확인 / 폐업의심 분류
        → 확인된 업체에 네이버플레이스 URL 매핑

6단계: 결과 확인
        → 결과 미리보기 화면 표시
        → 탭 구분: 전체 / 확인됨 / 미확인 / 폐업의심
        → 개별 레코드 수동 삭제·편집 가능

7단계: 엑셀 내보내기
        → [엑셀 내보내기] 버튼 클릭
        → 저장 경로 선택
        → 시트 분리된 .xlsx 파일 저장 완료
        → 수집 이력에 자동 기록
```
