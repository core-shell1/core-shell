# 박시우의 블로그+인스타 패키지 판매 크리에이티브 전략 보고서
## 타겟: 소상공인 마케팅 대행업 셀러 (월매출 300~1,500만원)

---

## 1. 전환 퍼널별 크리에이티브 설계

### 퍼널 구조 (CAC 15만원 → 8만원 목표)

```
인지(Awareness) → 관심(Interest) → 전환(Conversion) → 재구매(Retention)
   인스타릴스        블로그상세글      상세페이지         고객사례
```

---

## 2. 인스타그램 릴스: 3초 훅 콘텐츠 (인지 단계)

### 📱 릴스 포맷 3종 (A/B/C 테스트용)

#### **A안: 문제 폭로형** (참여율 8.24% 목표)

**대본 (15초)**
```
[0-3초] 화면: "블로그 운영 3개월, 유입 0명?"
       음성: "블로그 매일 쓰는데 왜 안 들어올까요?"
       
[4-8초] 화면: 네이버 검색 순위 100위 밖 스크린샷
       텍스트 오버레이: "❌ C-rank 무시 → 상위 노출 불가능"
       
[9-12초] 화면: 블로그 체류시간 30초 → 5분 전후 비교
        음성: "키워드 20% + 스토리 80% 공식 적용하니"
        
[13-15초] CTA: "지금 프로필 링크 → 무료 진단 신청"
          트렌드 오디오: "Oh My God" (바이럴 사운드)
```

**제작 사양**:
- 비율: 9:16 (1080x1920px)
- 자막: 흰색 고딕체, 48pt, 검은 테두리
- 트랜지션: 하드컷 (3초마다)
- 오리지널 요소: 셀러 본인 얼굴 2초 삽입 (AI 탐지 회피)

**캡션 구조** (AIDA):
```
🔥 블로그 3개월 돌렸는데 유입 0? [Attention]

네이버 C-rank 모르면 100% 망합니다 😱 [Interest]

✅ 키워드 밀도 1~2%
✅ 체류시간 5분 목표
✅ 발행시간 목~금 오전 8시

→ 프로필 링크에서 '블로그 무료 진단' 받기 [Action]

#마케팅대행 #블로그마케팅 #소상공인마케팅 #네이버블로그 #상위노출 #인스타운영대행
(28개 해시태그: 대형 10개 + 중형 10개 + 소형 8개 믹스)
```

---

#### **B안: 데모 증명형** (공유율 2배 목표)

**대본 (20초)**
```
[0-3초] 화면: "패키지 20만원 → 150만원 올린 비결"
       손으로 Before/After 매출 그래프 그리기
       
[4-10초] 화면: 인스타 릴스 편집 화면 (CapCut 실제 작업)
        텍스트: "3초 훅 + 완전 시청 100% 공식"
        
[11-15초] 화면: 클라이언트 DM 스크린샷 "문의 3배 늘었어요!"
         음성: "이 템플릿 그대로 따라하세요"
         
[16-20초] CTA: "댓글에 '템플릿' 입력 → 자동 전송"
          화면: 댓글창 클로즈업
```

**인터랙션 장치**:
- 댓글 자동응답 (manychat 연동): "템플릿" 입력 시 PDF 자동 발송
- 스티커: 설문 "당신의 블로그 체류시간은?" (30초/1분/5분)

---

#### **C안: 트렌드 편승형** (추천 노출 3배)

**대본 (30초)**
```
[0-3초] 트렌드 음악: "APT" 춤 → 화면 전환 "마케팅도 아파트처럼 쌓아야죠"
       
[4-12초] 층별 구조 애니메이션:
        1층 = 블로그 SEO
        2층 = 인스타 릴스
        3층 = 상세페이지 CTA
        옥탑 = 재구매 후기
        
[13-25초] 각 층 클릭 시 수치 팝업:
         "블로그 유입 3배 → 인스타 팔로워 500→2,000 → 전환율 9%↑"
         
[26-30초] CTA: "프로필 링크에서 '무료 아파트 설계도' 받기"
          화면: 건물 완성 + 셀러 얼굴 오버레이
```

**바이럴 요소**:
- 듀엣/스티치 유도 문구: "당신의 마케팅 층수는?"
- 해시태그: #APT챌린지 #마케팅아파트 (트렌드 태그 3개 포함)

---

| 지표 | 목표 | 액션 |
|------|------|------|

## 3. 네이버 블로그: SEO 상위 노출 글 (관심 단계)

### 📝 블로그 콘텐츠 2종 (키워드별 분리)

#### **콘텐츠 1: "소상공인 마케팅 블로그 운영법"** (월 검색량 1,200회)

**제목** (28자):
```
소상공인 마케팅 블로그 운영법 – 유입 0→500 실전 후기
```

**목차 구조** (H2/H3 활용):
```
📌 목차
1. 블로그 시작 3개월, 유입 0명이었던 이유
   1-1. C-rank 모르고 키워드만 채운 실패
2. 네이버 상위 노출 3단계 공식
   2-1. 제목: 지역+업종+후기 구조
   2-2. 소제목: 매 문단 H2 삽입
   2-3. 키워드 밀도 1~2% 유지법
3. 실전 적용 후 결과
   3-1. 발행 타이밍: 목요일 오전 9시 → 노출 2배
   3-2. 이미지 ALT: "소상공인 블로그 인테리어" 필수
4. 무료 템플릿 다운로드 (CTA)
```

**본문 샘플** (2,000자 중 일부):
```

문제는 '키워드만 채우면 된다'는 착각이었죠.

네이버 C-rank는 키워드 20% + 스토리 흐름 80%로 작동합니다.
탬블 클라이언트 사례를 보면, 키워드를 억지로 반복한 글은

반대로 '개인 경험 + 구체적 수치'를 넣은 글은
