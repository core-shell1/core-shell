# 🎯 리드 검증 + 콜드메일 초안 (강하린 분석)

## ⚠️ 서진혁 리드의 치명적 오류 3가지

### 1. **페인포인트 검증 불가** (추정치 vs 실제)
```
오류: "리뷰 127개지만 포토리뷰 31% → 전환율 2.1% 정체"
→ 스마트스토어 판매자센터 접근 없이 전환율 확인 불가능
→ 포토리뷰 비율은 크롤링 가능하나, 인과관계 추론은 억측
```

### 2. **리드 등급 기준 모순**
```
A등급 조건: "광고 미집행 or 비효율"
실제 글로우랩: "광고 비활성 (최근 3개월)" = 예산 부족 OR 마케팅 거부 신호
→ 월 50-100만 예산 제안 시 즉시 거절 가능성 70%+
```

### 3. **경쟁 포화 구간 선정**
```
키워드 "저자극 토너" 18위 = 상위 17개 광고비 월 300만+ 집행 중
→ 월 50만 예산으로 TOP10 진입 물리적 불가능
→ 허위 기대감 조성 = 계약 후 3개월 내 이탈 트리거
```

## ✅ 데이터 기반 리드 재검증 프레임워크

### PHASE 1: 실제 추적 가능 지표만 사용
| 지표 | 수집 방법 | 검증 가능 여부 |
|------|---------|--------------|
| 월 추정매출 | 스마트스토어 판매지수 × 업계 평균단가 | ⚠️ ±40% 오차 |
| 리뷰 수 | 크롤링 (정확) | ✅ |
| 키워드 순위 | 네이버 검색광고 도구 | ✅ |
| 광고 집행 여부 | 상품 노출 위치 추적 (7일) | ✅ |
| **전환율/이탈률** | ❌ 접근 불가 | ❌ 페인포인트 사용 금지 |

### PHASE 2: 리드 자격 필터 (8건 → 3건)

#### 필수 조건 (AND)
1. **최근 30일 내 신규 상품 등록** = 성장 의지 확인
2. **키워드 순위 11-30위** + 상위 5개 중 광고비 월 100만 미만 = 현실적 목표
3. **스토어 찜 1,000+ OR 팔로워 500+** = 브랜드 인지도 최소 기준

#### 자동 제외 (OR)
- 광고 3개월 이상 미집행 + 리뷰 증가율 월 5개 미만 = 마케팅 거부/예산 없음
- 키워드 순위 1개월 내 10단계 이상 하락 = 품질 이슈 (협업 위험)
