# 이도현의 프로젝트 실행 전략서
## 과제: 소상공인 마케팅 대행 블로그+인스타 패키지 판매 전략

---

## 📋 EXECUTIVE SUMMARY

**목표**: 소상공인 대상 블로그+인스타 통합 패키지 월 15건 판매 (목표 매출 900만원)  
**핵심 전략**: 3단계 납품 프로세스 + 실행사 2곳 교차 운영 + 마진율 35% 확보  
**실행 기간**: 3개월 (1개월 테스트 → 2개월 본격 확대)

---

## 1. 패키지 상품 설계 (원가·마진 기반)

### 📦 상품 라인업 (3단계 구조)

| 등급 | 패키지명 | 구성 | 판매가 | 원가 | **마진** | 타겟 |
|------|---------|------|--------|------|---------|------|
| **BASIC** | 블로그 시작 | 블로그 5건/월 + 인스타 15개/월 | **35만원** | 22만원 | **37%** | 창업 3개월 이내 |
| **STANDARD** | 통합 관리 | 블로그 10건/월 + 인스타 30개/월 + 월간 보고서 | **60만원** | 39만원 | **35%** | 매출 500만원 미만 |
| **PREMIUM** | 전략 패키지 | 블로그 15건/월 + 인스타 50개/월 + 리워드 트래픽 2,000건 + 주간 리뷰 | **95만원** | 61만원 | **36%** | 매출 1,000만원 이상 |

**원가 산출 근거** (실행사 견적 기반):

```
[BASIC 원가 분해]
- 블로그 체험단 5건: 12.5만원 (미블 단가 2.5만원/건 × 5)
- 인스타 포스팅 15개: 9만원 (서울오빠 단가 6천원/건 × 15)
- 관리비(QA/스케줄링): 0.5만원
= 총 22만원

[STANDARD 원가 분해]
- 블로그 10건: 23만원 (레뷰 장기계약 2.3만원/건)
- 인스타 30개: 16.5만원
- 월간 보고서(템플릿 자동화): 무료
- 관리비: 1만원
= 총 39만원 (마진 35% = 60만 - 39만)

[PREMIUM 원가 분해]
- 블로그 15건: 33만원 (레뷰 패키지 2.2만원/건)
- 인스타 50개: 25만원
- 리워드 트래픽 2,000건: 2만원 (1,000건당 1만원 시세)
- 주간 리뷰 회의(30분 × 4): 1만원
= 총 61만원
```

**최소 마진율 30% 경고선**: BASIC 37%, STANDARD 35%, PREMIUM 36% → **전 상품 안전**

## 2. 실행사 운영 전략 (2곳 교차 + 백업)

### 🏢 주 실행사 매트릭스

| 실행사 | 담당 영역 | 계약 형태 | 월 최소 물량 | 단가 조건 | 리드타임 | 품질 점수 |
|--------|----------|----------|-------------|----------|----------|----------|
| **레뷰** (1순위) | 블로그 체험단 (10건 이상 주문 시) | 월 50건 이상 장기계약 | 50건 | 2.2-2.3만원 | 10일 | 9.5/10 |
| **미블** (2순위) | 블로그 소량 (10건 미만 주문 시) | 건별 발주 | 10건 | 2.5만원 | 7일 | 8.5/10 |
| **서울오빠** | 인스타그램 전담 | 월 100개 이상 계약 | 100개 | 5-6천원/개 | 5일 | 8/10 |
| **리뷰플레이스** (백업) | 긴급 물량 처리 | 건별 | 5건 | 3만원 | 14일 | 7.5/10 |

**운영 원칙**:
1. **레뷰 메인**: 월 50건 이상 확보 시 2.2만원 단가 고정 → STANDARD/PREMIUM 고객에 집중 투입
2. **미블 서브**: BASIC 단건 또는 급한 납품 10건 미만 물량
3. **인스타는 서울오빠 독점**: 월 100개 계약으로 단가 5천원 확보 (10-15개 분산 시 6천원)
4. **백업 리뷰플레이스**: 레뷰/미블 동시 마감 시 긴급 투입 (고단가 3만원 감수)

**실행사 발주서 템플릿**:
```
[발주서 #202501-레뷰-001]
발주일: 2025-01-15
실행사: 레뷰
---
서비스: 블로그 체험단
수량: 50건
단가: 2.2만원
총 원가: 110만원
납기: 2025-01-25 (10일)
클라이언트 배분: A사(15건), B사(10건), C사(10건), D사(15건)
QA 체크리스트: ✅ 맞춤법, ✅ 키워드 3개 이상, ✅ 이미지 3장 이상, ✅ 링크 정상
---
담당: 이도현
승인: 리안 확인 필요 (마진율 35% 확인)
```

## 3. 프로젝트 관리 시스템 (노션 기반)

### 📊 클라이언트 대시보드 구조

**노션 DB 필드** (Clients 데이터베이스):

| 필드명 | 타입 | 용도 | 예시 |
|--------|------|------|------|
| 클라이언트명 | Text | 식별자 | "강남피부과A" |
| 패키지 | Select | 상품 분류 | STANDARD |
| 계약일 | Date | 시작일 | 2025-01-10 |
| 상태 | Status | 진행 단계 | 🟢진행중 / 🟡대기 / 🔴피드백 / ✅납품완료 |
| 이번달 블로그 | Number | 누적 건수 | 7/10 |
| 이번달 인스타 | Number | 누적 개수 | 22/30 |
| 다음 납품일 | Date | 마일스톤 | 2025-01-25 |
| 담당 실행사 | Relation | 연동 | @레뷰, @서울오빠 |
| 월 매출 | Formula | 패키지 × 1 | 60만원 |
| 월 원가 | Formula | 블로그단가×건수 + 인스타단가×개수 | 39만원 |
| **마진율** | **Formula** | **(매출-원가)/매출×100** | **35%** ⚠️30% 미만 시 빨간색 |

**자동화 규칙** (노션 Automations):
- 마진율 < 30% → 리안에게 슬랙 알림 "⚠️ {클라이언트명} 마진 위험 {마진율}%"
- 납품일 D-3 → 담당자에게 "📦 {클라이언트명} QA 체크 시작"
- 상태 = 납품완료 → 자동으로 월간 보고서 템플릿 생성

## 4. 납품 프로세스 (3단계 QA)

### ✅ QA 체크리스트 (엄격 버전)

**STAGE 1: 실행사 1차 납품 (D-7)**
```
[ ] 블로그 포스팅 맞춤법 검사 (부산대 맞춤법 검사기)
[ ] 키워드 밀도 2-3% (클라이언트 제공 키워드 3개 이상 포함)
[ ] 이미지 해상도 최소 1200px
[ ] 링크 클릭 테스트 (네이버 플레이스/스마트스토어)
[ ] 체류시간 1분 이상 유도 구조 (목차, 질문 유도)
```

**STAGE 2: 이도현 중간 검수 (D-3)**
```
[ ] 클라이언트 브랜드 톤앤매너 일치 (사전 가이드 대조)
[ ] 중복 표현 제거 (Copyscape 0% 목표)
[ ] 인스타그램 해시태그 30개 최적화 (#지역 + #업종 + #롱테일)
[ ] 스토리 하이라이트 연동 (PREMIUM 패키지만)
[ ] 리워드 트래픽 IP 분산 확인 (네이버 제재 방지)
```

**STAGE 3: 최종 납품 패키지 (D-day)**
```
📦 납품 폴더 구조:
/클라이언트명_2025년1월
  /블로그_포스팅
    - 포스팅1_URL.txt (링크 + 스크린샷)
    - 포스팅2_URL.txt
    ...
  /인스타그램
    - 게시물_링크_리스트.xlsx
    - 인사이트_캡처.png (도달, 참여 수치)
  /리워드트래
