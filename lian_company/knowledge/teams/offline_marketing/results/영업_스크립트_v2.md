> **예진 노트:** 이 PDF는 "문제 제시 기계"야. 해결법은 단 한 줄도 없어. 사장님이 읽고 나서 불안해지는 게 목표. 불안한 사람만 전화해.

## DM/문자 스크립트

### 1차 메시지 (3가지 버전)

**버전 A — 숫자 충격형 (Cold 타겟 / 처음 연락)**
```
안녕하세요 사장님, 갑자기 연락드려 죄송해요.
저 리안이라고 하는데요, 오늘 [업체명] 네이버 플레이스 
잠깐 들여다봤거든요.

'[지역] [업종]' 검색하면 지금 [N]위에 계세요.
이 키워드 한 달에 [X]번 검색되는데,
1위 업체가 그 손님을 다 받고 있어요.

진단 자료 간단히 정리해뒀는데, 보내드려도 될까요?
부담 없이 보시기만 하면 돼요.
```
→ 왜 이 문장: 숫자 2개([N]위, [X]번)로 즉시 현실을 충격화, "부담 없이"로 저항감 최소화

**버전 B — 경쟁자 자극형 (Cold 타겟 / 경쟁 업체 언급)**
```
사장님 안녕하세요, 리안이에요.
다름이 아니라 [업체명] 근처에
[경쟁업체명]이라고 아세요?

거기 지금 네이버에서 '[지역] [업종]' 1위예요.
이번 달 조회수만 [X]회거든요.
사장님 업체는 지금 그 검색에 안 보이세요.

어디서 차이 나는지 정리한 자료 있는데
한번 보시겠어요?
```
→ 왜 이 문장: 실명 경쟁자 언급으로 "내가 지고 있다"는 즉각적 자존심 자극 유발

**버전 C — 공감형 (Warm 타겟 / 이전에 한번 본 적 있는 업체)**
```
사장님, 리안이에요. 저번에 잠깐 말씀 나눴죠.

솔직히 말씀드리면,
사장님 가게 음식(서비스)은 진짜 좋은데
네이버에서는 지금 안 보이는 상황이에요.

검색하는 손님들한테
존재 자체를 못 알리고 계신 거거든요.

현황 딱 4페이지로 정리했는데,
한번만 봐주시겠어요?
```
→ 왜 이 문장: 서비스 칭찬 후 "보이지 않는 현실" 대비로 아쉬움과 긴박감 동시 유발

### 2차 메시지 (신뢰 형성 + 가치 증명)
```
사장님, 자료 보내드렸어요.

1페이지부터 순서대로 보시면
지금 사장님 업체 상황이 숫자로 딱 나와 있어요.

특히 3페이지에 '[지역] [업종]' 검색량 나오는데,
그 손님들이 지금 어디로 가고 있는지 보시면
아마 좀 놀라실 거예요.

궁금하신 거 생기면 바로 연락 주세요.
```

### 3차 메시지 (제안 + 클로징)
```
사장님, 자료 보셨어요?

사장님이랑 비슷한 [업종] 하시는 분이
같은 [지역]에서 지금 검색 상위를 다 차지하고 계세요.

이번 달 안에 저랑 딱 10분만 
직접 화면 보면서 얘기 나눠보시면
지금 상황이 정확히 뭔지 아실 수 있거든요.

이번 주 혹시 잠깐 시간 되세요?
```

### 4차 메시지 (마감 압박)
```
사장님, 마지막으로 한 번만 드리는 말씀이에요.

이번 달 [지역] [업종] 쪽에서
한 군데만 더 계약 진행할 수 있거든요.

지금 비슷한 업체 두 곳이랑 얘기 중인데,
사장님 먼저 연락드리는 거예요.

이번 주 안에 말씀 주시면
사장님 자리 잡아드릴게요.
연락 주세요.
```

### 재발송 메시지 (3일 후)
```
사장님, 며칠 전 연락드렸던 리안이에요.
바쁘실 것 같아서 기다렸다가 다시 드려요.

자료 아직 못 보셨더라도 괜찮아요.
부담 없이 한 번만 봐주시면 충분해요.

'[지역] [업종]' 검색량이 한 달에 [X]번인데,
그 손님들 지금 어디로 가는지만 확인해보세요.
다시 보내드릴게요.
```

**[섹션 B — 진단 체크리스트 (핵심 파트)]**

> **디자인 지시:** 항목별로 ✅(양호) / ❌(문제) 아이콘 크게 표시. ❌ 항목은 빨간색 배경 또는 빨간 텍스트로 시선 집중. ✅는 회색으로 처리해서 ❌가 더 눈에 띄게.

```
진단 항목                              현재 상태

대표 키워드 1페이지 노출               ❌  현재 [N]위 (1페이지 밖)
등록 사진 수 (권장 30장 이상)          ❌  현재 [X]장
최근 리뷰 (최근 30일 기준)             ❌  [0건 / 1건]
리뷰 답글 작성 여부                    ❌  미작성
저장 수 (관심 고객 수)                 ❌  [X]개 (지역 평균 [Y]개)
영업시간 / 메뉴 정보 최신화            ❌  [미등록 / 오래된 정보]
스마트플레이스 소식 게시               ❌  마지막 게시 [N]일 전 / 없음
```
