# 리안 컴퍼니 — 네이버 플레이스 영업 2차 PDF 전략 완성본

---

## 핵심 인사이트 (자료 + 리서치 분석)

**인사이트 1 — 손실이 이득보다 2배 강하게 움직인다** (출처: 손실회피 편향 자료)
"매출이 늘어날 수 있다"가 아니라 "지금 이 순간에도 경쟁자가 당신 손님을 가져가고 있다"가 결정을 만든다. 플레이스 방문자 1명 = 9,256원 매출이라는 실증 데이터가 있다. 이걸 "지금 몇 명 잃고 있나"로 뒤집어야 한다.

**인사이트 2 — 디지털 이해도가 낮을수록 숫자로 보여줘야 믿는다** (출처: 경기 북부 인식 수준 자료)
경기 북부 소상공인 디지털 마케팅 자가평가 41.1점. 말로 설명하면 흘려듣는다. 숫자, 순위, 검색량 수치를 시각화해서 "이게 사실"임을 눈으로 확인시키는 순간 신뢰가 생긴다.

**인사이트 3 — 앵커링은 반드시 최고가부터** (출처: 앵커링 효과 가격 제시 순서 전략 자료)
95만 → 56만 → 38만 순서로 제시할 때 56만이 "합리적 선택"처럼 보인다. 38만부터 보여주면 95만이 바가지처럼 느껴진다. 순서 자체가 전환율을 결정한다.

**인사이트 4 — 해결법을 주지 않는 것이 전략이다** (출처: Challenger Sale + Hook-Story-Offer 구조)
Challenger Sale의 핵심은 "당신이 모르는 문제를 내가 알고 있다"는 긴장감이다. 해결법까지 주면 긴장이 풀린다. 문제만 선명하게 보여주고 멈추면, 타겟이 스스로 해결책을 요청하게 된다. PDF는 문제 제시 기계여야 한다.

**인사이트 5 — 혼자 결정하는 사람은 리스크를 가장 두려워한다** (출처: 거절 이유 Top 5 + 소상공인 심리 자료)
배우자나 파트너와 상의 없이 혼자 결정하는 구조이기 때문에 "내가 잘못 선택하면 어떡하지"라는 두려움이 가장 크다. PDF는 "당신이 선택하지 않을 때의 리스크"를 보여줌으로써 이 두려움의 방향을 바꿔야 한다.

## 영업 전략 설계

### 타겟 선별 기준

**1순위 타겟 조건 — 이 조건 3개 이상 충족하는 업체만 접촉**

```
① 네이버 플레이스 등록은 되어 있으나 사진 10장 미만
② 메인 키워드 검색 순위 15위 밖 (1페이지 노출 안 됨)
③ 리뷰 50개 미만 또는 리뷰 답글 0
④ 업종: 미용실 / 식당 / 카페 (경기 북부 — 양주·포천·의정부·동두천·가평)
⑤ 운영 3년 이상 (망할 가게 아님, 결제 여력 있음)
```

**즉시 제외 기준:**
- 이미 마케팅 대행 계약 중인 업체
- 폐업 예정 징후 (영업시간 불규칙, 리뷰 최근 6개월 0건)
- 프랜차이즈 (본사 정책으로 자체 계약 불가)

---

### 접촉 전략

#### STEP 0 — 접촉 전 (사전 분석, 10분/업체)
네이버에서 해당 업체 키워드 순위, 사진 수, 리뷰 수, 저장수 직접 확인.
숫자 3개를 메모해두고 DM에 그대로 박아 넣는다.

```
확인할 것:
- '[지역] [업종]' 검색 시 현재 순위
- 등록 사진 수
- 최근 리뷰 날짜 + 총 리뷰 수
```

#### STEP 1 — 첫 접촉 DM (목표: 자료 수락)
**발송 시간: 오전 10시~11시 사이**
**하루 최대 25건**

```
[버전 A — 숫자 충격형]

안녕하세요 사장님.
[업체명] 네이버 플레이스 분석해봤는데요.
현재 '[지역 키워드]' 검색에서 [N]위에 계세요.
1위 업체 이번 달 조회수 [X]회인데
사장님 업체는 지금 그 노출을 못 받고 계신 거예요.
자료 간단히 정리해뒀는데 보내드려도 될까요?
```

**답장 없을 경우:** 3일 후 1회만 재발송. 이후 미련 버리기.

```
[재발송 — 3일 후]

사장님, 며칠 전 연락드렸던 리안이에요.
자료 아직 못 보셨을 것 같아서 다시 드려요.
부담 없이 한 번만 보시면 돼요.
```

#### STEP 2 — PDF 발송 (목표: 미팅 or 전화 수락)
DM 수락 후 즉시 PDF 발송.
발송 직후 2분 안에 아래 문자 추가 발송:

```
"사장님, 자료 보내드렸어요.
1페이지부터 순서대로 보시면
지금 업체 상황이 한눈에 나와 있어요.
보시고 궁금한 거 있으시면 편하게 연락 주세요."
```

#### STEP 3 — 48시간 후 팔로업 (목표: 미팅 확정)
PDF 발송 48시간 후, 열람 여부 무관하게 팔로업:

```
"사장님, 자료 보셨어요?
사장님이랑 비슷한 [업종] 하시는 분이
같은 지역에서 지금 검색 상위를 다 차지하고 계시더라고요.
직접 뵙고 5분만 설명드리면 딱 이해가 되실 거 같아서요.
이번 주 중에 10분만 시간 내주실 수 있으세요?"
```

#### STEP 4 — 미팅 (목표: 계약)
PDF를 화면으로 함께 보면서 설명.
설명 순서: 표지 → 현황 진단 → 키워드 분석 → 패키지.
미팅에서 절대 하지 말 것: 기술적 설명, 과정 설명, "저희가 이렇게 해드려요" 식 해결책 제시.
미팅의 목적은 "지금 얼마나 손해 보고 계신지"를 사장님 입으로 말하게 만드는 것.

**핵심 질문 2개:**
```
"사장님, 지금 한 달에 신규 손님 몇 명 정도 오세요?"
"근처 [경쟁 업체명] 아세요? 거기 지금 네이버에서 1위예요."
```

---

### 핵심 전환 포인트

**"YES"를 만드는 결정적 순간 — 3페이지 키워드 분석에서 일어난다**

타겟이 자기 키워드의 월간 검색량 수치를 처음 보는 순간이 전환점이다.
"'[지역] 미용실'이 한 달에 3,200번 검색되는데, 지금 사장님은 그 노출을 못 받고 계세요"라는 말에
사장님이 "그게 다 어디로 가요?"라고 물으면 계약의 80%는 끝났다.

이 순간에 해야 할 말:
```
"다 위에 있는 업체들한테 가요.
1위, 2위, 3위가 나눠서 가져가는 구조예요.
그 자리에 없으면 그냥 없는 가게예요, 검색 상에서는."
```

그리고 침묵.
설명 추가 금지. 사장님이 먼저 말할 때까지 기다린다.

### 거절 유형별 처리법

**거절 유형 1 — "비싸요"**
```
사장님: "좀 비싸네요..."

대응:
"네, 맞아요. 싸진 않아요.
근데 사장님, 지금 '[메인키워드]' 검색에서
1위 업체가 한 달에 [N]번 노출되고 있어요.
그게 다 거기로 가고 있는 거거든요.
지금 안 하시는 게 더 비싼 거 아닐까요?"

→ 침묵 후 시선38만 패키지 제시: "일단 이 옵션부터 시작하시는 분들 많으세요."
```

**거절 유형 2 — "효과가
