# 신규 기능 — 2026-04-05

## 1. 에이전트 실시간 상태 추적 시스템

### 파일
- `utils/status_tracker.py` — 상태 추적 핵심 로직

### 기능

#### 상태 파일 구조
```
.agent_status.json
├── agents:
│   └── 에이전트명:
│       ├── team: "팀명"
│       ├── status: "running" | "completed" | "error"
│       ├── detail: "현재 작업 설명"
│       ├── started_at: "ISO 타임스탬프"
│       └── updated_at: "ISO 타임스탬프"
└── last_updated: "ISO 타임스탬프"
```

#### API 함수
```python
from utils.status_tracker import update_status, clear_status, get_status, print_status, format_status_line

# 에이전트 시작
update_status("박탐정", "온라인영업팀", "running", "경쟁사 분석 중")

# 에이전트 완료
clear_status("박탐정")

# 상태 조회
status = get_status()

# 터미널 출력
print_status()

# 한 줄 포맷
log_line = format_status_line("박탐정")
```

#### 통합된 파이프라인
모든 팀의 `pipeline.py`에 자동으로 통합됨:
- `teams/온라인영업팀/pipeline.py` ✅
- `teams/온라인납품팀/pipeline.py` ✅
- `teams/온라인마케팅팀/pipeline.py` ✅
- `teams/offline_marketing/pipeline.py` ✅
- `teams/education/pipeline.py` ✅
- `teams/analysis/pipeline.py` ✅
- `core/pipeline.py` (이사팀) ✅

#### 사용 예시
```python
# 에이전트 실행 전에
if HAS_STATUS_TRACKER:
    update_status("박탐정", "온라인영업팀", "running", "경쟁사 분석 중")

try:
    result = 박탐정.run(context, client)
finally:
    if HAS_STATUS_TRACKER:
        clear_status("박탐정")
```

## 2. 팀 템플릿 관리 시스템

### 파일
- `utils/team_templates.py` — 템플릿 수입/수출 로직
- `templates/` — 템플릿 저장소
- `export_team_templates.py` — 기존 팀 템플릿 내보내기 스크립트

### 기능

#### 템플릿 내보내기
```bash
# 수동 내보내기
python export_team_templates.py

# 특정 팀 내보내기
from utils.team_templates import export_team
export_team("온라인영업팀")
```

#### 템플릿에서 새 팀 생성 (3초)
```bash
# 템플릿 목록 확인
python build_team.py --templates

# 템플릿 사용 (대화형)
python build_team.py "새로운영업팀" "팀 설명"
# → 비슷한 템플릿 제안 ("온라인영업팀"이 있을까요? [y/n])
# → y 선택 시 3초 내 팀 생성

# 자동파일럿 모드 (제안 자동 수락)
python build_team.py "새로운영업팀" "팀 설명"  (stdin이 tty가 아니면)
```

#### API 함수
```python
from utils.team_templates import (
    export_team,          # 기존 팀 → 템플릿
    import_team,          # 템플릿 → 새 팀
    list_templates,       # 템플릿 목록
    get_similar_template, # 키워드로 유사 템플릿 찾기
    print_templates,      # 템플릿 목록 출력
)

# 팀 내보내기
export_team("온라인영업팀")

# 템플릿에서 새 팀 생성
import_team("온라인영업팀", "새로운팀이름")

# 템플릿 목록 조회
templates = list_templates()

# 키워드로 찾기 (예: "영업" → "온라인영업팀")
similar = get_similar_template("영업")
```

#### 템플릿 파일 구조 (.json)
```json
{
  "team_name": "온라인영업팀",
  "created_at": "2026-04-05T03:09:40.603952",
  "agents": [
    {
      "name": "박탐정",
      "info": {
        "role": "잠재고객 분석",
        "model": "claude-opus-4-1",
        "content_preview": "[에이전트 코드 미리보기]"
      }
    }
  ],
  "pipeline_structure": {
    "agents": ["박탐정", "이진단", ...],
    "call_order": [...],
    "has_interview": true,
    "has_self_critique": true
  },
  "pipeline_code": "[파이프라인 전체 코드]"
}
```

## 3. 상태 추적 + 템플릿 상호작용

### 시나리오 1: 기존 팀 실행할 때
```bash
# 온라인영업팀 실행
python run_온라인영업팀.py "업무"

# 실행 중:
# 🟠 박탐정 (온라인영업팀) | 경쟁사 광고 분석 중 | 32초 경과
# 🟠 이진단 (온라인영업팀) | 온라인 현황 진단서 작성 중 | 18초 경과
# ...

# 완료 후:
# 🟢 박탐정 (온라인영업팀) | 경쟁사 광고 분석 중 | 1분 15초
# ✅ 완료
```

### 시나리오 2: 새로운 영업팀 빠르게 만들 때
```bash
# 1. 기존 온라인영업팀을 템플릿으로 저장 (최초 1회)
python export_team_templates.py

# 2. 새 팀 생성 (템플릿에서 3초)
python build_team.py "스타트업영업팀" "스타트업 고객 영업"
# → "온라인영업팀" 템플릿을 빠르게 사용할까요? [y/n]
# → y
# → ✅ 스타트업영업팀 생성 완료 (템플릿 기반)

# 3. 즉시 실행
python run_스타트업영업팀.py "첫 영업"
# → 상태 실시간 추적
```

### 시나리오 3: 자동파일럿 모드 (스크립트에서 호출)
```bash
# subprocess/백그라운드에서 호출
python build_team.py "자동생성팀" "목적"
# → stdin이 tty가 아니면 템플릿 자동 사용
# → 템플릿 없으면 교육팀 자동 실행
```

## 4. 활용 팁

### Tip 1: 상태 추적 로그
```python
# 에이전트가 뭘 하고 있는지 알고 싶을 때
from utils.status_tracker import print_status
print_status()

# 출력:
# ========================== 70글자 ==========================
# 📊 에이전트 상태
# ========================== 70글자 ==========================
# 🟠 박탐정 (온라인영업팀)
#    상태: 실행 중 | 경과: 32초
#    작업: 경쟁사 광고 분석 중
# ...
```

### Tip 2: 템플릿 재사용 (한 줄)
```bash
# 새 팀을 30초 안에 만들고 실행
python build_team.py "카페영업팀" "카페 고객 영업" && python run_카페영업팀.py "첫 업무"
```

### Tip 3: 직접 상태 조회 (코드)
```python
from utils.status_tracker import get_status, format_status_line

# 전체 상태
status = get_status()
for agent_name, info in status["agents"].items():
    print(format_status_line(agent_name))

# 특정 에이전트만
if "박탐정" in status["agents"]:
    print(f"박탐정 상태: {status['agents']['박탐정']}")
```
