## 빠른 시작

### 1단계: 디스코드 개발자 포털에서 봇 토큰 얻기

1. https://discord.com/developers/applications 로 이동
2. "New Application" → 앱 이름 입력
3. "Bot" → "Add Bot"
4. "TOKEN" 복사
5. `.env` 파일에 추가:
   ```
   DISCORD_BOT_TOKEN=여기에_복사한_토큰_붙여넣기
   ```

### 2단계: 봇 권한 설정

**개발자 포털에서:**
1. "Intents" 섹션:
   - ✅ **MESSAGE CONTENT INTENT** (필수!)
   - ✅ PRESENCE INTENT
   - ✅ SERVER MEMBERS INTENT

2. "OAuth2" → "URL Generator":
   - Scopes: `bot` 선택
   - Permissions: 
     - Send Messages
     - Read Messages/View Channels
     - Read Message History
     - Embed Links
   - 생성된 URL을 브라우저에서 열어 봇을 서버에 추가

### 3단계: 리안 유저 ID 등록 (보안)

1. 디스코드 설정 → "고급" → "개발자 모드" 활성화
2. 자신의 프로필 우클릭 → "사용자 ID 복사"
3. `.env`에 추가:
   ```
   DISCORD_LIAN_USER_ID=복사한_ID
   ```

### 4단계: 봇 실행

```bash
cd "C:/Users/lian1/Documents/Work/core/lian_company"
./venv/Scripts/python.exe discord_bot.py
```

성공하면:
```
✅ 봇 로그인: YourBotName#0000
   리안 ID: 123456789012345678
   채널 ID: (미설정 — 모든 채널)
```

## 설정

### 환경변수 (.env)

| 변수 | 필수 | 설명 |
|------|------|------|
| `DISCORD_BOT_TOKEN` | ⭐ | 디스코드 봇 토큰 (필수) |
| `DISCORD_LIAN_USER_ID` | ⭐ | 리안 유저 ID (보안 권장) |
| `DISCORD_CHANNEL_ID` | - | 특정 채널에서만 명령 허용 (선택) |

**예시:**
```
DISCORD_BOT_TOKEN=MjkzNzAyNzc4MTUyNTg2Nzg0.XvKc2w.D3p7xL...
DISCORD_LIAN_USER_ID=123456789012345678
DISCORD_CHANNEL_ID=987654321098765432
```

### 자동 실행 설정 (Windows)

PC 시작 시 봇을 자동으로 실행하려면:

1. Windows 검색 → "작업 스케줄러" 실행
2. "기본 작업 만들기..."
3. 설정:
   - 이름: `Lian Company Discord Bot`
   - 트리거: "시스템 시작 시"
   - 동작 → 새 작업:
     - 프로그램: `C:/Users/lian1/Documents/Work/core/lian_company/venv/Scripts/python.exe`
     - 인수: `discord_bot.py`
     - 시작 위치: `C:/Users/lian1/Documents/Work/core/lian_company`
4. "마침"

## 문제 해결

### 봇이 메시지에 응답하지 않음

**체크리스트:**
- [ ] `DISCORD_BOT_TOKEN`이 `.env`에 올바르게 설정됐나?
- [ ] 개발자 포털에서 **MESSAGE CONTENT INTENT**가 활성화됐나?
- [ ] 봇이 채널에 메시지 보낼 권한이 있나?
- [ ] 봇이 다운됐나? (콘솔 창 확인)

**해결:**
1. 토큰 재확인
2. Intents 다시 확인
3. 봇의 역할(Role) 권한 확인
4. 봇 재시작: `Ctrl+C` → `python discord_bot.py`

---

### "Invalid token" 에러

```
discord.errors.LoginFailure: Improper token has been passed.
```

**원인:**
- 토큰이 잘못됨
- 토큰에 공백이 있음
- 토큰이 만료됨

**해결:**
1. 개발자 포털에서 새 토큰 생성
2. `.env`에서 공백 제거
3. 새 토큰으로 다시 시도

---

### "Missing Permissions" 에러

```
discord.errors.Forbidden: 403 Forbidden
```

**원인:**
- 봇이 채널에 메시지 보낼 권한 없음
- 봇의 역할 위치가 너무 낮음

**해결:**
1. 서버 설정 → 역할 → 봇 역할 확인
2. 봇의 역할을 관리자 역할 위에 배치
3. 채널 권한에서 봇에게 "Send Messages" 허용

---

**확인:**
```bash

```

패턴 매칭이 제대로 작동하는지 확인합니다.

**다시 시도할 메시지:**
```
이사팀 (공백) 아이디어
영업 (공백) 태스크
```

공백이 빠진 경우 인식 안 됩니다.

---

### 봇이 자주 끊김

**원인:**
- 인터넷 연결 불안정
- 디스코드 서버 점검
- 봇 프로세스 충돌

**해결:**
1. 봇 재시작
2. 네트워크 연결 확인
3. 자동 실행 설정으로 자동 복구

## FAQ

**Q. 여러 채널에서 명령을 받을 수 있나?**

A. 기본값: 모든 채널. `DISCORD_CHANNEL_ID`를 설정하면 특정 채널만.

**Q. 다른 사람도 명령을 실행할 수 있나?**

A. 기본값: 모두 가능. `DISCORD_LIAN_USER_ID`를 설정하면 리안만.

**Q. 봇이 실행하는 명령의 로그는 어디?**

A. `보고사항들.md` 파일에 자동 저장됨. 디스코드에서 `보고` 명령으로도 확인 가능.

**Q. 실행 중인 작업을 취소할 수 있나?**

A. 불가능. 완료될 때까지 기다려야 합니다.

**Q. 명령 실행 시간이 얼마나 걸나?**

A. 명령별로 다름:
- 이사팀: 3~5분
- 영업/납품/마케팅팀: 1~3분
- 데일리: 5~10분
