# Wave 2 CPO↔CTO 합의 — 지역 소상공인 010번호 + 인스타 수집 툴

## 확정 결정

| 항목 | 결정 | 비고 |
|------|------|------|
| 구글맵 처리 | **선택 옵션 (기본값: OFF)** | 봇 차단 리스크 높음. 체크박스로 활성화 가능 |
| 카카오맵 | **기본 ON, 핵심 소스** | 가장 안정적인 새 소스 |
| 네이버플레이스 검증 | **선택 옵션 (기본값: ON)** | 100건 기준 15~20분 소요. 체크박스로 건너뛸 수 있게 |
| 인스타 수집 정확도 | 50~70% 예상 | UI에 "(참고용)" 표기 |
| 수집 제한 | 1회 500건, 일 1000건 | 하드코딩 |
| Rate Limit | 네이버 0.8~1.2s, 카카오 1.0~1.5s, 구글 2.0~3.0s | 랜덤 딜레이 |

## 최종 MVP 기능 목록

1. ✅ 지역명 + 업종 키워드 입력
2. ✅ 수집 플랫폼 선택 (네이버 기본ON / 카카오맵 기본ON / 당근 기본ON / 인스타보조 기본ON / 구글맵 기본OFF)
3. ✅ 네이버플레이스 검증 선택 옵션
4. ✅ 010 번호 필터링 (자동)
5. ✅ 중복 제거 (자동)
6. ✅ 엑셀 내보내기 (시트 분리)
7. ✅ 수집 진행 현황 (텍스트 로그 + 프로그레스 바)
8. ✅ 수집 이력 관리
