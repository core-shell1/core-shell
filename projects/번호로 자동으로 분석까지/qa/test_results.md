### 버그 #1: PPT 플레이스홀더 `(업체명)` 미치환

**증상**
- 슬라이드 2, 3번에 `(업체명)` 플레이스홀더가 그대로 남아있음
- 업체명이 일부 슬라이드에서만 교체되고, 나머지는 미교체 상태

**원인**
- `ppt_generator.py`가 슬라이드별로 개별 처리하지만, 일부 슬라이드에서 교체 로직이 실행되지 않음
- 전체 PPT 대상 일괄 교체 로직이 없었음

**수정 내용**
- `ppt_generator.py` 563라인 이후에 아래 코드 추가:
```python
# 먼저 전체 PPT에서 공통 플레이스홀더 일괄 교체
global_replacements = {
    "(업체명)": business_name,
    "업체명": business_name,
}
replace_all_text(prs, global_replacements)
log("PPT", f"전체 슬라이드에서 플레이스홀더 교체 완료: {business_name}")
```

**수정 결과**
- 재실행 후 모든 슬라이드에서 `(업체명)` 플레이스홀더 정상 교체 확인
