# CDO 분석 — 네이버 플레이스 자동 진단 영업 제안서 시스템

## 디자인 비전
"사장님이 자기 문제를 직접 발견하게 만드는 거울 제안서"
숫자가 감정을 자극하고, 비교가 위기감을 만들고, 해결책이 자연스럽게 계약으로 이어지는 심리적 여정.

---

## UX 원칙

### 1. 숫자는 충격, 문장은 공감
- **충격**: 경쟁사 대비 34%, 리뷰 68개 vs 내 12개
- **공감**: "알림 설정이 안 되어 있습니다" (비난 X, 사실 O)

### 2. 비교가 곧 위기감
- 절대 점수(D등급)보다 상대 점수(상위 평균 대비 -34%)가 더 강력함
- 색상으로 차이 강조: 경쟁사는 초록, 내 업체는 빨강/회색

### 3. 해결책은 마지막까지 숨겨라
- 문제 → 위기 → 비교 → "그래서 어떻게?" → 패키지 등장
- 5슬라이드까지 문제만 보여주고, 6슬라이드부터 해결책

### 4. 한 슬라이드 한 메시지
- 슬라이드당 핵심 메시지 1개 (숫자는 3개까지)
- 사장님이 읽지 않아도 차트/색상만 봐도 이해 가능

### 5. 모바일 퍼스트 (사장님은 폰으로 본다)
- 폰트 최소 18pt (제목 32pt+)
- 차트는 단순하게 (항목 최대 5개)
- 스크린샷은 실제 모바일 화면 크기

---

## PPT 슬라이드 재구성 (심리 흐름 기반)

### 현재 문제점
- 문제(진단) → 해결책(패키지)이 너무 빨리 나옴
- 경쟁사 비교가 없어서 "내가 왜 못하는지" 모름
- 숫자가 산재되어 있어 충격이 분산됨

### 제안: 11슬라이드 구조 (심리 단계별)

---

#### **Phase 1: 현실 인식 (슬라이드 1~3)**

**슬라이드 1: 표지**
- 현재 그대로 유지
- "○○○ 마케팅 분석 보고서"

**슬라이드 2: 업체 개요 (숫자로 현실 보여주기)**
- **배치**:
  - 좌측: 업체명, 카테고리, 주소, 전화번호
  - 우측 상단: 현재 순위 (큰 숫자) "18위"
  - 우측 중간: 키워드 조회수 (PC / 모바일 / 전체)
  - 우측 하단: 타겟 분석 (성별/연령 차트 썸네일)
- **심리**: "아, 우리 이 정도 순위구나" (아직 위기감 없음)

**슬라이드 3: 현재 검색 결과 (스크린샷)**
- **배치**:
  - 좌측: 네이버 검색 상단 (플레이스 1~5위)
  - 우측: 블로그 검색 상단 (1~3위)
  - 하단: "○○○ 검색 시 실제 노출 화면 (2026.03.25 기준)"
- **심리**: "우리가 안 보이네?" (첫 위기감)

---

#### **Phase 2: 위기감 형성 (슬라이드 4~6)**

**슬라이드 4: 경쟁사 비교 — 핵심 지표 3개 (신규)**
- **제목**: "상위 업체와의 격차"
- **배치**:
  - 3개 막대 비교 차트 (가로형)
    1. 리뷰 수: 경쟁사 평균 68개 vs 내 업체 12개
    2. 저장 수: 경쟁사 평균 234개 vs 내 업체 45개
    3. 사진 수: 경쟁사 평균 18장 vs 내 업체 3장
  - 색상: 경쟁사 막대 = 진한 초록, 내 업체 = 연한 회색 (or 주황)
  - 각 항목 옆에 차이 퍼센트: "-82%" (빨간 글씨)
- **심리**: "나만 뒤처지고 있구나" (강한 위기감)

**슬라이드 5: 22개 항목 종합 진단 (신규)**
- **제목**: "네이버 플레이스 종합 진단 결과"
- **배치**:
  - 좌측 (50%): 등급 원형 차트
    - 중앙 큰 글씨: "D등급" (빨강)
    - 아래 작은 글씨: "상위 평균 대비 -34%"
  - 우측 (50%): 22개 항목 체크리스트 (아이콘 + O/X)
    - ✅ 영업시간 등록
    - ❌ 대표 사진 5장 미만
    - ❌ 새소식 미등록
    - ❌ 리뷰 답글 없음
    - ... (22개 전부 표시)
- **심리**: "이렇게 많이 못하고 있었어?" (자기 인식)

**슬라이드 6: 세부 진단 — 치명적 항목 (신규)**
- **제목**: "개선이 시급한 3가지"
- **배치**:
  - 3개 블록 (세로 배치)
    1. 🔴 리뷰 답글 0% — "경쟁사는 95% 답글 작성 중"
    2. 🔴 새소식 방치 — "마지막 게시물: 3개월 전"
    3. 🟠 저장 수 격차 — "상위 업체의 19% 수준"
  - 각 블록: 문제 설명 + 경쟁사 비교 숫자
- **심리**: "당장 뭐라도 해야겠다" (행동 욕구)

---

#### **Phase 3: 확장 가능성 (슬라이드 7)**

**슬라이드 7: 확장 키워드 전략**
- **제목**: "○○○ 외 7개 키워드로 유입 확대 가능"
- **배치**:
  - 7개 키워드 카드 (3-2-2 배치)
  - 각 카드: 키워드명 + 월간 검색량 (큰 숫자)
  - 예: "양주 축구레슨" / 2,340회
- **심리**: "다른 키워드로도 손님이 올 수 있구나" (가능성)

---

#### **Phase 4: 해결책 제시 (슬라이드 8~10)**

**슬라이드 8: 패키지 소개 — 한눈에 비교**
- **제목**: "맞춤 마케팅 패키지"
- **배치**:
  - 3개 패키지 카드 (가로 배치)
    - 시선 패키지 (Basic)
    - 집중 패키지 (Pro) ⭐ 추천
    - 주목 패키지 (Premium)
  - 각 카드: 핵심 3줄 + 가격
- **심리**: "어떤 걸 선택해야 하지?" (선택의 순간)

**슬라이드 9: 패키지 상세 (장기 고객용)**
- **제목**: "집중 패키지 상세 내역"
- **배치**:
  - 좌측: 포함 항목 리스트 (체크박스)
  - 우측: 예상 효과 (숫자)
    - 순위 상승: 18위 → 5위권 목표
    - 리뷰 증가: +30개/월
    - 저장 증가: +50개/월
- **심리**: "이 정도면 해볼만한데?" (신뢰)

**슬라이드 10: 타임라인 (신규)**
- **제목**: "3개월 로드맵"
- **배치**:
  - 가로 타임라인 (1개월 → 2개월 → 3개월)
  - 각 구간: 작업 내용 + 예상 지표
  - 예: "1개월: 기본 세팅 완료 + 순위 12위"
- **심리**: "언제 결과가 나오는지 알겠네" (구체성)

---

#### **Phase 5: 계약 유도 (슬라이드 11)**

**슬라이드 11: 견적서**
- 현재 그대로 유지
- **추가 요소**: 하단에 "이번 주 계약 시 10% 할인" (긴급성)

## 핵심 UX 원칙 (최종 정리)

### 1. 숫자가 말하게 하라
- 문장보다 숫자가 먼저 눈에 띄도록
- 큰 숫자 (조회수, 순위) = 32pt+
- 비교 숫자 (경쟁사 vs 내) = 막대 차트로

### 2. 색상은 감정의 언어다
- 빨강 = 위기 (차이, 미달성)
- 주황 = 경고 (개선 필요)
- 초록 = 성공 (경쟁사, 목표)
- 회색 = 현재 (내 업체, 미흡)

### 3. 비교는 항상 옆에
- 경쟁사와 내 업체를 같은 화면에
- 막대/원형 차트로 시각적 차이 극대화
- "평균" 보다 "상위 5개" 비교가 더 강력

### 4. 한 슬라이드 한 감정
- 슬라이드 4: 충격 (격차)
- 슬라이드 5: 자기 인식 (등급)
- 슬라이드 6: 긴급성 (치명적 3가지)
- 슬라이드 8: 희망 (해결책)

### 5. 모바일에서 읽힌다
- 사장님은 카톡으로 받아서 폰으로 봄
- 최소 폰트: 18pt (본문), 28pt (제목)
- 차트 항목: 최대 5개
- 스크린샷: 실제 모바일 크기로

## CTO에게 요청

### 1. 데이터 수집 추가 필요
- **경쟁사 데이터** (상위 5개 업체):
  - 리뷰 수, 저장 수, 사진 수
