# 팀 설계

팀 이름: 인하우스 마케터·프리랜서용 통합 마케_팀

팀 업무:
아이디어: 인하우스 마케터·프리랜서용 통합 마케팅 대시보드 SaaS: GA4·메타·네이버SA 데이터를 한 화면에서 보고, AI가 이번 주 뭘 고쳐야 하는지 알려주는 툴. 타겟: 중소기업 인하우스 마케터(1~5인팀, 2~7년 경력) + 프리랜서 마케터(월 3~10개 클라이언트). 핵심: GA4+메타+네이버SA 3채널 통합 + AI 인사이트 + SaaS 구독(월 49달러~). 글로벌 툴이 네이버SA/카카오 미지원하는 빈틈.

## 시은의 자동 명확화

- **유형**: SaaS
- **타겟**: 중소기업 인하우스 마케터(1~5인팀, 경력 2~7년) + 프리랜서 마케터(클라이언트 3~10개 동시 운영)
- **핵심**: GA4·메타·네이버SA 3채널 통합 대시보드 + AI 주간 액션 인사이트 — 글로벌 툴의 네이버SA 미지원 공백 공략
- **수익 모델**: SaaS 구독 월 49달러~ (추정: 프리랜서 플랜 / 팀 플랜 티어 구조, 연간 할인)
- **우리 자산 활용**:
  - `naver-diagnosis` → 네이버SA 데이터 해석 로직 재활용 가능
  - 기존 소상공인·스마트스토어 클라이언트 → 초기 베타 유저 풀
  - 리안의 마케팅 대행 경험 → AI 인사이트 프롬프트 설계 (어떤 지표가 "고쳐야 할 신호"인지 도메인 지식)
  - 온라인팀 운영 노하우 → 프리랜서 마케터 커뮤니티 접근 채널
- **상용화 여부**: 상용화 (글로벌 SaaS, 한국 시장 우선 진입 후 확장 구조)

### STEP 0 변수 (서윤 Phase 1 검색용)

- **DOMAIN**: `마케팅 통합 대시보드 SaaS`
- **SUB_DOMAINS**:
  - `멀티채널 데이터 통합 (GA4·메타·네이버SA)`
  - `AI 마케팅 인사이트 자동화`
  - `프리랜서 마케터 클라이언트 관리`
  - `네이버SA / 카카오모먼트 API 연동`
  - `마케팅 리포트 자동화`

- **REGION**: 한국 (1차 타겟) → 글로벌 확장 옵션 보유

- **TARGET_SEGMENT**:
  - `B2B SMB`
  - 페르소나 A — *인하우스 마케터*: 직원 10~100인 중소기업, 혼자 또는 소수팀으로 GA4·메타·네이버SA 동시 운영, 매주 보고서 만드는 데 시간 낭비 중, 툴 구독 예산 월 5~15만원 수준
  - 페르소나 B — *프리랜서 마케터*: 클라이언트 3~10개 동시, 채널별 로그인 전환이 병목, 클라이언트별 리포트 자동화 니즈 강함, 툴에 월 3~10만원 지불 의향

- **DOMAIN_COMMUNITIES**:
  - 네이버 카페 `마케터들의 모임` / `디지털마케팅 실무자 모임`
  - 오픈카톡 `퍼포먼스 마케터 네트워크`
  - 추정: Facebook 그룹 `한국 디지털 마케터`
  - Reddit `r/PPC`, `r/analytics` (글로벌 경쟁사 동향 파악용)
  - 추정: 링크드인 `한국 마케터 커뮤니티` 그룹

- **OKR 추정**:
  - **Objective**: 한국 마케터가 "네이버SA까지 되는 유일한 통합 대시보드"로 인식하게 만든다
  - **KR1**: 베타 출시 3개월 내 유료 전환 유저 200명 달성 (MRR $9,800 이상)
  - **KR2**: 프리랜서 마케터 세그먼트 NPS 40 이상 (리포트 자동화 기능 만족도 기준)
  - **KR3**: 네이버SA + 메타 + GA4 동시 연동 완료 유저의 주간 리텐션(WAU) 60% 이상 유지
