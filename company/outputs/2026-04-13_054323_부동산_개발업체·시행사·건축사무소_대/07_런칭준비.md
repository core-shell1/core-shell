# 런칭 준비 플랜 — 토지 사업성 분석 자동화 SaaS

> **시은 판단 기준**: "내일 당장 이렇게" 수준. 이론 제거, 실행 가능한 것만.

---

## 1. 타겟

| 항목 | 내용 |
|------|------|
| **1순위 타겟** | 연 10~30건 토지 검토하는 **시행사 사업기획팀 실무자** (사원~차장급) — 엑셀·수작업으로 3~5일 소요, 반복 고통 명확 |
| **2순위 타겟** | **건축사무소 소장/실장** — 클라이언트에게 초기 사업성 리포트 제공해야 하는 포지션, 외주 비용 절감 니즈 |
| **3순위 (나중에)** | 부동산 개발업체 임원 — Go/No-Go 판단용. 단, 의사결정자라 영업 사이클 길어서 초기엔 후순위 |
| **구체적 타겟 정의** | 서울·수도권 소재, 직원 5~50인, 연매출 50억~500억 시행사 또는 건축사무소. 토지 검토 월 2건 이상 |
| **타겟이 모이는 곳** | ① 네이버 카페 "시행사 실무" 계열 ② 오픈카톡 시행사·시공사 실무자 그룹 ③ 건축사협회 세미나·교육 ④ 부동산 개발 유튜브 채널 댓글·커뮤니티 탭 ⑤ LinkedIn 한국 부동산 개발 그룹 |
| **예상 TAM** | 국내 시행사 약 3,000~5,000개 + 건축사무소 약 8,000개 = 약 1만 개사. 월 50만 원 기준 TAM 약 600억/년 |
| **현실적 SAM** | 수도권 집중, 디지털 툴 수용 가능한 곳 약 1,500~2,000개사 → SAM 약 90~120억/년 |
| **얼리어답터 특성** | ① 현재 엑셀+수작업 쓰면서 "이거 자동화되면 좋겠다" 말한 적 있는 사람 ② 스타트업 툴 써본 경험 있음 (노션, 슬랙 등) ③ 30~40대 실무 담당자 ④ 월 2건 이상 토지 검토 — 반복 고통이 있어야 지불 의사 생김 |

## 2. 경쟁사 벤치마크

| 경쟁사 | 상품 | 가격 (추정 포함) | 우리 대비 약점 |
|--------|------|----------------|--------------|
| **빅밸류** | AVM 기반 토지 시세 산정, 딥러닝 가격 비교 | API 연동 월 300~500만 원 (기업 계약) | 시세 산정에 집중 — **인허가 승인 확률·특례법 판단 없음**. 개발사 직접 사용 UI 미흡 |
| **밸류맵 (VDN)** | 매물 탐색·매칭 + 시세 비교 + 제안서 자동 생성 | 무료~월 수십만 원 (티어 미공개) | 거래 중개 플랫폼 성격 강함 — **사업성 분석 깊이 얕음**. 규제·특례법 판단 없음 |
| **디스코** | 역산 시뮬레이션 (임대→건축→토지 수익률) | 월 10~30만 원 추정 (소빌 타겟) | 소규모 빌딩 중심 — **대형 개발·특례법 적용 프로젝트 커버 안 됨** |
| **랜드업** | 개발 전 과정 연결 (기획-금융-시공) | 플랫폼 수수료 기반 (가격 불명확) | 플랫폼 중개 모델 — **분석 툴이 아닌 중개 서비스**. 실무자 단독 사용 어려움 |
| **건축사 수작업 (현재 표준)** | 엑셀 + 법령정보 수동 검색 + 외부 컨설턴트 | 건당 외주 50~200만 원 + 내부 3~5일 공수 | **속도 느림, 비용 높음, 사람 의존** — 이게 우리가 대체할 대상 |

> **블루오션 확인**: 인허가 승인 확률 예측 + 특례법 해당 여부 자동 판단을 **실무자가 직접 쓸 수 있는 UI**로 제공하는 SaaS — **현재 없음**

## 3. 우리 상품

| 플랜 | 가격 | 포함 내용 | 타겟 등급 |
|------|------|---------|----------|
| **스타터** | 월 29만 원 (건당 크레딧 5회) | ① 용적률·건폐율 자동 산출 ② 용도지역 규제 체크 ③ 기본 사업성 리포트 PDF 출력 | 건축사무소 소장 — 클라이언트 제출용 리포트 필요한 곳 |
| **프로** | 월 59만 원 (건당 크레딧 15회) | 스타터 전체 + ④ 특례법 해당 여부 자동 판단 ⑤ 인허가 승인 확률 스코어 ⑥ 투자금 범위 산정 ⑦ 정책 흐름 알림 | 시행사 사업기획팀 — 핵심 타겟 |
| **팀** | 월 129만 원 (무제한 + 멀티시트 5인) | 프로 전체 + ⑧ 팀 협업 (공유·코멘트) ⑨ 과거 분석 이력 관리 ⑩ 전담 온보딩 1회 | 중형 개발업체 팀 단위 도입 |

> **가격 설정 근거**: 현재 외주 컨설팅 건당 50~200만 원 → 월 2건만 써도 스타터 ROI 즉시 발생. 프로 59만 원은 내부 공수 1일치(실무자 일당 환산) 수준으로 저항 낮음

> **초기 전략**: 스타터·프로 2개만 먼저 출시. 팀 플랜은 유료 고객 10개사 이후 추가

## 4. 영업 채널

| 채널 | 자동화 가능? | 법적 제한 | 우선순위 |
|------|------------|---------|---------|
| **콜드 DM (링크드인)** | 반자동 (메시지 템플릿 + 수동 발송) | 스팸 정책 — 1일 20~30건 이내 유지, 자동 발송 툴 사용 시 계정 제한 위험 | ★★★★★ 1순위 — 시행사 임원·실무자 직접 접근 가능 |
| **네이버 카페 DM/댓글** | 수동 (자동화 불가 — 어뷰징 감지) | 카페 운영 규정상 홍보 게시물 제한. 직접 광고 금지 카페 多 | ★★★★ 2순위 — 단, 가치 제공형 게시물로 자연 유입 유도 |
| **오픈카톡 실무자 그룹** | 수동 | 그룹 규정 확인 필수. 무단 홍보 시 강퇴 | ★★★ 3순위 — 신뢰 쌓은 후 소개 방식으로 |
| **이메일 콜드메일** | 자동화 가능 (Instantly, Lemlist 등) | 정보통신망법 — 수신 동의 없는 영리 목적 이메일 발송 **원칙적 위법**. 단, B2B 업무 관련성 있는 경우 예외 인정 판례 있음. 수신거부 링크 필수 | ★★★ 3순위 — 법적 리스크 관리하며 소량 운영 |
| **건축사협회·시행사 협회 세미나** | 불가 (오프라인) | 없음 | ★★★ 3순위 — 신뢰도 높음, 단 시간 비용 큼 |
| **유튜브 광고 (타겟팅)** | 자동화 가능 | 없음 | ★★ 4순위 — 초기엔 예산 낭비 가능성. 고객 10개사 이후 집행 |

> **초기 영업 핵심 전략**: 링크드인 콜드 DM + 네이버 카페 가치 제공형 콘텐츠 2개 채널 집중. 리안이 직접 시행사 네트워크 활용해 첫 5개사 파일럿 무료 제공 → 후기 확보 → 유료 전환
