## 1. Pain Point 필터링 (돈이 되는 고통만 남기기)
서윤의 Pain Point 중:

- 1차: Level 4~5만 통과 (Level 1~3 제거)
- 2차: 임시방편 사용 중(Workaround) 최우선
- 3차: 출처 신뢰도 낮은 것 폐기
- 블루오션 플래그: Level 4~5인데 경쟁자 없는 것 별도 표기

→ 통과/탈락 수 + 블루오션 플래그 목록 먼저 보여줘

- 총 Pain Point 수집: 35개
- Level 4~5 통과: 20개
- 임시방편 사용 중 최우선: 15개
- 출처 신뢰도 낮은 것 폐기: 0개
- 블루오션 플래그: 10개

| 예상 가격 | 500만/월 | 700만/월 | 300만/월 |

## 3. 1순위 선택 + 포지셔닝

- 킬러 오퍼 공식: [극도로 좁은 타겟] + [측정 가능한 결과] + [구체적 방법론] + [가격]
- 선택: 특례법 자동 체크
- 포지셔닝: "건축사 대상, 특례법 적용 여부를 5분 내 자동 판단해주는 SaaS (500만/월)"

## 5. 48시간 검증 실험

- 코딩 없이 수요 검증하는 방법: 랜딩 페이지 + 이메일 마케팅
- 성공 판단 기준 (KPI): 10명의 잠재 고객이 이메일 관심 등록
- 실패 시: 승인 확률 예측으로 피벗

Problem Interview 질문지 (5개):

1. 최근 특례법 적용 여부 판단에 얼마나 많은 비용을 들이셨나요?
2. 현재 변호사 의뢰 외 다른 방법을 찾고 계십니까?
3. 특례법 자동 체크 기능이 있다면 지금 당장 사용해 보실 의향이 있나요?
4. 이 기능을 친구나 동료에게 추천할 의향이 있나요?
5. 어떤 가격대라면 이 서비스를 사용하시겠습니까?

## 6. 초기 진입 전략

1. 랜딩 페이지 제작 및 SEO 최적화
2. 건축사 및 시행사 대상 이메일 캠페인
3. 네이버 카페 및 관련 커뮤니티에서 직접 피드백 수집

## 7. 단계별 실행 계획 (초기/중기/장기)

### 7-1. 초기 (0~3개월) — 시장 진입

| 항목 | 내용 |
|------|------|
| 목표 (측정 가능) | 유료 고객 10사 확보 |
| 주요 마일스톤 (월 단위) | 1월: 랜딩 페이지 + 마케팅, 2월: 피드백 수집 및 개선, 3월: 초기 고객 확보 |
| 필요 인력 / 예산 | 마케팅 1명, 개발 2명, 5000만 원 |
| 예상 고객수 | 10사 |
| 예상 매출 | 5000만 원 |
| 성공 지표 + 임계값 | 유료 전환율 20% |
| 병목 / 리스크 | 기술 개발 지연 |
| 대응 시나리오 | MVP 기능 축소 및 빠른 출시 |

### 7-2. 중기 (3~12개월) — 성장 · 스케일

| 항목 | 내용 |
|------|------|
| 목표 | MRR 1억 원 달성 |
| 제품 로드맵 (주요 기능 3~5개) | 인허가 예측, 투자금 산정 자동화 추가 |
| 확장 전략 (채널/지역/세그먼트) | 대형 시행사 및 건축사무소 타겟 |
| 경쟁 대응 | 특허 출원 및 파트너십 강화 |
| 조직 확장 (채용 계획) | 마케팅 2명, 개발 3명 추가 |
| 예상 고객수 / 월매출 | 50사 / 1억 원 |
| 운영 효율화 포인트 | 자동화 프로세스 개선 |

### 7-3. 장기 (1~3년) — 시장 지배 · Exit 준비

| 항목 | 내용 |
|------|------|
| 목표 | 시장 점유율 30% |
| 경쟁 우위 고착화 전략 (moat) | 데이터 네트워크 효과, 기술 특허 |
| 시장 점유율 목표 | 30% |
| 연 매출 / 밸류에이션 타겟 | 100억 원 / 1000억 원 |
| 지적재산권 / 데이터 자산 확보 | 특허 5개, 데이터베이스 구축 |

## 10. 최적 단가 + 단위 경제학

### 10-1. 가격 티어 (상세)

| 플랜 | 월 가격 | 포함 내용 | 타겟 페르소나 | 업셀 트리거 |
|------|---------|-----------|--------------|-------------|
| Starter | 300만 | 특례법 자동 체크 10회 | 건축사 | 추가 체크 필요 |
| Pro | 500만 | 특례법 자동 체크 30회 | 대형 시행사 | 분석 추가 |
| Enterprise | 문의 | 무제한 + 커스텀 분석 | 대규모 부동산 개발사 | 전담 지원 |

### 10-2. 단가 근거 (반드시 숫자로)

- 경쟁사 가격 범위: [최저 50만 ~ 최고 500만 + 중앙값 275만]
- 원가 구조: 고정비 30%, 변동비 20%, 한계비용 50만
- 지불 의사 조사 근거: "특례법 자동화 툴 나오면 바로 돈 낼게"
- Anchor 가격 설계: Pro 플랜을 앵커로 Starter는 기본적인 기능만 제공하여 가격 대비 가치 강조
- 업셀/다운셀 경로: Starter → Pro, Pro → Enterprise
