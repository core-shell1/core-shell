## [1] Problem Statement

**"월 매출 500만~3,000만 스마트스토어·쿠팡 셀러는 광고·콘텐츠·SNS를 여러 에이전시에 분산 계약(월 300만)하거나 직접 수작업(월 150시간)하는 상황에서 데이터 불일치와 ROAS 하락을 겪고 있다. 이는 통합 운영 대행사 부재 때문이며, 현재 OSC(컨설팅만), Cafe24(앱 산재), Adure(콘텐츠만)로 임시방편을 쓰고 있어 월 450만 상당 비용·시간 손실이 발생한다."**

**근거**:
- Pain #8: "기술 역량 부족, 콘텐츠 부담으로 많은 브랜드들이 플랫폼 의존" (Groobee) — 여러 에이전시 분산 계약 월 300만 + 데이터 불일치
- Pain #1: "콘텐츠 생성 부담으로 많은 브랜드들이 플랫폼 의존" (Groobee) — 직접 스마트폰 촬영 월 50시간
- Pain #22: "콘텐츠 생성 부담" (Groobee) — A/B 테스트 수작업 월 50시간

### Targeting (1개 선택)

**MVP 최초 공략**: 세그먼트 ② — 월 1,000만~3,000만 스마트스토어·쿠팡 셀러

**선택 근거**:
- **Pain 강도**: ★★★★★ (5점) — Pain #8 "여러 에이전시 월 300만 + 데이터 불일치" + Pain #1 "콘텐츠 직접 제작 월 50시간" 직격, workaround 비용 월 450만 상당으로 최고
- **도달 가능성**: 
  - 네이버 카페 "스마트스토어 판매자 센터 공식 카페" (회원 15만+, 일 방문 5천+)

  - 네이버 카페 "쿠팡 파트너스·로켓그로스 셀러 모임" (회원 8천+)
  - 리안 컴퍼니 기존 스마트스토어 셀러 DB 100명 (주목/집중 패키지 고객)
- **지불 의사**: Pain #8 "월 300만 분산 계약" 실제 지출 중 → 49만 패키지는 85% 절감, ROI 명확
- **의사결정자 명확성**: 2~5인 팀 대표 또는 마케팅 담당자 1명이 의사결정, 결재 라인 단순

### Positioning (한 문장)

**"월 1,000만~3,000만 스마트스토어·쿠팡 셀러에게, OSC(컨설팅만)·Cafe24(앱 산재) 대비 광고·콘텐츠·SNS 통합 운영으로 월 300만 절약 + ROAS 3.5 보장하는, 월 49만 원 리테이너 서비스"**

**차별화 근거**:

- Cafe24 약점 (Layer 5): "앱 마켓 산재로 통합 부재" — CRM 앱, 광고 앱 따로 설치 필요, 데이터 불일치 → 리안은 통합 대시보드 제공
- Adure 약점 (Layer 5): "콘텐츠 제작만" — 광고 연동 미비 → 리안은 광고·콘텐츠 동시 운영

## [3] Value Proposition Canvas

### 고객 프로필 (오른쪽)

**Customer Jobs** (3가지 층)
- **기능적 Jobs**: 
  - 광고 ROAS 3.5 이상 달성 (월 광고비 100만 → 매출 350만)
  - 상세페이지 클릭률 15% 향상 (현재 5% → 목표 20%)
  - SNS 팔로워 월 1천 증가 (현재 주 1회 포스팅 → 주 3회)
- **감정적 Jobs**: 
  - "여러 에이전시 관리하느라 지친 상태에서 벗어나고 싶다" (Pain #8)
  - "콘텐츠 직접 만들다 밤샘하는 피로에서 해방되고 싶다" (Pain #1)
  - "ROAS 하락 보며 느끼는 불안감 해소하고 싶다" (Pain #3)
- **사회적 Jobs**: 
  - 동료 셀러들에게 "우리는 전문 대행사 쓴다"고 자랑하고 싶다
  - 가족에게 "이제 주말에 일 안 해도 된다"고 말하고 싶다

**Pains** (Pain # 인용)
- **P1**: Pain #8 — "여러 에이전시 분산 계약으로 월 300만 쓰고도 데이터 불일치로 ROAS 추적 어려움"
- **P2**: Pain #1 — "광고 운영만 신경 쓰다 콘텐츠 제작 시간 부족으로 월 50시간 직접 촬영, 품질 저하"
- **P3**: Pain #22 — "A/B 테스트 콘텐츠 수작업으로 월 50시간 소모, 통계 부족으로 최적 콘텐츠 식별 못함"
- **P4**: Pain #13 — "SNS 콘텐츠 일관성 유지 실패, 주 1회 포스팅으로 팔로워 증가 정체"
- **P5**: Pain #19 — "SNS-광고 연동 미비로 크로스 채널 유입 시너지 없음"

**Gains**
- **G1**: 월 300만 에이전시 비용 → 49만 절감으로 월 251만 순이익 증가
- **G2**: 월 150시간 콘텐츠·광고 수작업 → 0시간으로 주말 해방
- **G3**: ROAS 2.0 → 3.5 달성으로 월 매출 75% 증가 (광고비 100만 기준)
- **G4**: 통합 대시보드로 광고·콘텐츠·SNS 데이터 한눈에 파악, 의사결정 속도 3배

### 가치 맵 (왼쪽)

**Products & Services**
- 광고 운영 (네이버 쇼핑·쿠팡 광고 월 관리)
- 상세페이지 A/B 테스트 자동화 (2개 버전 업로드 + 전환율 비교)
- SNS 콘텐츠 제작·포스팅 (인스타·블로그 주 3회)
- 통합 리포트 대시보드 (광고·콘텐츠·SNS 데이터 실시간 연동)
- naver-diagnosis 스토어 진단 툴 (무료 진단 → 유료 전환)

**Pain Relievers** (각 Pain에 1:1 대응)
- **P1 → 통합 운영**: 광고·콘텐츠·SNS를 하나의 팀(리안 컴퍼니)이 운영, 데이터 통합 대시보드로 불일치 해소 + 월 300만 → 49만 절감
- **P2 → 콘텐츠 제작 대행**: 온라인팀이 상세페이지 촬영·편집 대행, 월 50시간 → 0시간 절감
- **P3 → A/B 테스트 자동화**: 2개 버전 자동 업로드 + 전환율 자동 비교, 월 50시간 → 5시간 절감
- **P4 → SNS 통합 운영**: 주 3회 포스팅 + 일관된 톤앤매너, 팔로워 월 1천 증가 보장
- **P5 → 크로스 채널 연동**: SNS 포스팅 → 광고 리타겟팅 자동 연동, 유입 2배 증가

**Gain Creators**
- **G1 → 가격 절감**: 월 49만 패키지로 여러 에이전시 월 300만 대체, 월 251만 순이익 증가
- **G2 → 시간 해방**: 광고·콘텐츠·SNS 모두 대행, 월 150시간 → 0시간 절감
- **G3 → ROAS 보장**: 첫 3개월 ROAS 3.5 미달 시 전액 환불 정책, 리스크 제로
- **G4 → 통합 인사이트**: 주간 자동 리포트 발송, 의사결정 속도 3배 향상

## [4] 공감 지도 (Empathy Map)

**페르소나**: 김지영 (32세 여성, 월 1,500만 매출 스마트스토어 패션 셀러, 2인 팀 대표)

- **Think & Feel**: 
  - "광고는 A사, 콘텐츠는 B사, SNS는 직접... 이거 관리하느라 정작 제품 기획할 시간이 없다"
  - "ROAS가 2.0에서 안 올라가는데, 광고사는 '콘텐츠 문제'라 하고 콘텐츠사는 '광고 문제'라 한다. 누구 말이 맞는 거지?"
  - "주말에도 상세페이지 사진 찍느라 가족 시간 못 가진다. 언제까지 이럴 건가..."
  - "동료 셀러들은 전문 대행사 쓴다는데, 우리는 왜 이렇게 쪼개서 할까. 부끄럽다"

- **See**: 
  - 네이버 카페에서 "통합 마케팅 대행 추천" 글 보며 부러워함
  - 경쟁사 상세페이지는 전문가 촬영인데, 우리는 스마트폰 사진이라 초라함
  - OSC 블로그에서 "쿠팡 전략" 읽지만 실행 방법 모호
  - Cafe24 앱 마켓에서 CRM 앱 보지만 "또 설치해야 하나..." 망설임

- **Hear**: 
  - 동료 셀러: "우리는 대행사 하나로 다 맡겼어. 너도 그래"
  - 광고 대행사: "ROAS 낮은 건 상세페이지 때문이에요. 콘텐츠 개선하세요"
  - 콘텐츠 외주: "광고 타겟팅이 잘못됐나 봐요. 광고사한테 물어보세요"
  - 가족: "주말에도 일해? 쉬어야지..."
  - 고객 리뷰: "사진이 실물이랑 달라요" (콘텐츠 품질 저하 지적)

- **Say & Do**: 
  - 겉으로: "광고·콘텐츠 다 관리하고 있어요" (자신감 있는 척)
  - 실제: 매일 밤 11시까지 엑셀로 광고비·매출 수동 정리 (Pain #10)
  - 겉으로: "SNS 마케팅도 하고 있어요"
  - 실제: 주 1회 포스팅이 고작, 팔로워 증가 정체 (Pain #13)
  - Workaround: 광고 A사(월 150만), 콘텐츠 B사(월 100만), SNS 직접(월 50시간) 분산

- **Pain**: 
  - 가장 큰 두려움: "이렇게 비효율적으로 하다가 경쟁사에 밀려 망하는 거 아닐까"
  - 좌절: "여러 에이전시 관리하느라 정작 제품 기획·고객 응대 시간 없다. 본업을 못 한다"

- **Gain**: 
  - "이것만 해결되면" 이상적 상태: 
    - "광고·콘텐츠·SNS를 하나의 팀이 알아서 운영해주고, 나는 주간 리포트만 보면서 의사결정만 하면 된다"
    - "월 49만으로 월 300만 절약하고, 주말엔 가족과 시간 보낸다"
    - "ROAS 3.5 달성해서 매출 2배 증가, 동료들에게 자랑한다"

## [9] Flywheel + Engine of Growth

### Flywheel 초안 (6개 요소)

```
1. 무료 진단 (naver-diagnosis) → 2. Pain 시각화 (ROAS·콘텐츠 부재) → 3. 유료 전환 (집중 49만)
     ↑                                                                              ↓
6. 입소문 확산 (추천 0.5명) ← 5. 재계약·업셀 (시선 89만) ← 4. ROAS 3.5 달성 (만족)
```

**각 연결고리 작동 원리**:
- **1→2**: naver-diagnosis 무료 진단 툴로 스토어 데이터 입력 → CRM 부재, 콘텐츠 품질 저하, 광고 ROAS 2.0 미만 시각화 → "이거 고쳐야겠다" 인지
- **2→3**: 진단 결과 하단에 "집중 49만 패키지로 즉시 개선" CTA + "첫 3개월 ROAS 3.5 미달 시 전액 환불" 보장 → 리스크 제로로 전환율 5% 달성
- **3→4**: 광고 운영 + 콘텐츠 A/B 테스트 + 통합 대시보드로 1개월 내 ROAS 3.0 달성 → 2개월 차 ROAS 3.5 달성 → 고객 만족도 상승 (NPS 40+)
- **4→5**: 3개월 계약 종료 시 "시선 89만으로 SNS·CRM 추가하면 매출 2배" 제안 → 업셀 전환율 30% + 재계약율 70%
- **5→6**: 만족한 고객이 네이버 카페·오픈채팅에서 "리안 컴퍼니 추천" 후기 작성 → 고객 1명당 평균 0.5명 추천 (Viral Coefficient 0.5)
- **6→1**: 추천받은 신규 고객이 naver-diagnosis 무료 진단 신청 → 선순환 가속

### Engine of Growth 선택

**선택**: **Sticky (리텐션 기반)**

**근거**:
- 이커머스 마케팅은 지속 운영 필요 (광고·콘텐츠 월 관리) → 한 번 계약하면 빠져나가기 어려움
- 통합 대시보드에 광고·콘텐츠·SNS 데이터 누적 → Switching Cost 높음 (다른 대행사로 이전 시 데이터 손실)
- ROAS 3.5 달성 시 "이탈하면 다시 ROAS 2.0으로 떨어진다" 두려움 → 유지율 70% 목표
- Viral 엔진은 부적합 (B2B 특성상 바이럴 확산 느림), Paid 엔진은 초기 CAC 1.7만으로 효율적이나 장기적으로 Sticky가 LTV 극대화

**가속 트리거**: 
- **첫 달 ROAS 3.0 달성** — 고객이 "효과 있다" 체감 → 2개월 차 이탈률 급감
- 통합 대시보드 주 1회 로그인 습관 형성 → 데이터 의존도 증가 → 이탈 심리적 장벽

**해자 (Moat) 가능성**:
