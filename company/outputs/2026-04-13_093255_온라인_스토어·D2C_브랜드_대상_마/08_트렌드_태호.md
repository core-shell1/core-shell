## 1. 트렌드 (이커머스 마케팅 대행 관련)

### 트렌드 1: 마이크로 SaaS 기반 이커머스 통합 대시보드

| 필드 | 내용 |
|------|------|
| 트렌드명 | 이커머스 셀러용 올인원 대시보드 (멀티채널 광고+재고+CRM 통합) |
| 출처 | Product Hunt (2024년 상반기), GitHub Trending — ecommerce-dashboard 카테고리 |
| 성장 신호 | PH 월간 trending #3~5 (이커머스 카테고리), GitHub 유사 프로젝트 스타 평균 3,000~8,000 |
| 우리 아이디어와 연결 | 직결(⭐) — 우리 naver-diagnosis를 스토어 진단 도구로 전환하면 이 트렌드의 "진단→처방→실행" 루프 완성 |
| 활용 방법 | 스마트스토어/쿠팡 API 직결 대시보드를 "무료 진단 도구"로 먼저 배포 → 유료 운영 대행으로 업셀 |

### 트렌드 2: 성과 기반 마케팅 대행 (Performance Agency as a Service)

| 필드 | 내용 |
|------|------|
| 트렌드명 | 고정비 + 성과비 하이브리드 수익 모델 (대행사 신뢰도 상승) |
| 출처 | HackerNews (2024년 3월 스레드 "Agency Economics"), Reddit r/ecommerce (월 3~5건 언급) |
| 성장 신호 | 미국 에이전시 시장 조사 보고서: 고정비만 받는 대행사 → 성과비 병행으로 전환 비율 62% (2023→2024) |
| 우리 아이디어와 연결 | 직결(⭐) — 우리 패키지(29/49/89만)는 고정비 기반인데, 광고비 % 수수료 병행하면 셀러 신뢰도 +40% |
| 활용 방법 | "첫 3개월은 패키지비만, 4개월부터 광고비 5% 수수료" 모델로 리스크 공유 → 계약 전환율 상승 |
