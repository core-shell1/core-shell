## 1. 타겟

| 항목 | 내용 |
|------|------|
| **구체적 타겟 (1순위)** | 월 매출 500만~1,500만 / 스마트스토어 단독 운영 / 1~2인 / 광고는 네이버 쇼핑 검색광고만 돌리는 셀러 |
| **구체적 타겟 (2순위)** | 월 매출 1,500만~3,000만 / 스마트스토어+쿠팡 병행 / 2~5인 / 외주 디자이너·광고 대행 따로 쓰다 비용·소통 피로 누적 |
| **제외 타겟** | 월 매출 3,000만 초과 (내부 마케터 고용 가능 구간), 쿠팡 로켓배송 전용 (콘텐츠 자율도 낮음) |
| **타겟이 모이는 곳** | 네이버 카페 '스마트스토어 판매자 센터 공식 카페' (회원 50만+) / 카카오 오픈채팅 '스마트스토어 셀러 모임' / 유튜브 '스마트스토어 강의' 댓글·커뮤니티 / 셀스타·판다랭크 카페 / 오프라인: 네이버 커넥트재단 셀러 교육 행사 |
| **예상 TAM** | 스마트스토어 활성 판매자 약 50만 명 중 월 매출 500만 이상 추정 5~8만 명 → 대행 서비스 수요층 약 1~2만 명 (전환율 2% 가정 시 연 200~400계약) |
| **얼리어답터 특성** | ① 이미 광고비 월 30만 이상 집행 중 ② ROAS·전환율 용어 알지만 직접 최적화 못 함 ③ 인스타·블로그 콘텐츠 직접 찍다 지침 ④ "한 곳에 다 맡기고 싶다"는 말 커뮤니티에서 반복 노출 ⑤ 뷰티·식품·생활용품 카테고리 집중 |
| **페르소나 한 줄** | "네이버 광고는 돌리는데 ROAS가 뭔지 모르고, 상세페이지는 2년 전 그대로고, 재구매 문자는 한 번도 안 보낸 스마트스토어 3년차 셀러" |

## 2. 경쟁사 벤치마크

| 경쟁사 | 상품 | 가격 (월) | 우리 대비 약점 |
|--------|------|-----------|--------------|
| **OSC (oscsnm.com)** | 스마트스토어 광고 운영 단독 | 30만~70만 추정 | 콘텐츠·CRM 미포함, 통합 운영 불가 |
| **플레이스엠 계열 소형 대행사** | 퍼포먼스 광고 (네이버·메타) | 50만~100만 | 이커머스 특화 아님, 상세페이지 별도 견적 |
| **크몽 프리랜서 (상위 노출)** | 상세페이지 제작 단건 | 20만~80만 (건당) | 운영 지속성 없음, 광고·CRM 연계 불가 |
| **마플·브랜디 연계 대행** | D2C 자사몰 마케팅 | 100만~300만 | 소규모 셀러엔 가격 장벽, 최소 계약 조건 있음 |
| **네이버 공식 광고 대행사 (NAS)** | 검색광고 운영 | 광고비 10~15% 수수료 | 콘텐츠 없음, 담당자 교체 잦음, 소규모 셀러 우선순위 낮음 |
| **직접 경쟁 없음 (블루오션)** | 광고+콘텐츠+CRM 통합 / 소규모 셀러 전용 / 월 29만~89만 | — | **이 구간 통합 패키지 공급자 사실상 없음** |

> **핵심 인사이트:** 경쟁사는 광고 OR 콘텐츠 OR CRM 중 하나만 함. 통합 운영을 29만~89만에 제공하는 곳 없음. 이게 우리 포지션.

## 3. 우리 상품

| 플랜 | 가격 (월) | 포함 내용 | 타겟 등급 |
|------|-----------|---------|----------|
| **주목** | **29만원** | 스토어 진단 리포트 1회 + 네이버 쇼핑 검색광고 세팅·운영 (광고비 별도, 월 30만 이하) + 월간 성과 리포트 1회 | 월매출 500만~800만 / 광고 처음 맡기는 셀러 |
| **집중** | **49만원** | 주목 전체 + 상세페이지 리뉴얼 1건/월 + SNS 콘텐츠 4건/월 (인스타·스마트스토어 공용) + 격주 리포트 | 월매출 800만~1,500만 / 콘텐츠+광고 동시 필요 |
| **시선** | **89만원** | 집중 전체 + 카카오 알림톡·문자 CRM 세팅+운영 (재구매 시퀀스 3종) + 쿠팡 광고 병행 운영 + 주간 리포트 + 전담 담당자 슬랙 연결 | 월매출 1,500만~3,000만 / 멀티채널 통합 원하는 셀러 |
| **애드온 (선택)** | **+15만원~** | 라이브커머스 기획·진행 / 자사몰(카페24) 마케팅 세팅 / 광고비 대행 수수료 (광고비 10%) | 시선 플랜 고객 대상 |

> **가격 정책:** 광고비는 별도 (클라이언트 직접 집행). 대행 수수료는 월정액에 포함. 광고비 10% 수수료는 시선 플랜 이상 옵션.
> **최소 계약:** 3개월 (중도 해지 시 위약금 1개월치)

## 4. 영업 채널

| 채널 | 자동화 가능? | 법적 제한 | 우선순위 |
|------|------------|---------|---------|
| **기존 고객 DB 재접촉** (스마트스토어 셀러 기존 고객) | ✅ 카카오 알림톡 자동화 가능 | 수신동의 확인 필수 (개인정보보호법) | **1순위 — 내일 당장** |
| **네이버 카페 DM / 댓글 영업** (스마트스토어 공식 카페 등) | ❌ 수동 (자동화 시 계정 제재) | 카페 운영 규정상 광고성 DM 금지 → 가치 제공형 댓글로 우회 | **2순위** |
| **카카오 오픈채팅 참여** | ❌ 수동 | 오픈채팅 광고 금지 → 전문가 포지셔닝으로 신뢰 쌓기 | **3순위** |
| **인스타그램 DM 콜드아웃리치** | ⚠️ 반자동 (Manychat 등) | 스팸 계정 제재 위험 — 일 20건 이하 수동 권장 | **4순위** |
| **네이버 블로그 SEO 인바운드** | ✅ 콘텐츠 발행 자동화 가능 | 없음 | **5순위 (중장기)** |
| **네이버·메타 광고 (리드 광고)** | ✅ 완전 자동화 | 없음 | **6순위 (예산 확보 후)** |

> **법적 주의:** 수신동의 없는 문자·이메일 발송은 정보통신망법 위반 (과태료 3,000만원 이하). 기존 고객 DB는 동의 여부 먼저 확인.
