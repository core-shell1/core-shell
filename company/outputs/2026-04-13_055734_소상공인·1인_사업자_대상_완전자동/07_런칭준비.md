# 런칭 준비 구체화 — 소상공인 완전자동 마케팅 SaaS

---

## 1. 타겟

| 항목 | 내용 |
|------|------|
| **구체적 타겟 (1순위)** | 월매출 300~1,500만원 네이버 스마트스토어 1인 셀러 — 상세페이지·광고·SNS 직접 돌리는 중, 자동화 니즈 명확, 결제 경험 있음 |
| **구체적 타겟 (2순위)** | 직원 1~3명 뷰티샵(네일·피부·헤어) 원장, 30~42세 — 인스타·네이버 블로그 직접 운영하다 지쳐있음, 월 마케팅 예산 20~50만원 |
| **제외 타겟** | 카페 (마진 낮고 결제 전환율 낮음, 2차 공략), 프리랜서 (LTV 낮음) |
| **타겟이 모이는 곳** | 네이버 카페 "스마트스토어 판매자 모임" (회원 38만+) / 네이버 카페 "소상공인 마케팅 연구소" / 카카오 오픈채팅 "스토어 셀러 모임" / 인스타 해시태그 #스마트스토어운영 #네일샵원장 |
| **예상 TAM → SOM** | TAM: 국내 소상공인 500만 / SAM: 디지털 마케팅 집행 중인 77만 / SOM (1년 내 현실 목표): 1,000개사 → MRR 1,500만원 |
| **얼리어답터 특성** | 이미 ChatGPT·뤼튼 써본 적 있음 / 월 5~10만원 SaaS 결제 경험 있음 (크몽·미리캔버스 등) / "자동화"라는 단어에 반응함 / 후기·사례 보고 결정하는 타입 |
| **페인포인트 1문장** | "콘텐츠 만들 시간도 없고, 대행사 맡기자니 비싸고, 혼자 하자니 뭘 해야 할지 모르겠다" |

## 2. 경쟁사 벤치마크

| 경쟁사 | 상품 | 가격 | 우리 대비 약점 |
|--------|------|------|--------------|
| **PostDot** | 블로그 콘텐츠 자동 생성·발행 (네이버·티스토리 등 4개 플랫폼) | 월 55,000원 | 블로그만 됨. 진단·전략·SNS·성과추적 없음. 단순 발행 도구 |
| **Mirra** | SNS 콘텐츠 스케줄링 + AI 숏폼 자동 생성 | 무료~월 $19 (약 2.6만원) | 한국 채널(네이버 블로그·스마트플레이스) 연동 없음. 전략 없음 |
| **Buffer** | SNS 예약 발행 | 무료~채널당 월 $6 | 콘텐츠 생성 없음. 한국어 최적화 없음. 완전 수동 |
| **뤼튼** | AI 글쓰기 도구 | 무료~월 9,900원 | 발행 자동화 없음. 진단·전략·성과추적 없음. 범용 도구 |
| **크몽 마케팅 대행** | 콘텐츠 제작·SNS 운영 대행 | 월 30~100만원 | 사람이 하는 대행. 느리고 비쌈. 확장성 없음 |
| **마케팅 대행사 패키지** | 블로그+인스타 운영 대행 | 월 50~200만원 | 비용 압도적으로 높음. 소상공인 접근 불가 |
| **Jasper** | AI 카피라이팅 | 월 $49~ | 한국어 품질 낮음. 발행 자동화 없음. 소상공인 UX 아님 |

**결론**: 현재 시장에 "진단→전략→생성→배포→추적"을 한국어로 원스톱 제공하는 소상공인 특화 SaaS 없음. **이게 우리 포지션.**

## 3. 우리 상품

| 플랜 | 가격 | 포함 내용 | 타겟 등급 |
|------|------|---------|----------|
| **Starter (주목)** | 월 29,000원 | ① AI 마케팅 진단 리포트 (월 1회) ② 블로그 콘텐츠 자동 생성 월 8건 ③ 네이버 블로그 자동 발행 ④ 기본 성과 대시보드 | 뷰티샵 원장, 막 시작한 스토어 셀러 — "일단 써보자" 층 |
| **Pro (집중)** | 월 49,000원 | Starter 전부 + ① 콘텐츠 월 20건 ② 인스타·블로그 멀티채널 발행 ③ AI 전략 방향 설계 (월 1회 업데이트) ④ 성과 추적 + 개선 제안 리포트 | 월매출 500만+ 스토어 셀러, 직원 있는 뷰티샵 |
| **Enterprise (시선)** | 월 89,000원 | Pro 전부 + ① 콘텐츠 무제한 ② 카카오채널·유튜브쇼츠 배포 추가 ③ 경쟁사 모니터링 ④ 전담 AI 에이전트 커스터마이징 ⑤ 월 1회 전략 리뷰 (줌 30분) | 멀티채널 운영 중인 중규모 소상공인, 프랜차이즈 가맹점 |
| **베타 특가** | 월 19,000원 (3개월 한정) | Pro 기능 전체 — 초기 100개사 한정, 후기 제공 조건 | 얼리어답터 확보 + PMF 검증용 |

**가격 근거**: 경쟁사 최고가(PostDot 5.5만) 아래 진입, 대행사(30만+) 대비 90% 절감 포지셔닝. "커피 한 잔 값으로 마케팅 자동화" 메시지 가능.

## 4. 영업 채널

| 채널 | 자동화 가능? | 법적 제한 | 우선순위 |
|------|------------|---------|---------|
| **네이버 카페 DM·댓글** | 반자동 (템플릿 작성 후 수동 발송) | 스팸성 홍보 댓글 → 카페 운영 정책 위반 위험. 직접 홍보 금지 카페 많음. **정보성 글 먼저, DM은 반응자에게만** | ★★★★★ 1순위 |
| **인스타 DM** | 반자동 (ManyChat 등 활용) | 대량 자동 DM → 계정 정지 위험. 하루 20~30건 수동 한도 권장 | ★★★★☆ 2순위 |
| **카카오 오픈채팅 참여** | 수동 | 홍보 채팅방 → 강퇴 위험. 가치 제공 후 자연스럽게 소개 | ★★★☆☆ 3순위 |
| **네이버 블로그 SEO** | 완전 자동화 가능 | 제한 없음 | ★★★★★ 1순위 (장기) |
| **구글 SEO** | 완전 자동화 가능 | 제한 없음 | ★★★☆☆ 3순위 (장기) |
| **메타 광고 (인스타·페북)** | 자동화 가능 (광고 세팅 후) | 광고 심사 기준 준수 필요. 허위·과장 광고 금지 | ★★★★☆ 2순위 |
| **이메일 콜드메일** | 자동화 가능 (Mailchimp 등) | 정보통신망법 — 수신 동의 없는 광고성 메일 발송 **불법**. 반드시 수신동의 확보 후 발송 | ❌ 초기 제외 |
| **크몽·탈잉 플랫폼 입점** | 수동 등록 후 자동 유입 | 플랫폼 수수료 20% | ★★★☆☆ 보조 채널 |
