# 추론된 리안 워크플로우:

1. **고객 유입**: 기존 온·오프라인 마케팅 대행 고객(뷰티/카페/스마트스토어) → 패키지(주목/집중/시선) 상담 → 계약
2. **진단·설계**: naver-diagnosis 로직으로 업체 현황 분석 → 채널 전략 수동 설계 (현재 리안 또는 온라인팀이 직접 수행)
3. **콘텐츠 생성·배포**: 온라인팀이 SNS/네이버 콘텐츠 제작 → 수동 업로드·스케줄링 (자동화 미비 구간)
4. **성과 보고**: 월 단위 리포트 수동 작성 → 고객사 전달 (반복 공수 발생 구간)
5. **재계약·업셀**: 성과 기반 패키지 업그레이드 유도 → 현재 사람이 직접 관리 (이탈 리스크 구간)

> **핵심 병목 추론**: 3(콘텐츠 생성·배포)과 4(성과 보고)가 수동 반복 구간 → SaaS 자동화의 1순위 타깃
