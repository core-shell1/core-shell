- **페르소나 2**: 동네 카페 사장 / 38~45세 / 네이버·인스타 병행, 마케팅 예산 월 30~50만 원인데 효과 모름, 프리랜서에게 월 50만 원 씀

## 💢 4. 진짜 고통 (3~5개)
- **콘텐츠 만드는 데 시간이 너무 많이 걸림**: 매일 인스타·블로그 글 직접 씀 → Canva 수동 편집 + 엑셀 템플릿 사용 → 주 25시간, 월 60만 원 인건비 상당 씀
- **마케팅 전문가 고용 불가**: 뭘 어떻게 해야 할지 모름 → 프리랜서 월 1회 아웃소싱 → 월 50만 원 쓰는데 일관성 없음
- **여러 채널에 따로따로 올리는 게 너무 번거로움**: 인스타·네이버·카카오 각각 수동 포스팅 → 스프레드시트로 일정 관리 → 주 10시간 소비
- **내 마케팅이 잘 되고 있는지 알 수가 없음**: 성과 데이터가 여기저기 흩어짐 → 주말마다 엑셀로 수동 집계 → 주 8시간 낭비
- **AI 쓰고 싶은데 고객 정보 유출될까 봐 무서움**: 병원·뷰티샵 고객 데이터 걱정 → 그냥 수동으로 버팀 → 연 100만 원 이상 기회비용

## 🏆 6. 왜 이길 수 있나
1. **AI오투오는 진단만 하고 끝** ← 우리는 진단 → 콘텐츠 생성 → SNS 배포 → 성과 추적까지 한 번에 연결
2. **메타플라이어는 데이터 보안 불만이 가장 큰 약점** ← 우리는 개인정보보호법 준수 + "데이터 안전 보장" 마케팅으로 병원·뷰티샵 공략
3. **채널톡·똑똑한 매니저는 마케팅 자동화가 핵심이 아님** ← 우리는 소상공인 마케팅 하나만 파고, 리안의 naver-diagnosis 진단 로직이 이미 있어서 개발 시간 단축
