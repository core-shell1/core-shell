📋 리안 요약 (한 장)

📋 1. 한 줄 설명

소상공인(카페·뷰티샵·스마트스토어 등)이 업체 정보만 입력하면 AI가 마케팅 진단 + SNS 글 3개를 30초 안에 자동으로 만들어주는 월 구독 서비스.

🧭 2. 사업 성격 분류

유형: 하이브리드

초기 3년은 월 구독료로 꾸준히 현금 창출 → 고객 1만 명·연 매출 50억 원 달성 후 네이버·카카오에 M&A 매각 목표.
분류 근거: SaaS 구독 구조라 매달 현금이 쌓이고, 데이터 자산이 커질수록 인수 매력도도 올라감.

🎯 3. 누구한테 팔 건가

  • 페르소나 1: 뷰티샵 원장 / 30~35세 / 인스타 직접 운영 중, 콘텐츠 아이디어 고갈, 파트타임 직원에게 월 60만 원 씀
  • 페르소나 2: 동네 카페 사장 / 38~45세 / 네이버·인스타 병행, 마케팅 예산 월 30~50만 원인데 효과 모름, 프리랜서에게 월 50만 원 씀

💢 4. 진짜 고통 (3~5개)

  • 콘텐츠 만드는 데 시간이 너무 많이 걸림: 매일 인스타·블로그 글 직접 씀 → Canva 수동 편집 + 엑셀 템플릿 사용 → 주 25시간, 월 60만 원 인건비 상당 씀
  • 마케팅 전문가 고용 불가: 뭘 어떻게 해야 할지 모름 → 프리랜서 월 1회 아웃소싱 → 월 50만 원 쓰는데 일관성 없음
  • 여러 채널에 따로따로 올리는 게 너무 번거로움: 인스타·네이버·카카오 각각 수동 포스팅 → 스프레드시트로 일정 관리 → 주 10시간 소비
  • 내 마케팅이 잘 되고 있는지 알 수가 없음: 성과 데이터가 여기저기 흩어짐 → 주말마다 엑셀로 수동 집계 → 주 8시간 낭비
  • AI 쓰고 싶은데 고객 정보 유출될까 봐 무서움: 병원·뷰티샵 고객 데이터 걱정 → 그냥 수동으로 버팀 → 연 100만 원 이상 기회비용

💰 5. 얼마에 팔까

플랜 가격 타겟 이유
기본 월 29만 원 카페·소규모 가게 프리랜서 아웃소싱(월 50만 원)보다 저렴, 진입 장벽 낮춤
프로 월 49만 원 뷰티샵·스마트스토어 SNS 자동 배포 + 성과 리포트 포함, 파트타임 직원(월 60만 원) 대체
엔터프라이즈 월 89만 원 프랜차이즈·병원 전담 매니저 + 맞춤 콘텐츠, 대행사 수준 서비스

🏆 6. 왜 이길 수 있나

  1. AI오투오는 진단만 하고 끝 ← 우리는 진단 → 콘텐츠 생성 → SNS 배포 → 성과 추적까지 한 번에 연결
  2. 메타플라이어는 데이터 보안 불만이 가장 큰 약점 ← 우리는 개인정보보호법 준수 + "데이터 안전 보장" 마케팅으로 병원·뷰티샵 공략
  3. 채널톡·똑똑한 매니저는 마케팅 자동화가 핵심이 아님 ← 우리는 소상공인 마케팅 하나만 파고, 리안의 naver-diagnosis 진단 로직이 이미 있어서 개발 시간 단축

🔧 7. 회사 자산 활용도

활용도 점수: 7/10

  • naver-diagnosis: 진단 엔진의 핵심 로직으로 그대로 재활용 → 개발비 약 30% 절감
  • 온라인팀 운영 노하우: 소상공인 마케팅 실전 경험 → AI 프롬프트 설계·콘텐츠 품질 검수에 직접 투입
  • 기존 소상공인 고객 데이터: 첫 베타 사용자 50명 확보 경로로 활용 (카카오톡 메시지 발송)

우선순위 기능 설명 이유 의존 기능

P1 콘텐츠 히스토리 생성된 콘텐츠 목록, 배포 일시, 성과 기록 사용자 편의성 콘텐츠 자동 생성
P1 월간 리포트 자동 생성 월별 콘텐츠 생성 건수 + 배포 건수 + 총 조회수 요약 (이메일 발송) 구독 가치 강화 성과 추적
P2 글로벌 다국어 지원 영어/일본어/베트남어 콘텐츠 생성 글로벌 확장 콘텐츠 자동 생성
P2 고급 진단 (경쟁사 분석) 사용자 입력 → 경쟁사 자동 검색 → 비교 분석 고급 기능, 고가격 티어용 마케팅 진단
P2 AI 챗봇 (마케팅 상담) 사용자 질문 → GPT-4 → 맞춤형 마케팅 조언 고객 지원 자동화 없음

Must NOT (범위 외)

  • 광고 집행 자동화 — 네이버/구글 광고 자동 생성·입찰은 P2 이상. 초기 복잡도 과다, 규제 리스크 높음.
  • CRM 통합 — 고객 관리 기능은 제외. 마케팅 자동화에만 집중.
  • 전자상거래 연동 — 스마트스토어 상품 정보 자동 수집은 P2. 초기는 수동 입력만.
  • 영상 콘텐츠 생성 — 이미지/영상 생성은 비용·시간 과다. 텍스트 콘텐츠만.
  • 오프라인 채널 지원 — 카톡 채널톡, 문자 발송 등은 제외.
  • 다중 팀 협업 — 1인 사업자 기준. 팀 기능은 나중에.

User Flow

시나리오 1: 신규 사용자 → 진단 → 콘텐츠 생성 (30분)

1단계: 사용자가 랜딩페이지 방문 → "지금 시작하기" 클릭
→ 시스템: 회원가입 페이지 표시 (이메일 + 비밀번호)

2단계: 회원가입 완료 → 이메일 인증 링크 클릭
→ 시스템: 대시보드로 리다이렉트, "업체 정보 입력" 모달 표시

3단계: 사용자가 업체 정보 5개 필드 입력 (업체명, 업종, 상품, 타겟 고객, 현재 채널)
→ 시스템: 입력값 검증, "진단 시작" 버튼 활성화

4단계: 사용자가 "진단 시작" 클릭
→ 시스템: 로딩 화면 표시, 백그라운드에서 GPT-4 호출 (진단 생성, 약 10초)

5단계: 진단 완료 → 대시보드에 진단 결과 표시 (강점/약점/기회/위협)
→ 시스템: 자동으로 콘텐츠 생성 시작 (GPT-4, 약 15초)

6단계: 콘텐츠 3개 생성 완료 → 대시보드에 표시 (인스타 캡션 + 네이버 블로그 제목 + 카톡 공지)
→ 사용자: 각 콘텐츠 개별 수정 가능, 다운로드 또는 복사

7단계: 사용자가 "결제하기" 클릭
→ 시스템: 토스페이먼츠 결제 페이지 표시 (월 15만원, 첫 달 무료 옵션)

8단계: 결제 완료
→ 시스템: 구독 활성화, 대시보드에 "다음 진단 예약" 버튼 표시

시나리오 2: 기존 사용자 → 월간 재진단 (5분)

1단계: 사용자가 대시보드 접속
→ 시스템: "이번 달 진단 가능" 배너 표시 (월 1회 무료)

2단계: 사용자가 "새 진단 시작" 클릭
→ 시스템: 이전 업체 정보 자동 로드, 수정 옵션 제공

3단계: 사용자가 정보 수정 후 "진단 시작" 클릭
→ 시스템: 진단 + 콘텐츠 생성 (총 25초)

4단계: 결과 표시 → 사용자가 콘텐츠 다운로드/복사
→ 시스템: 히스토리에 기록

  3. 데이터 안전 + 프라이버시 내장:
    - 경쟁사(메타플라이어) 최대 불만: "보안·데이터 프라이버시 리스크". 우리는 AES-256 암호화 + 개인정보보호법 준수 + 명시적 데이터 정책 공개.
    - 구체적 가치: 병원/뷰티샵 고객 데이터 안전 보장 → 신뢰 기반 고객 확보 → 경쟁사 대비 차별화.
