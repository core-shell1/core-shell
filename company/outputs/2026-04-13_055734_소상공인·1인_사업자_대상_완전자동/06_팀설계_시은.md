# 팀 설계

팀 이름: 완전자동 마케팅 SaaS 실행팀

팀 업무:
1. **제품 기획·설계**: 업체 정보 입력 → 진단 → 전략 → 콘텐츠 생성 → 채널 배포 → 성과 추적까지 6단계 자동화 플로우 설계 및 기능 명세 작성
2. **핵심 엔진 개발**: naver-diagnosis 로직을 진단 모듈로 전환, AI 콘텐츠 생성·자동 배포·성과 리포팅 파이프라인 구축 (Starter/Pro/Enterprise 3티어 기준)
3. **콘텐츠·프롬프트 설계**: 뷰티샵·카페·스마트스토어 셀러 업종별 콘텐츠 생성 프롬프트 및 채널별(네이버·SNS) 배포 워크플로우 제작
4. **초기 고객 검증·영업**: 기존 대행 고객 대상 48시간 내 MVP 테스트, 월 15만원 Starter 플랜 100개사 유치 목표로 온보딩 프로세스 설계 및 실행
5. **성과 추적·개선**: MRR·이탈률·ARPU 지표 모니터링, 월 단위 데이터 기반 플랜 업셀 및 글로벌 확장 준비
