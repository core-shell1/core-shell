# 런칭 준비 구체화 — 이커머스 마케팅 대행 버티컬

---

## 1. 타겟

| 항목 | 내용 |
|------|------|
| **구체적 타겟** | 월 매출 500만~3,000만 원 / 스마트스토어 메인 + 쿠팡 병행 운영 / 뷰티·생활용품·식품 카테고리 / 1~3인 운영 / 광고는 직접 돌리지만 ROAS 들쭉날쭉 / 상세페이지 직접 만들거나 프리랜서 단건 의뢰 중 |
| **타겟이 모이는 곳** | ① 네이버 카페 "스마트스토어 판매자 센터" (공식, 회원 100만+) ② 네이버 카페 "셀러허브" ③ 카카오 오픈채팅 "스마트스토어 셀러 모임" (키워드 검색 상위 5개 방 각 1,000~5,000명) ④ 유튜브 — 스마트스토어 운영 채널 댓글란 ⑤ 네이버 블로그 — "스마트스토어 광고 세팅" 검색 유입층 |
| **예상 TAM** | 국내 스마트스토어 활성 셀러 약 50만 명 중 월매출 500만~3,000만 구간 추정 5~8만 명 / 이 중 외주 의향 있는 셀러 10~15% = **5,000~12,000명** / 월 49만 원 기준 연 SOM 약 **294억~706억 원** |
| **얼리어답터 특성** | "광고비 월 50만~200만 직접 집행 중인데 ROAS 1.5 이하" / "상세페이지 직접 만들었는데 전환율 1% 미만" / "재구매율 0%, 카카오채널 있는데 방치 중" / 이미 돈 쓰고 있어서 설득 비용 낮음 |

> **얼리어답터 1순위 시그널:** 네이버 광고 관리자 월 집행 50만 원 이상 + 스마트스토어 리뷰 100개 이상 + 직원 없음. 이 3개 겹치면 즉시 영업 대상.

## 2. 경쟁사 벤치마크

| 경쟁사 유형 | 대표 상품 | 가격 | 우리 대비 약점 |
|------------|----------|------|--------------|
| **대형 퍼포먼스 대행사** (나스미디어·메조미디어 계열) | 네이버·카카오 광고 운영 대행 | 월 150만~300만 원 + 광고비 10~15% | 최소 광고비 300만 이상 요구 / 소규모 셀러 안 받음 / 콘텐츠·CRM 별도 |
| **프리랜서 마케터** (크몽·숨고) | 스마트스토어 광고 세팅 단건 | 건당 10만~30만 원 | 지속 운영 불가 / 광고만, 콘텐츠·CRM 없음 / 책임 소재 불명확 |
| **상세페이지 제작사** (디자인 스튜디오) | 상세페이지 제작 단건 | 건당 30만~80만 원 | 제작 후 끝 / 광고·CRM 연결 없음 / 전환율 보장 없음 |
| **소규모 통합 대행사** (지역 기반) | 광고+SNS 패키지 | 월 30만~60만 원 | 이커머스 특화 아님 / 스마트스토어·쿠팡 알고리즘 이해 낮음 / CRM 없음 |
| **SaaS 툴** (채널톡·아임웹 마케팅) | CRM 자동화 툴 | 월 5만~20만 원 | 직접 운영해야 함 / 셋업 러닝커브 높음 / 광고·콘텐츠 없음 |

> **핵심 갭:** 월 50만~100만 원 구간에서 광고+콘텐츠+CRM을 **하나의 팀이 통합 운영**해주는 대행사가 없음. 대형사는 너무 비싸고, 프리랜서는 파편화됨.

## 3. 우리 상품

| 플랜 | 가격 | 포함 내용 | 타겟 등급 |
|------|------|----------|----------|
| **주목** | 월 29만 원 | ① 스마트스토어 진단 리포트 (naver-diagnosis 툴 활용) ② 네이버 쇼핑 광고 세팅 + 월 1회 최적화 리포트 ③ 카카오채널 자동 메시지 1종 세팅 (재구매 유도) | 월매출 500만~1,000만 / 광고 처음 맡기는 셀러 |
| **집중** | 월 49만 원 | 주목 전체 + ① 상세페이지 리뉴얼 1건/월 ② 네이버·쿠팡 광고 동시 운영 ③ 카카오채널 자동화 3종 (첫구매/재구매/이탈) ④ 월간 성과 리포트 (ROAS·재구매율·전환율) | 월매출 1,000만~2,000만 / 광고 직접 돌리다 지친 셀러 |
| **시선** | 월 89만 원 | 집중 전체 + ① 인스타·블로그 콘텐츠 월 8건 제작 ② 자사몰 CRM 풀 세팅 (신규/VIP/이탈 세그먼트) ③ 쿠팡 로켓그로스 광고 최적화 ④ 월 2회 전략 미팅 (줌) ⑤ 광고비 집행 대행 (광고비 별도, 수수료 10%) | 월매출 2,000만~3,000만 / 자사몰 병행 또는 준비 중인 셀러 |

> **가격 설계 근거:** 프리랜서 단건(10~30만) 대비 지속성 제공 / 대형 대행사(150만+) 대비 진입장벽 낮춤 / 집중(49만)이 주력 — 얼리어답터 설득 단가로 최적

## 4. 영업 채널

| 채널 | 자동화 가능? | 법적 제한 | 우선순위 |
|------|------------|---------|---------|
| **naver-diagnosis 툴 → 인바운드 전환** | ✅ 완전 자동화 가능 (진단 결과 → 상담 신청 CTA) | 없음 | **1순위** — 이미 자산 있음 |
| **네이버 카페 DM / 댓글 영업** | ⚠️ 반자동 (타겟 필터링 자동, 발송은 수동) | 카페 규정상 홍보 댓글 금지 多 / 스팸 신고 위험 | **2순위** — 직접 가치 제공 댓글로 우회 |
| **카카오 오픈채팅 DM** | ⚠️ 반자동 | 오픈채팅 운영자 규정 / 무차별 DM 차단 위험 | **3순위** — 소량 정밀 타겟팅 |
| **네이버 블로그 SEO** | ✅ 콘텐츠 자동화 가능 (GPT 초안 + 수동 발행) | 없음 | **2순위** — 장기 인바운드 |
| **인스타그램 유료 광고** | ✅ 자동화 가능 (Meta 광고 자동 최적화) | 없음 | **3순위** — 예산 확보 후 |
| **콜드 이메일** | ✅ 자동화 가능 (스마트스토어 공개 연락처 수집) | ⚠️ 정보통신망법 — 수신 동의 없는 영리 목적 이메일 발송 금지 (과태료 3,000만 원 이하) | **보류** — 법적 리스크 |
| **유튜브 댓글 영업** | ⚠️ 반자동 | 플랫폼 스팸 정책 | **4순위** |

> **핵심 전략:** 콜드 아웃바운드보다 **naver-diagnosis 툴로 인바운드 만들기**가 법적으로 안전하고 전환율 높음. 셀러가 직접 진단 받으러 오는 구조.
