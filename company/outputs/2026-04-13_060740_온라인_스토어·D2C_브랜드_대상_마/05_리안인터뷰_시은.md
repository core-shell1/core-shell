# 추론된 리안 워크플로우:

1. **영업 훅**: naver-diagnosis 툴로 스마트스토어 셀러 무료 진단 → 문제 시각화 → 패키지 제안 (주목 29만~시선 89만)
2. **계약 온보딩**: 광고(퍼포먼스) + 콘텐츠(상세페이지·SNS) + CRM(재구매) 니즈 파악 → 패키지 배정 → 월정액 + 광고비 % 수수료 구조 확정
3. **실행 루틴**: 온라인팀이 광고 세팅·콘텐츠 제작·CRM 자동화 병렬 운영 → 리안은 전략 방향·클라이언트 커뮤니케이션 담당 (직접 실행 최소화)
4. **성과 보고**: ROAS·재구매율·콘텐츠 성과 데이터 취합 → 클라이언트에게 월 1회 리포트 → 리텐션 유지 근거로 활용
5. **업셀 트리거**: 3개월 성과 데이터 기반 → 상위 패키지 전환 or 추가 채널(쿠팡·자사몰) 확장 제안
