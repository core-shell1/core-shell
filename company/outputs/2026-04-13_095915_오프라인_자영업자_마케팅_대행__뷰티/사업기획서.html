1 📋 한 줄 설명

오프라인 자영업자(뷰티샵·카페·식당)를 위해 인스타·네이버플레이스·블로그 통합 마케팅을 월 구독형으로 대행하는 서비스 (29~89만원/월)

2 🧭 사업 성격 분류

마케팅 대행 서비스 (B2B, 월 구독형 리테이너)
- 타입: 아웃소싱 대행 + AI 파이프라인 기반
- 상용화 방식: 외부 판매 서비스
- 기존 비즈니스와의 관계: 리안 자체 온라인팀 운영 경험 기반

3 🎯 누구한테 팔 건가

1~5인 오프라인 자영업자
- 주요층: 35세 네일샵/헤어샵 원장, 카페 사업가, 식당 사장
- 마케팅 예산: 월 30~100만원대
- 특징: 직접 운영 여력 없고, 디지털 마케팅에 소극적이거나 피로도 높음
- 시장규모 SOM: 500억원 (1,000개 고객 × 49만원 기준)

4 💢 진짜 고통 (3~5개)
  1. 인스타그램 콘텐츠 직접 제작 시간 부족 — 사업에 집중 못함 (Pain #1, Level 4)
  2. 네이버플레이스 리뷰 관리 수작업 — 이미지/리뷰 엑셀 관리 오류 (Pain #2, #14, Level 4)
  3. 광고·블로그 운영 스킬 부족 — 무료 툴 시행착오, 효과 측정 불가 (Pain #4, #21, Level 4)
  4. 플랫폼 수수료로 마케팅 예산 소진 — 자체 채널 확보 필요 (Pain #5, Level 4)
  5. 다중 플랫폼 동시 관리 복잡성 — 흩어진 데이터, 통합 성과 측정 불가 (Workaround 관찰 26개)
5 💰 얼마에 팔까

월 정기 구독 3단계 패키지
- 기본(주목): 29만원/월 — 기본 콘텐츠 + 네이버플레이스 관리
- 중급(집중): 49만원/월 — 콘텐츠 + 광고운영 + 성과보고 ⭐ 타겟
- 프리미엄(시선): 89만원/월 — 풀 서비스 + 광고비 대행 수수료

업셀 수익: 광고비 운영 시 20% 수수료 추가

경쟁 우위 요소 (점수 7/10)
1. naver-diagnosis 영업 훅: 네이버플레이스 진단 → 계약 전환의 구체적 진입점 (직접 경쟁사 없음)

3. 기존 온라인팀 운영 노하우: 리안 자체 경험으로 신뢰도 높음
4. "오프라인 자영업자 전문" 버티컬 포지셔닝: 24개 블루오션 Pain Point 중 우리 것

Phase 2 — 경쟁사 5-Layer 역기획

경쟁사 1: Cafe24 (https://www.cafe24corp.com)

Layer 1 — IA (Information Architecture)

Cafe24
├── 솔루션
│   ├── 전자상거래 플랫폼
│   │   ├── 쇼핑몰 구축
│   │   ├── 모바일 앱
│   │   └── POS 시스템
│   ├── 마케팅 솔루션
│   │   ├── 광고 자동화
│   │   ├── 이메일 마케팅
│   │   └── SNS 연동
│   └── 배송 및 정산
├── 가격
│   ├── 스탠더드
│   ├── 비즈니스
│   └── 엔터프라이즈
├── 파트너 & 마켓플레이스
│   ├── 앱 스토어
│   ├── 전문가 네트워크
│   └── 통합 파트너
└── 고객 지원

    ├── 커뮤니티
    └── 기술 지원

Cafe24의 메인 네비게이션은 솔루션(상품) → 가격 → 파트너 → 지원의 4단계 구조로 B2B SaaS 표준을 따르고 있습니다. 서브메뉴를 포함하면 약 30개 이상의 세부 페이지로 구성되어 있으며, 타겟 기능(예: 쇼핑몰 구축)까지 평균 3-4클릭이 소요됩니다. 전체 네비게이션은 엔터프라이즈 고객과 중소 자영업자를 모두 대상으로 설계되어 정보량이 많고 복잡한 편입니다.

사용자가 가장 자주 사용하는 경로는 솔루션 → 전자상거래 플랫폼 → 쇼핑몰 구축 → 요금제 비교 → 가입으로 추정됩니다. 식당이나 카페 사업자라면 POS 시스템 → 배송 연동 → SNS 광고 연동의 경로를 따를 가능성이 높습니다. 이는 약 6-8단계의 결정 단계를 거치게 되며, 중소 자영업자 입장에서는 과도하게 복잡한 구조입니다.

소상공인 및 1~5인 사업장 관점에서 IA 평가는 낮은 이해도, 높은 복잡도입니다. 전체 서비스가 엔터프라이즈 솔루션 중심으로 구성되어 있어, "우리 같은 작은 가게에는 어떤 기능이 필요한가?"라는 질문에 명확한 답변을 찾기 어렵습니다. 카테고리 구분이 기능 중심이지 비즈니스 규모나 산업 중심이 아니라는 점이 진입 장벽을 높입니다.

Layer 2 — User Flow

Cafe24의 전형적인 사용자 흐름은 다음과 같습니다:

  1. 회원가입 (2-3분): 이메일 입력 → 비밀번호 설정 → 추가 정보 입력 (사업자 번호, 은행계좌) — 허들: 사업자 번호 필수, 금융정보 요구로 초기 진입 거부감
  2. 상품 선택 (5-10분): 3개 티어 중 선택 (스탠더드 vs 비즈니스 vs 엔터프라이즈) — 허들: 각 플랜의 기능 차이가 명확하지 않아 선택 지연
  3. 쇼핑몰 템플릿 선택 (10-20분): 100개 이상 템플릿 중 선택 — 허들: 선택지 과다로 의사결정 지연 또는 포기
  4. 기본 설정 (30-60분): 상품 등록, 배송 규칙, 결제 연동 — 허들: 매우 복잡한 설정 절차, 엑셀 업로드 요구, 기술 지식 필요

추정 아하 모먼트: 사용자가 "첫 번째 상품이 실제로 판매되거나 첫 고객 리뷰가 등록되는 시점"입니다. 온라인 쇼핑몰 플랫폼의 특성상 대시보드에서 매출 수치가 증가하는 순간이 가장 강력한 아하 모먼트입니다. 하지만 리테일 자영업자(카페, 네일숍)에게는 온라인 쇼핑몰 자체가 필수 기능이 아니므로, 이들에게 아하 모먼트는 "SNS 광고 연동으로 첫 방문객 유입"이 될 것입니다.

아하 모먼트 도달 시간: 평균 1-2주일. 첫 상품 등록만 해도 2-3일이 소요되고, 실제 판매까지는 추가 1-2주일이 필요합니다. 식당/카페 대상이라면 네이버플레이스 연동만 해도 2-3일 내 로컬 검색 노출 가능하므로 아하 모먼트 도달이 더 빠를 수 있습니다.

주요 이탈 가능 지점 3개:
1. 가입 후 결제 정보 입력 단계 (허들: 사업자 번호/통장 계좌 요구로 신뢰도 확인 단계 → 이탈률 추정 30-40%). Cafe24 신뢰도가 높은 기업이지만, 초기 자영업자는 온라인 결제 시스템에 불안감을 느낍니다.
2. 템플릿 선택 후 상품 등록 단계 (허들: 100개 이상의 템플릿은 선택 마비 현상 유발 → 이탈률 추정 25-35%). 어떤 템플릿을 선택해야 "우리 사업에 맞는가"에 대한 가이드 부족.
3. 첫 주문 발생 전까지의 대기 기간 (허들: 설정만 하고 실제 매출 없이 월 구독료만 나가는 상황 → 이탈률 추정 40-50%). 특히 오프라인 카페/식당 사장은 "온라인 쇼핑몰"이 수익 창출로 이어진다는 확신이 부족합니다.

가격 티어 사이 전환 허들:
- Free → Starter (스탠더드): 무료 플랜이 없으므로 초기 비용 부담. 최소 월 29,000원(스탠더드)부터 시작하며, 스탠더드 → 비즈니스(월 49,000원) 업셀링은 "상품 등록 수 증가" 또는 "대역폭 초과"가 발생했을 때만 유도됩니다. 명확한 업그레이드 트리거가 부족해 자연 업셀링 전환율이 낮습니다. 비즈니스 → 엔터프라이즈(맞춤형 가격)는 영업 협상 필요로 진입 장벽이 높습니다.

Layer 3 — UI/UX

항목 평가 근거
3초 인지 테스트 △ 헤더의 "솔루션 / 가격 / 파트너"는 명확하지만, "우리 매장에 맞는 도구는 뭔가?"라는 질문에 3초 내 답변 불가능. 너무 많은 정보가 한 화면에 제시됨.
로그인 후 첫 화면 정보 과부하 × 대시보드에 "주문 관리 / 상품 관리 / 통계 / 배송 / 결제 / 마케팅"이 모두 노출되어 뷰티샵 사장이 "어디서부터 시작하나?"를 판단하기 어려움. 온보딩 가이드 부재.
CTA 명확성 (외부/내부) ○ 상세 페이지의 "지금 시작하기" 또는 "무료 체험" 버튼은 충분히 눈에 띔. 내부 네비게이션의 CTA도 비교적 명확함.
빈 상태 처리 △ 신규 가입 후 "상품이 없습니다"라는 메시지만 표시되고, "첫 상품을 추가하려면?" 같은 가이드가 불충분. 빈 상태를 기회로 활용하지 못하고 있음.
에러/로딩 상태 ○ 대체로 적절한 로딩 표시와 에러 메시지 제공. 기술 지원 연락처 제시.
모바일 대응 △ 모바일 앱이 별도로 제공되어 기본 기능 접근 가능하지만, 복잡한 설정은 PC 버전 필수. 카페 사장이 휴대폰으로 주문 확인은 되지만, 상품 등록은 어려움.
온보딩 가이드 품질 × 신규 가입 후 체계적인 튜토리얼 부재. 유튜브 링크나 PDF 매뉴얼로 대체하고 있으나, 인앱 가이드가 거의 없음. 1~5인 사업장이 따라가기 어려움.

종합 평가: Cafe24는 기술 수준은 높지만, 소상공인 UX에 최적화되지 않음. 정보량이 너무 많고, 초보자 온보딩이 약함. 디자인은 깔끔하지만 "뷰티샵 원장이 직관적으로 이해할 수 있는가?"라는 관점에서 △ 평가입니다.

Layer 4 — Business Model

수익 모델 유형: Cafe24는 구독형 SaaS + 거래 수수료 하이브리드 모델을 채택하고 있습니다. 주요 수익원은 (1) 월 구독료(기본 플랜), (2) 결제 수수료(카드 결제 시 2.9% + 수수료), (3) 배송 연동 수수료, (4) 부가 기능 애드온(예: 이메일 마케팅)입니다. 근거: Cafe24는 공개적으로 "쇼핑몰 구축 + 결제 연동 + 배송 통합"을 핵심 서비스로 홍보하고 있으며, 모든 온라인 거래 시 플랫폼이 수수료를 취득합니다.

가격 티어 상세:
- 스탠더드 (월 29,000원): 상품 50개 제한, 기본 배송 관리, 이메일 마케팅 미포함
- 비즈니스 (월 49,000원): 상품 500개, 고급 배송 규칙, 이메일 마케팅 포함, API 접근
- 엔터프라이즈 (맞춤형, 추정 월 100,000원 이상): 무제한 상품, 전담 지원, 커스터마이제이션

앵커링 전략 사용 여부: 명확한 앵커링 전략이 없음. 3개 플랜의 기능 차이가 선형적(상품 수 제한)이지, 가치 제안이 명확하지 않습니다. "비즈니스 플랜을 쓰면 ROI가 2배 증가한다"는 주장이 없고, 순전히 기술 스펙(상품 수, 대역폭)으로만 구분하고 있습니다. 이는 중소 자영업자 입장에서 "과연 업그레이드할 가치가 있는가?"를 판단하기 어렵게 만듭니다.

무료→유료 전환 트리거: 무료 플랜 자체가 없으므로, 체험 → 유료로의 전환이 아니라 "처음부터 결제" 구조입니다. Cafe24는 14일 무료 체험(신용카드 입력 필수)을 제공하지만, 신용카드 정보를 요구하는 순간 많은 초보자가 이탈합니다. 체험 기간 내에 "첫 주문 발생", "매출 $100 도달" 같은 명확한 성공 지표를 보여주지 못하면, 체험 만료 후 자동 결제 대신 해지를 선택할 가능성이 높습니다.

추정 MAU/ARPU/연 매출:
- MAU (Monthly Active Users): 공개되지 않지만, 한국 중소 쇼핑몰 시장의 약 30-40%가 Cafe24를 사용한다는 추정으로 약 50,000-100,000 MAU (한국 기준)
- ARPU (Average Revenue Per User): 가입자 구성이 스탠더드(월 29K) 50%, 비즈니스(월 49K) 40%, 엔터프라이즈(월 100K+) 10% 추정 시, 평균 약 48,000원/월. 여기에 결제 수수료(거래액의 2.9%)를 더하면 실제 ARPU는 월 60,000~80,000원 추정.
- 연 매출 (추정): 70,000 MAU × 70,000원/ARPU × 12개월 = 약 590억원 추정 (공개되지 않은 수치이므로 참고만 함)

추정 TAM/SAM:
- TAM (Total Addressable Market): 한국 전자상거래 시장 약 25조원 (2025년 기준). 쇼핑몰 구축 솔루션 시장만 해도 1-2조원.
- SAM (Serviceable Available Market): 한국 중소 쇼핑몰 운영자 약 300,000명 × 평균 ARPU 70,000원 = 약 2,520억원 (연)

Layer 5 — 약점 + 기회

실제 사용자 불만 TOP 3 (Pain Point와 상관성 분석):

  1. 불만: "기존 툴에 돈 내고 있지만 기능 없어서 짜증" (Pain #4의 evidence_quote 참조)
    - 근본 원인: Cafe24의 SNS 광고 연동 기능이 Instagram/Facebook 광고 설정만 지원할 뿐, 실제 광고 운영(A/B 테스트, 타겟팅 최적화, 성과 분석)은 수동입니다. 카페/네일숍 사장이 기대하는 "자동으로 광고해주는" 수준의 지능화가 부족합니다.
    - 공략 기회: 우리는 "AI 기반 광고 최적화"를 제공하여, Cafe24 사용자도 우리 서비스에 유입시킬 수 있습니다. 특히 "네이버플레이스 광고 + 인스타 광고를 동시에 최적화"하는 기능은 Cafe24에 없습니다.

  2. 불만: "리뷰 하나하나 관리하느라 피곤" (Pain #2의 evidence_quote 참조)
    - 근본 원인: Cafe24의 네이버플레이스 연동은 단순 동기화만 지원합니다. 리뷰 관리, 평점 분석, 자동 응답 같은 고급 기능이 없습니다. 네이버플레이스는 Cafe24와는 별개 플랫폼이므로, 사용자가 양쪽을 모두 관리해야 합니다.
    - 공략 기회: 우리의 "네이버플레이스 리뷰 자동 응답 + 평점 관리"는 Cafe24 사용자들이 필요로 하는 정확한 기능입니다. Cafe24에서 온라인 판매를 주로 하다가, 오프라인 고객(지역 카페 고객)도 관리하려는 사용자가 전환할 수 있습니다.

  3. 불만: "첫 상품 등록 후 실제 매출까지 2-3주 걸리는데, 그 동안 아무도 도와주지 않음" (추정: 온보딩 격차)
    - 근본 원인: Cafe24는 엔터프라이즈 솔루션이라 신규 사용자를 위한 전담 온보딩 담당자가 없습니다. 커뮤니티 또는 이메일로만 지원하므로, 응답 시간이 24-48시간입니다. 소상공인은 "지금 당장 도움이 필요"한데, Cafe24의 지원 속도는 너무 느립니다.
    - 공략 기회: 우리는 "구독형 리테이너 서비스"이므로 전담 계정 매니저 또는 AI 챗봇으로 24시간 실시간 지원을 제공할 수 있습니다. Cafe24 사용자가 "왜 우리는 아무도 도와주지 않는가?"에 대한 불만을 해결할 수 있습니다.

- 구조적 이유 1: Cafe24는 쇼핑몰 플랫폼이므로, 사용자가 자신의 매출로 ROI를 측정합니다. 온라인 거래액이 커질수록 Cafe24의 수수료도 증가합니다. 반면, 오프라인 카페/식당 고객은 온라인 거래를 하지 않으므로 Cafe24의 핵심 수익원(결제 수수료)이 발생하지 않습니다. 따라서 Cafe24는 이들을 타겟하지 않습니다.
- 구조적 이유 2: Cafe24의 기술팀은 "결제 연동 / 배송 통합 / 인벤토리 관리"라는 전자상거래 기술에 최적화되어 있습니다. "소셜 미디어 콘텐츠 제작 자동화 + 로컬 마케팅"은 완전히 다른 도메인의 기술이므로, 내재화하기 어렵습니다. 외부 파트너 통합을 시도하지만, 이는 Cafe24의 "올인원 솔루션" 가치 제안을 약화시킵니다.

우리 리안 컴퍼니가 이 약점을 어떻게 공략할 것인가?:
- 자산 활용: 우리는 이미 naver-diagnosis (네이버플레이스 진단 → 영업 훅) 기능이 있으므로, Cafe24 사용자도 "네이버플레이스 진단"을 통해 유입시킬 수 있습니다. 진단 결과에서 "리뷰 관리 문제 발견" → "월 49만원 집중 플랜 제안"으로 전환 가능합니다.
- 차별성: Cafe24는 온라인 거래 중심이므로 오프라인 매장 운영(카페, 네일숍, 식당)에는 최적화되지 않았습니다. 우리는 "오프라인 자영업자 전문" 대행사로서, Cafe24의 한계를 직접 해결하는 포지셔닝을 할 수 있습니다. 특히 "인스타 + 네이버플레이스 + 블로그"의 통합 마케팅은 Cafe24가 절대 제공하지 못합니다.
- AI 파이프라인: 우리의 AI 콘텐츠 생성 + 광고 최적화는 Cafe24의 SNS 광고 연동 기능을 실질적으로 고도화합니다. Cafe24는 "광고 플랫폼 연결"만 하고, 우리는 "광고 운영까지 자동화"합니다.
