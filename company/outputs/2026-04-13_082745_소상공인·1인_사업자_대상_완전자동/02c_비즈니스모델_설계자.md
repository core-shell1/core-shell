## [1] Problem Statement

**"스마트스토어 셀러와 소상공인은 월 80만 원 이상의 광고비를 네이버·인스타그램·구글에 분산 투입하는 상황에서 실제 어느 채널이 얼마의 매출을 발생시켰는지 정확히 측정할 수 없어 광고비를 낭비하고 있다. 이는 채널별 매출 귀속(attribution) 기술이 부재하고 각 플랫폼의 성과 데이터가 분산되어 있기 때문이며, 현재 엑셀 수동 합산(월 25시간 소모)이나 외부 컨설턴트 고용(월 25만 원)으로 임시방편을 쓰고 있어 시간·비용·기회 손실이 월 평균 32만 원 발생한다."**

**근거**:
- Pain #5 evidence_quote: "정확한 성과를 알기가 어렵다. 실제 매출이 얼마나 발생했는지 측정하는 기술이 없어" (workaround_cost 월 25만 원 외주)
- Pain #24 evidence_quote: "소상공인 마케팅 돕고 광고사기 막는다" (workaround_cost 월 30만 원 광고 사기 손실)
- Pain #9 evidence_quote: "소상공인의 74.1%는 플랫폼을 통한 매출이 전체의 절반 이상" (workaround_cost 월 20만 원 외주 A/B 테스트)
- Pain #31 evidence_quote: "연매출 10억원이었던 곳이 ... 800억원으로 껑충" (성과 측정 투명성이 매출 증가 직결 증명, workaround_cost 월 100만 원 실험 비용)

## [3] Value Proposition Canvas

### 고객 프로필 (오른쪽)

**Customer Jobs** (3가지 층)
- **기능적 Jobs**: 
  - 네이버·인스타·구글 광고 성과를 한눈에 확인해 월 광고비 80만 원 중 효율 낮은 채널 10만 원 삭감
  - 어느 상품이 어느 채널에서 잘 팔리는지 파악해 재고 발주 최적화
  - 광고 사기 클릭(봇 트래픽) 탐지해 월 30만 원 손실 방지
- **감정적 Jobs**: 
  - "내가 쓴 광고비가 헛되지 않았다"는 안도감
  - "경쟁 셀러보다 데이터 기반으로 똑똑하게 운영한다"는 자부심
  - 엑셀 수작업 월 25시간에서 해방되는 홀가분함
- **사회적 Jobs**: 
  - 스마트스토어 커뮤니티에서 "성과 좋은 셀러"로 인정받고 싶음
  - 가족·지인에게 "사업 잘 되고 있다" 증명하고 싶음 (매출 증가 그래프 공유)

**Pains** (반드시 서윤 Pain # 인용)
- **P1**: (Pain #5) "성과 추적 대시보드가 없어 엑셀로 매출 연동 수작업" — 월 25시간 소모, 오류 잦음, 실시간성 없음
- **P2**: (Pain #24) "이커머스 광고 사기 피해" — 봇 클릭으로 월 30만 원 손실, 탐지 늦음
- **P3**: (Pain #9) "온라인 플랫폼 매출 비중 높으나 마케팅 최적화 못함" — 74.1% 플랫폼 의존하나 채널별 기여도 불명, 수동 A/B 테스트 월 20만 원 외주
- **P4**: (Pain #18) "마케팅 ROI 개선 어려워 예산 낭비" — 네이버 광고 수동 조정, 데이터 기반 아님, 월 40만 원 손실
- **P5**: (Pain #31) "매출 10억→800억 사례 있지만 재현 어려움" — 성과 측정 투명성 부재로 성공 요인 파악 불가, 월 100만 원 실험 비용

**Gains**
- **G1**: 광고비 대비 매출(ROAS) 실시간 확인으로 월 광고비 10% 절감 (80만 원 → 72만 원, 연 96만 원 절약)
- **G2**: 채널별 매출 기여도 파악으로 효율 높은 채널 집중 투자 → 월 매출 15% 증가 (500만 원 → 575만 원, 연 900만 원 증가)
- **G3**: 엑셀 수작업 월 25시간 → 1시간으로 단축 (시급 1만 원 기준 월 24만 원 절약)
- **G4**: 광고 사기 탐지로 월 30만 원 손실 방지
- **G5**: 데이터 기반 의사결정으로 "똑똑한 셀러" 이미지 구축 (커뮤니티 인정)

### 가치 맵 (왼쪽)

**Products & Services**
- 네이버 쇼핑·인스타그램·구글 애즈 3개 채널 API 자동 연동
- 실시간 성과 대시보드 (1분 단위 업데이트, 매출·ROAS·채널별 기여도 3개 핵심 지표)
- UTM 자동 태깅 (매출 귀속 추적, 상품별·채널별 분리)
- 광고 사기 탐지 알고리즘 (봇 트래픽 필터링)
- 모바일 앱 (iOS/Android, 외출 중 실시간 확인)
- 주간 성과 리포트 이메일 자동 발송

**Pain Relievers** (각 Pain에 1:1 대응)
- **P1 → 엑셀 수작업 제거**: API 자동 연동으로 데이터 수집·합산 자동화, 월 25시간 → 1시간 (96% 절감)
- **P2 → 광고 사기 탐지**: 봇 트래픽 필터링 알고리즘으로 월 30만 원 손실 방지, 실시간 알림
- **P3 → 채널별 최적화**: 채널별 매출 기여도 자동 계산, A/B 테스트 불필요 (월 20만 원 외주 절감)
- **P4 → ROI 자동 개선**: ROAS 실시간 확인 + AI 인사이트 (효율 낮은 광고 자동 추천 중단), 월 40만 원 손실 방지
- **P5 → 성공 재현 가능**: 매출 증가 요인 데이터 기반 파악 (어느 상품·채널·시간대가 효과적인지), 월 100만 원 실험 비용 절감 (MVP 범위 밖 — 중기 AI 인사이트 기능 추가 시 대응)

**Gain Creators**
- **G1 → 광고비 절감**: ROAS 실시간 모니터링으로 효율 낮은 채널 즉시 중단, 월 8만 원 절감
- **G2 → 매출 증가**: 채널별 기여도 파악 후 효율 높은 채널 집중 투자, 월 매출 15% 증가 (75만 원 증가)
- **G3 → 시간 절약**: 엑셀 수작업 96% 단축, 월 24만 원 절약
- **G4 → 손실 방지**: 광고 사기 탐지로 월 30만 원 손실 방지
- **G5 → 사회적 인정**: 데이터 기반 운영으로 커뮤니티 내 "성과 좋은 셀러" 포지셔닝

## [4] 공감 지도 (Empathy Map)

**페르소나**: 스마트스토어 패션 셀러 김민지 (32세, 여성, 월 매출 500만 원, 광고비 월 80만 원)

- **Think & Feel** (속으로 무슨 생각/느낌):
  - "광고비 80만 원 쓰는데 진짜 효과 있는 건지 모르겠어. 매출은 늘었는데 광고 덕분인지 계절 탓인지..."
  - "엑셀로 정리하는 거 너무 귀찮아. 실수하면 숫자 틀리고, 그럼 또 처음부터..."
  - "경쟁 셀러들은 다 잘하는 것 같은데 나만 뒤처지는 느낌. 데이터 분석 못하면 망하는 거 아냐?"
  - "광고 사기 당했다는 얘기 커뮤니티에서 봤는데 나도 당하고 있는 건 아닐까? 불안해."

- **See** (주변에서 뭘 보는가):
  - 네이버 카페에서 "월 매출 1억 달성" 후기 글 (부러움 + 압박감)
  - 유튜브에서 "스마트스토어 광고 최적화" 튜토리얼 (따라하기 어려움)
  - 경쟁 셀러들이 인스타 릴스로 바이럴 성공 (나도 해야 하나 고민)
  - 스토어링크 광고 ("10만 셀러 이용 중" — 나도 써야 하나?)
  - 네이버 쇼핑 알림 "광고 예산 80% 소진" (초조함)

- **Hear** (동료/상사/고객/가족한테 뭘 듣는가):
  - 커뮤니티 동료: "요즘 광고비 너무 비싸. ROAS 3 넘기기 힘들어."
  - 남편: "광고비 그렇게 쓰는데 매출 얼마나 늘었어? 본전은 뽑았어?"
  - 고객 리뷰: "배송 빠르고 좋아요!" (광고 효과인지 모름)
  - 유튜브 광고: "AI 마케팅 자동화로 매출 3배!" (의심스럽지만 궁금)
  - 네이버 쇼핑 담당자: "광고 집행 늘리면 노출 더 올라갈 거예요." (믿어야 하나?)

- **Say & Do** (겉으로 뭘 하는가):
  - 매일 아침 네이버 쇼핑·인스타·구글 애즈 각각 로그인해서 성과 확인 (30분 소요)
  - 엑셀에 채널별 광고비·클릭·매출 수동 입력 (주 1회, 2시간)
  - 네이버 카페에 "광고 효율 어떻게 올리나요?" 질문 글 작성 (답변 애매)
  - 유튜브 튜토리얼 보면서 광고 설정 따라하기 (이해 안 되면 포기)
  - 남편한테 "이번 달 매출 좀 올랐어!" (실제 광고 덕분인지 확신 없음)

- **Pain** (가장 큰 두려움·좌절):
  - "광고비만 날리고 매출 안 늘면 어쩌지? 적자 나면 사업 접어야 하나..."
  - "경쟁 셀러들은 다 데이터 분석 잘하는데 나만 못하는 것 같아. 뒤처지면 망해."
  - "광고 사기 당해도 모르고 계속 돈 쓰고 있는 건 아닐까? 누가 확인해줄 사람 없어."

## [9] Flywheel + Engine of Growth

### Flywheel 초안 (6개 요소)

```
1. 스마트스토어 셀러 유입 (네이버 카페·유튜브)
     ↓
2. 무료 체험 (채널 1개 연동, 첫 주 매출 귀속 확인)
     ↓
3. 아하 모먼트 ("이게 자동으로?!" — 엑셀 수작업 해방)
     ↓
4. 유료 전환 (Starter 월 49만 원, 채널 3개 확장)
     ↓
5. 성과 증명 (월 매출 15% 증가 체감 → 커뮤니티 후기 작성)
     ↓
6. 바이럴 확산 (후기 보고 신규 유입 증가)
     ↑
1. 스마트스토어 셀러 유입 (선순환 완성)
```

**각 연결고리 작동 원리 1줄**:
- **1→2**: 네이버 카페 게시글 "무료 베타 테스터 모집" 클릭 → 랜딩 페이지 가입 (이메일 1분) → 즉시 네이버 쇼핑 API 연동 유도
- **2→3**: 첫 주 매출 귀속 확인 (UTM 태깅으로 "인스타 광고로 50만 원 매출 발생" 표시) → "엑셀 안 해도 되네!" 체감
- **3→4**: 2주 차 인스타·구글 연동 시도 시 "Starter 플랜 필요" 팝업 → 월 49만 원 결제 (광고비 대비 33% 수준으로 합리적)
- **4→5**: Starter 사용 1개월 후 채널별 매출 기여도 확인 → 효율 낮은 채널 중단 (광고비 10% 절감) + 효율 높은 채널 집중 (매출 15% 증가) → 커뮤니티 "월 매출 75만 원 증가!" 후기 작성
- **5→6**: 네이버 카페 후기 게시글 조회수 5,000+ → 댓글 "나도 써봐야겠다" 50개 → 신규 유입 50명 증가 (바이럴 계수 1.5)
- **6→1**: 신규 유입 50명 중 무료 체험 전환율 80% (40명) → 1번 단계 재진입 (선순환 완성)
