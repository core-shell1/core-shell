| **타겟이 모이는 곳** | 네이버 카페 "스마트스토어 판매자 모임" (회원 23만+) / 네이버 카페 "소상공인 마케팅 연구소" / 인스타 해시태그 #스마트스토어창업 #뷰티샵원장 / 오픈카톡 "스토어 운영자 모임" |

## 2. 경쟁사 벤치마크

| 경쟁사 | 상품 | 가격 | 우리 대비 약점 |
|--------|------|------|--------------|
| **PostDot** | 블로그 콘텐츠 자동 생성·발행 (네이버·티스토리 등 4개 플랫폼) | 월 55,000원 | 콘텐츠 생성만 됨. 진단·전략·성과추적 없음. SNS 배포 없음 |
| **Mirra** | SNS 콘텐츠 스케줄링 + AI 숏폼 자동 생성 | 무료~월 $19 (약 2.5만원) | 한국 특화 약함. 네이버 블로그·스마트스토어 연동 없음 |
| **Buffer** | SNS 예약 발행 (인스타·페북·링크드인) | 무료~채널당 월 $6 | 콘텐츠 생성 없음. 단순 스케줄러. 한국 채널 미지원 |
| **뤼튼** | AI 글쓰기 도구 (범용) | 무료~유료 | 마케팅 특화 아님. 배포·성과추적 없음. 소상공인 워크플로우 없음 |
| **채널톡** | AI 챗봇 + 고객 응대 자동화 | 월 3~10만원대 | 마케팅 콘텐츠 생성·배포 없음. 인바운드 응대에 국한 |
| **일반 마케팅 대행사** | SNS 운영 대행 (월 포스팅 8~12개) | 월 30~80만원 | 비쌈. 느림. 수정 번거로움. 성과 불투명. 사장님 개입 필요 |

**결론**: 진단 → 전략 → 콘텐츠 생성 → 배포 → 성과추적을 **하나의 플로우**로 묶은 경쟁사 없음. 특히 **네이버 생태계 특화 + 원스톱**은 현재 블루오션.

## 3. 우리 상품

| 플랜 | 가격 | 포함 내용 | 타겟 등급 |
|------|------|---------|----------|
| **스타터** (구 주목) | 월 29,000원 | ① 업체 정보 입력 → AI 마케팅 진단 리포트 1회/월 ② 네이버 블로그 포스팅 AI 초안 4개/월 ③ 기본 성과 대시보드 (조회수·유입) | 막 시작한 셀러 / 뷰티샵 원장 / "일단 써보자" 층 |
| **그로스** (구 집중) | 월 59,000원 | 스타터 전부 + ① 블로그 포스팅 12개/월 자동 발행 ② 인스타 피드·릴스 캡션 자동 생성 8개/월 ③ 채널 2개 자동 배포 ④ 주간 성과 리포트 자동 발송 | 월매출 500만+ 스마트스토어 / 인스타 운영 중인 뷰티샵 |
| **풀오토** (구 시선) | 월 99,000원 | 그로스 전부 + ① 블로그+인스타+스마트스토어 상세페이지 콘텐츠 무제한 ② 채널 4개 자동 배포 ③ 경쟁사 모니터링 ④ 월 1회 전략 리포트 + 개선 제안 ⑤ 카카오톡 알림 연동 | 월매출 1,000만+ / 마케팅에 진심인 셀러 / 확장 준비 중 |
| **베타 특가** | 월 19,000원 (3개월 한정) | 그로스 플랜 동일 기능 / 피드백 제공 조건 | 얼리어답터 100명 한정 모집용 |

> **가격 근거**: PostDot 55,000원보다 낮게 시작해서 진입장벽 낮춤 → 그로스에서 기능 체감 → 풀오토 업셀. 대행사 대비 1/5 가격에 "자동화"라는 차별점.

## 4. 영업 채널

| 채널 | 자동화 가능? | 법적 제한 | 우선순위 |
|------|------------|---------|---------|
| **네이버 카페 DM·댓글** | 반자동 (템플릿 작성 후 수동 발송) | 스팸성 홍보 시 카페 강퇴·계정 제재 위험. 카페 운영진 허락 필요 | ★★★★★ 1순위 |
| **인스타 DM** | 반자동 (ManyChat 등 활용) | Meta 정책상 대량 자동 DM 계정 정지 위험. 하루 20~30건 수동 권장 | ★★★★☆ 2순위 |
| **카카오톡 채널** | 자동화 가능 (알림톡 API) | 수신 동의 없는 발송 불법 (정보통신망법). 반드시 옵트인 필요 | ★★★☆☆ 3순위 (DB 확보 후) |
| **이메일 콜드메일** | 자동화 가능 (Mailchimp·Stibee) | 수신 동의 없는 발송 스팸법 위반. 옵트인 DB 필요 | ★★★☆☆ 3순위 (DB 확보 후) |
| **네이버 블로그 체험단 모집** | 반자동 | 제한 없음. 단 허위 후기 작성 강요 시 공정위 제재 | ★★★★☆ 2순위 |
| **유튜브·인스타 광고** | 완전 자동화 가능 | 광고비 필요. 허위·과장 광고 심의 주의 | ★★★☆☆ 예산 확보 후 |
| **오픈카톡 커뮤니티 참여** | 수동 | 홍보 목적 입장 시 강퇴 위험. 가치 제공 먼저 | ★★★★☆ 2순위 |

> **핵심 전략**: 첫 2주는 네이버 카페 + 오픈카톡 **수동 영업**으로 베타 유저 30명 확보. 자동화는 DB 쌓인 후.
