# 토지 사업성 분석 자동화 SaaS — 런칭 준비 구체화

> **시은 노트**: 이사팀 GO 판정 기준으로 작성. 현재 Pain Point Level 4~5 실증 데이터 부족이 가장 큰 리스크 — 이걸 첫 주에 해결하는 구조로 짰음.

---

## 1. 타겟

| 항목 | 내용 |
|------|------|
| **1순위 타겟** | 연간 1~5건 토지 매입·개발 진행하는 **중소 시행사 실무 담당자** (직원 5~30인, 서울·수도권 소재) |
| **2순위 타겟** | 건축사무소 소장 또는 인허가 담당자 — 고객사 제안용 사전 검토 리포트 필요 |
| **3순위 타겟** | 부동산 개발업체 심사팀 — 다수 토지 동시 스크리닝 필요 |
| **얼리어답터 특성** | 현재 외주 컨설팅(건당 100~300만 원)에 돈 쓰고 있는 곳 / 엑셀·수작업으로 3~7일 소요 경험 있는 곳 / 스타트업 툴 거부감 없는 40대 이하 실무자 |
| **타겟이 모이는 곳** | 네이버 카페 "부동산 스터디", "시행사 실무자 모임", 오픈카톡 시행사·시공사 네트워크, 건축사협회 교육 세미나, 부동산 개발 관련 유튜브 댓글 |
| **예상 TAM** | 국내 시행사 약 3,000~5,000개사 + 건축사무소 약 8,000개소 → 잠재 고객 약 1만 개사 / 월 구독 30만 원 기준 TAM 약 360억 원/년 |
| **현실적 SAM (1년)** | 수도권 중소 시행사·건축사무소 500~1,000개사 → MRR 목표 500만 원 = 약 17~20개사 확보 필요 |

## 2. 경쟁사 벤치마크

| 경쟁사 | 상품 | 가격 (추정) | 우리 대비 약점 |
|--------|------|------------|--------------|
| **밸류맵** | 토지 시세 조회, 실거래가 분석, VDN AI 에이전트(2026 출시 예정) | 무료~월 10~30만 원 구간 추정 | 시세 조회 중심 — **승인 확률·특례법 판단·인허가 가능성 분석 없음** |
| **빅밸류** | AVM 기반 토지 가치 산정, 딥러닝 시세 모델 | B2B 커스텀 계약 (건당·월정액 혼합) | 금융기관·대형사 타겟 — **중소 시행사 접근성 낮음, UI 복잡** |
| **디스코(DISCO)** | 토지 정보 조회, 용도지역 확인, 건축 가능 면적 계산 | 무료~월 5만 원 이하 | 정보 조회 수준 — **사업성 판단·확률 산출 없음** |
| **건축 인허가 컨설팅 업체** | 수작업 사업성 검토 리포트 | 건당 100~500만 원 | 납기 3~14일, 비용 고정 — **반복 사용 불가, 스케일 안 됨** |
| **스페이스워크** | AI 건축 설계 자동화 (건물 배치·용적률 시뮬레이션) | 월 30~100만 원 구간 추정 | 설계 자동화 중심 — **인허가 승인 확률·정책 흐름 분석 없음** |

> **공백 확인**: 개요서·법률서 업로드 → 승인 확률 + 특례법 해당 여부 + 정책 흐름 **원스톱 자동 산출**하는 제품은 현재 국내 없음. 이게 우리 포지션.

## 4. 영업 채널

| 채널 | 자동화 가능? | 법적 제한 | 우선순위 |
|------|------------|---------|---------|
| **네이버 카페 DM / 댓글 참여** | 반자동 (템플릿 작성 후 수동 발송) | 스팸성 대량 DM 금지 — 커뮤니티 규칙 확인 필수. 1:1 자연스러운 접근만 가능 | ★★★★★ 1순위 |
| **콜드 이메일 (시행사·건축사무소 직접)** | 자동화 가능 (Apollo, Instantly 등) | 수신 동의 없는 영리 이메일 → 정보통신망법 위반 가능. **반드시 수신 동의 확보 후 발송** 또는 링크드인 InMail 활용 | ★★★★ 2순위 (법적 주의) |
| **링크드인 아웃리치** | 반자동 (Sales Navigator + 템플릿) | 국내 부동산 개발 종사자 링크드인 활동 낮음 — 효율 제한적 | ★★★ 3순위 |
| **건축사협회·시행사 세미나 스폰서/발표** | 수동 | 없음 | ★★★★ 2순위 (신뢰도 최고) |
| **유튜브 댓글 / 부동산 개발 크리에이터 협업** | 수동 | 없음 | ★★★ 3순위 (중장기) |
| **네이버 검색광고 (키워드: 토지 사업성 분석, 인허가 검토)** | 자동화 가능 | 없음 | ★★★★ 2순위 (예산 필요) |

> **현실 조언**: 첫 20개사는 자동화 말고 **발로 뛰어야** 함. 카페 직접 참여 + 세미나 1회 = 파일럿 고객 확보 최단 경로.
