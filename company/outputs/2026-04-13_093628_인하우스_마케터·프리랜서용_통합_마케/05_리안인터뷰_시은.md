# 추론된 리안 워크플로우:

1. **클라이언트 온보딩 → 채널 접근권 수집**: 소상공인·스마트스토어 셀러 신규 계약 시 GA4·메타·네이버SA 계정 공유받아 수동으로 각 플랫폼 로그인 후 데이터 확인
2. **주간 데이터 취합**: 채널별 플랫폼을 개별 접속해 수치 복사 → 엑셀/스프레드시트에 수동 정리 (추정 소요: 클라이언트당 1~2시간)
3. **성과 분석 → 액션 도출**: 취합된 데이터 보고 "이번 주 뭐가 문제인지" 직접 판단 → 패키지(주목/집중/시선)별 대응 액션 결정
4. **클라이언트 보고**: 정리된 수치 기반으로 카카오톡·이메일·보고서 형태로 주간 리포트 전달 (수작업 포맷팅 포함)
5. **반복 실행**: 위 사이클을 다수 클라이언트에 동시 반복 → 클라이언트 수 늘수록 취합·보고 공수가 선형으
