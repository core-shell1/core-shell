## 2. 경쟁사 벤치마크

| 경쟁사 | 상품 | 가격 (월) | 우리 대비 약점 |
|--------|------|-----------|--------------|
| **Adriel** | 멀티채널 광고 대시보드 + AI 인사이트 | $5,000~$15,000 | 네이버SA·카카오 미지원 / 가격이 중소기업·프리랜서 접근 불가 수준 |
| **Looker Studio** | 구글 데이터 시각화 (무료) | 무료 (커스텀 커넥터 유료) | 네이버SA 공식 커넥터 없음 / AI 인사이트 없음 / 세팅에 개발 지식 필요 |
| **Supermetrics** | 멀티채널 데이터 수집 → BI 연동 | $69~$599 | 네이버SA 미지원 / 대시보드 없음 (데이터 파이프라인만) / AI 없음 |
| **Funnel.io** | 광고 데이터 통합 + 리포팅 | $400~$1,500 | 네이버SA 미지원 / 한국어 지원 없음 / 중소기업엔 과잉 |
| **국내 대행사 리포트** | 월간 PDF 리포트 수작업 제공 | 30만~100만 원/월 | 실시간 아님 / 인사이트 없음 / 클라이언트가 직접 볼 수 없음 |
| **엑셀+수작업** | 현재 대부분의 인하우스·프리랜서 현실 | 0원 (시간 비용 월 20~40시간) | 자동화 없음 / 실시간 아님 / 인사이트 없음 |

> **핵심 공백**: $100 이하 + 네이버SA 지원 + AI 인사이트 = **현재 이 조합을 제공하는 툴이 없음**

## 4. 영업 채널

| 채널 | 자동화 가능? | 법적 제한 | 우선순위 | 실행 방법 |
|------|------------|---------|---------|---------|
| **커뮤니티 직접 참여** (네이버 카페, 오카방) | ❌ 수동 필수 | 스팸 금지 / 카페 규정 준수 필요 | ⭐⭐⭐⭐⭐ 1순위 | 질문 글에 진짜 답변 달면서 자연스럽게 툴 언급 / 주 3회 |
| **크몽·숨고 프리랜서 DM** | ❌ 수동 | 플랫폼 외부 유도 금지 조항 확인 필요 | ⭐⭐⭐⭐ 2순위 | "리포트 자동화 툴 베타 테스터 모집" 메시지 / 하루 10~20명 |
| **링크드인 아웃리치** | 🔶 반자동 (템플릿+수동 발송) | 1촌 외 메시지 제한 (InMail 크레딧 필요) | ⭐⭐⭐ 3순위 | 마케터 직군 필터 → 연결 요청 + 짧은 메시지 / 하루 20명 |
| **인스타 DM** | 🔶 반자동 | 자동 DM 계정 정지 위험 — **수동만** | ⭐⭐⭐ 3순위 | #퍼포먼스마케터 #인하우스마케터 팔로워 DM / 하루 10명 |
| **콜드 이메일** | ✅ 자동화 가능 (Instantly, Lemlist) | 수신 동의 없는 발송 = 정보통신망법 위반 위험 — **B2B 기업 이메일만, 옵트인 필수** | ⭐⭐ 4순위 | 중소기업 마케팅 담당자 이메일 수집 → 시퀀스 3단계 |
| **유료 광고 (메타/구글)** | ✅ 자동화 | 없음 | ⭐⭐ 4순위 | 베타 후기 확보 후 집행 — 지금은 시기상조 |

> **법적 주의**: 국내 콜드 DM/이메일은 정보통신망법상 사전 동의 없으면 스팸. B2B 공개 이메일은 그레이존이나 리스크 있음. **커뮤니티 + 크몽 직접 접근이 가장 안전하고 효과적.**
