**마케팅 통합 대시보드 SaaS**를 시작하면 이것도 필요해:

---

🔴 **지금 당장 필요 (없으면 손해)**

- **혜경님 프로젝트와 즉시 통합 회의**: 같은 타겟(인하우스+프리랜서)인데 두 프로젝트가 따로 돌면 리소스 낭비 — 지금 당장 "같은 거냐 다른 거냐" 결론 내야 함
- **네이버SA API 접근 권한 확인**: 네이버 검색광고 API는 승인제 — 개발 시작 전에 신청 안 하면 3~4주 날림
- **베타 유저 10명 확보 (선약속)**: 리안 컴퍼니 기존 고객사 중 마케터 있는 곳 바로 컨택 — 빌드 전에 "쓸게요" 받아야 방향 안 흔들림
- **랜딩페이지 (최소 1장)**: 구독 대기자 이메일 수집용 — 없으면 GO 판정 이후 관심 증발

---

🟡 **곧 필요 (초기 3개월 내)**

- **GA4 / 메타 API 연동 개발자**: 리안은 비개발자 — 외주든 코파운더든 기술 파트너 없으면 SaaS 자체가 안 만들어짐
- **프라이싱 페이지 + 결제 연동 (Stripe)**: 49달러 받을 인프라 — 베타
