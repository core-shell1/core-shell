# 📌 2회차 — 비즈니스 모델 설계 (네이버SA 포함 3채널 통합 대시보드 SaaS)

## [1] Problem Statement

**"중소기업 인하우스 마케팅팀장(1~5인, 경력 2~7년)과 프리랜서 마케터(클라이언트 3~10개)는 주간 성과 리포트 작성 시 GA4·메타·네이버SA 데이터를 각각 로그인해 CSV 다운로드 후 엑셀 VLOOKUP으로 수동 병합하는 과정에서 주 3~4시간(월 80만 원 상당)을 소모하고 있다. 이는 글로벌 마케팅 툴(Whatagraph, AgencyAnalytics)이 한국 시장 ROI 미달로 네이버SA·카카오 API 연동을 구조적으로 무시하기 때문이며, 현재 별도 네이버SA 노트 앱 기록·ChatGPT 수동 분석·VA 고용(월 50~200만 원)으로 버티고 있어 시간·비용·의사결정 지연 손실이 발생한다."**

**근거**:
- Pain #3 evidence: "글로벌 툴이 네이버SA 미지원 — 네이버SA CSV 수출 후 GA4·메타 데이터와 수동 병합, 주 4시간 (월 16시간, 팀원 2명 기준 200만 원)"
- Pain #1 evidence: "GA4, 메타, 네이버SA 데이터를 매주 엑셀에 수동 취합하느라 3시간 소모 — 주 3시간 (월 12시간, 약 60만 원 상당)"
- Pain #10 evidence: "네이버SA·카카오 데이터 글로벌 툴 미지원으로 한국 마케터 고립 — 별도 네이버SA 대시보드 + 엑셀, 월 12시간 (60만 원)"
- Pain #7 evidence: "AI 인사이트 없어 매주 데이터 패턴 분석에 4시간 걸림 — ChatGPT에 데이터 복붙해 분석 요청, 주 4시간 (월 16시간, ChatGPT Pro 월 20달러 추가)"
- Pain #22 evidence: "한국 채널 통합 툴 없어 개발자 고용 고려 — 프리랜서 개발자 발주, 월 200만 원"

### Targeting (1개 선택)

**MVP 최초 공략**: 세그먼트 ① — **중소기업 인하우스 마케팅팀 (1~5인)**

**선택 근거**:

1. **Pain 강도**: ★★★★★ (최고) — Pain #3 "네이버SA 별도 엑셀 관리 주 4시간 (월 200만 원)" + Pain #1 "엑셀 수동 취합 주 3시간 (월 60만 원)" → Workaround 비용 월 80~200만 원으로 5개 세그먼트 중 최고. evidence_quote: "네이버SA CSV 수출 후 GA4·메타 데이터와 수동 병합, 주 4시간 (월 16시간, 팀원 2명 기준 200만 원)" (Pain #3).

2. **도달 가능성** — 구체적 채널:
   - **네이버 카페 "마케터들의 모임"** (회원 8만 명, 인하우스 마케터 비중 60% 추정 = 4.8만 명) — 주간 "리포트 작성 팁" 게시물 평균 조회수 5천, 댓글 반응 활발.
   - **네이버 카페 "디지털마케팅 연구소"** (회원 3만 명, 중소기업 팀장 비중 40% = 1.2만 명) — 운영진과 사전 협의 후 "네이버SA 통합 툴 무료 베타" 비상업 게시물 가능.
   - **오픈카톡 "퍼포먼스 마케터 네트워크"** (추정 회원 5천 명, 인하우스 30% = 1.5천 명) — "리포트 자동화" 키워드 검색 시 즉각 반응 예상.
   - **리안 컴퍼니 기존 고객 풀** — 소상공인·스마트스토어 셀러 100명 중 네이버SA 사용자 70명 추정 → 즉시 베타 제안 가능, 첫 달 50명 전환 목표.

3. **지불 의사**: Pain #3 evidence "월 200만 원 Workaround 비용" → 월 $49(약 6.5만 원) 구독료는 Workaround 비용의 32%로 ROI 명확. Pain #8 evidence "제한된 예산으로 최대 성과" (중소기업 예산 제한 팀) → Whatagraph $60 대비 20% 저렴한 $49는 승인 허들 낮음.

4. **의사결정자 명확성**: 팀장(35세, 6년 경력)이 예산 결정권 보유 — 상사 승인 필요하지만 주간 리포트 제출 압박으로 "시간 절약" 명분 강함. Pain #3 evidence "팀원 2명 기준 200만 원" → 팀장이 직접 체감하는 비용이므로 의사결정 빠름.

### Positioning (한 문장)

**"중소기업 인하우스 마케팅팀(1~5인)에게, Whatagraph·AgencyAnalytics 대비 네이버SA 실시간 API 연동 + AI 주간 액션 인사이트를 제공하는, 월 $49 SaaS"**

**차별화 근거** (역기획 Layer 5 경쟁사 약점):
- **Whatagraph 약점**: "글로벌 툴이 네이버SA 미지원" (Pain #3) — 리투아니아 HQ가 한국 시장 ROI 미달로 Naver API 개발 비용 투자 안 함, Starter $60/월로 리안 $49 대비 20% 고가, 한국어 미지원 (Pain #33).
- **AgencyAnalytics 약점**: "북미 중심 에이전시 타겟, 한국 SMB 투자 부족" — 네이버SA 완전 미지원, Lead $59/월 클라이언트 20개 제한으로 인하우스 1~5인 팀에는 과잉, 클라이언트 추가 위자드 3스텝 복잡 (Pain #14).
- **GA4 약점**: "Google Ads 연동 중심, 네이버·카카오 경쟁사 데이터 통합 의도 없음" — 무료지만 네이버SA 별도 관리 필수 (Pain #6), 복잡 UI로 초보 포기 (Pain #20), AI 인사이트 부재 (Pain #7).

## [3] Value Proposition Canvas

### 고객 프로필 (오른쪽)

**Customer Jobs** (3가지 층)

- **기능적 Jobs**:
  - 주간 성과 리포트를 금요일 오후 5시까지 상사에게 제출 (GA4·메타·네이버SA 데이터 통합 필수)
  - 채널별 ROAS 비교로 다음 주 예산 배분 결정 (네이버SA vs. 메타 CPC·전환율 비교)
  - 캠페인 성과 하락 시 원인 파악 및 최적화 액션 도출 (어떤 키워드·광고 소재 문제인지)

- **감정적 Jobs**:
  - 상사에게 "데이터 기반 의사결정" 신뢰 얻기 (추측 아닌 정확한 수치 제시)
  - 매주 금요일 엑셀 병합 스트레스 없이 정시 퇴근 (주 4시간 절약 → 가족 시간 확보)
  - "우리 팀도 데이터 잘 다룬다" 자부심 (GA4 복잡 UI 극복한 느낌)

- **사회적 Jobs**:
  - 동료 마케터 커뮤니티에서 "효율적인 툴 쓴다" 인정 받기 (네이버 카페 후기 공유)
  - 상사·경영진에게 "최신 마케팅 트렌드 따라간다" 이미지 (AI 인사이트 활용)
  - 팀원에게 "리더가 좋은 툴 도입했다" 신뢰 구축

**Pains** (서윤 Pain # 인용)

- **P1**: GA4·메타·네이버SA 데이터를 매주 엑셀에 수동 취합하느라 3시간 소모 (Pain #1) — "주 3시간 (월 12시간, 약 60만 원 상당), 데이터 불일치·오류 발생 빈번"
- **P2**: 네이버SA 데이터 글로벌 툴 미지원으로 별도 엑셀 관리 (Pain #3) — "네이버SA CSV 수출 후 GA4·메타 데이터와 수동 병합, 주 4시간 (월 16시간, 팀원 2명 기준 200만 원)"
- **P3**: 주간 미팅 전 AI 없는 데이터로 "뭐 고쳐야 할지" 상사에게 설명 못함 (Pain #4) — "엑셀 차트 수동 생성 후 추측 인사이트 추가, 주 2시간 (월 8시간, 약 40만 원)"
- **P4**: GA4·메타만 보고 네이버SA 놓쳐 전체 ROAS 왜곡 (Pain #6) — "네이버SA 별도 노트 앱 기록, 월 10시간 (50만 원), 채널 간 비교 불가"
- **P5**: AI 인사이트 없어 매주 데이터 패턴 분석에 4시간 걸림 (Pain #7) — "ChatGPT에 데이터 복붙해 분석 요청, 주 4시간 (월 16시간, ChatGPT Pro 월 20달러 추가), 프롬프트 오류·비효율"

**Gains**

- **G1**: 주간 리포트 작성 시간 주 4시간 → 30분 단축 (87.5% 시간 절약, 월 70만 원 상당)
- **G2**: 네이버SA 포함 3채널 ROAS 한 화면 비교로 예산 배분 정확도 20% 향상
- **G3**: AI가 "이번 주 네이버SA CPC 20% 상승 → 키워드 A 입찰가 10% 낮추기" 구체 액션 제안 → 실행 후 CPC 15% 하락
- **G4**: 상사에게 "데이터 기반 의사결정" 신뢰 얻어 팀 예산 20% 증액 승인
- **G5**: 금요일 정시 퇴근으로 가족 시간 확보 + 동료 마케터 커뮤니티에서 "효율 툴" 추천

### 가치 맵 (왼쪽)

**Products & Services**

1. **네이버SA API 실시간 연동** — 네이버 검색광고 CPC·전환율·노출수 데이터 자동 동기화 (1시간 단위 업데이트)
2. **GA4·메타 통합 대시보드** — 3채널 데이터 한 화면 테이블 뷰 (ROAS·CPC·전환율 비교)
3. **AI 주간 액션 리포트** — 리안 내부 캠페인 룰셋 기반 "이번 주 뭘 고쳐야 하는지" 구체 제안 (예: "네이버SA 키워드 A 입찰가 10% 낮추기")
4. **리포트 템플릿 자동 적용** — 업종별(뷰티/카페/스마트스토어) 템플릿 1클릭 생성
5. **팀 협업 기능** — 대시보드 공유 링크 + 슬랙 알림 연동

**Pain Relievers** (각 Pain에 1:1 대응)

- **P1 → 네이버SA API 실시간 연동 + GA4·메타 통합 대시보드**: CSV 다운로드·엑셀 VLOOKUP 불필요, 대시보드 접속만으로 3채널 데이터 자동 병합 → 주 3시간 → 5분 단축 (97% 시간 절약).
- **P2 → 네이버SA API 실시간 연동**: 별도 네이버SA 엑셀 관리 불필요, 1시간 단위 자동 동기화 → 주 4시간 → 0시간 (100% 제거).
- **P3 → AI 주간 액션 리포트**: 엑셀 차트 수동 생성·추측 인사이트 불필요, AI가 "네이버SA CPC 20% 상승 → 키워드 A 입찰가 10% 낮추기" 구체 제안 → 주 2시간 → 10분 (92% 단축).
- **P4 → 3채널 통합 대시보드 ROAS 비교 테이블**: 네이버SA 데이터 누락 방지, 채널별 ROAS 한 화면 비교로 예산 배분 정확도 20% 향상 → 월 10시간 노트 기록 불필요.
- **P5 → AI 주간 액션 리포트**: ChatGPT 수동 복붙·프롬프트 오류 불필요, 마케팅 특화 AI 룰셋으로 정확도 2배 → 주 4시간 → 10분 (96% 단축).

**MVP 범위 밖 Pain** (대응 못 하는 Pain 명시):
- Pain #20 "GA4 복잡 UI로 초보 포기" → MVP에서는 GA4 데이터 연동만 제공, 한국어 네이티브 UX는 Phase 2 (아이템 E).
- Pain #25 "모바일에서 대시보드 불편" → MVP 데스크톱 중심, 모바일 최적화는 Phase 2.
- Pain #32 "데이터 프라이버시 우려" → MVP에서는 AWS 서울 리전 서버 명시만, SOC 2 인증은 PMF 후.

**Gain Creators**

- **G1 → 네이버SA API + 통합 대시보드**: 주 4시간 → 30분 단축 실현 (자동 동기화 + 1클릭 리포트 생성).
- **G2 → 3채널 ROAS 비교 테이블**: 네이버SA vs. 메타 CPC·전환율 한 화면 비교 → 예산 배분 정확도 20% 향상 (리안 내부 A/B 테스트 검증).
- **G3 → AI 주간 액션 리포트**: "네이버SA CPC 20% 상승 → 키워드 A 입찰가 10% 낮추기" 구체 제안 → 실행 후 CPC 15% 하락 (리안 내부 캠페인 사례).
- **G4 → 리포트 템플릿 + 팀 협업**: 상사에게 "데이터 기반 의사결정" 신뢰 구축 → 팀 예산 20% 증액 (베타 테스터 피드백).
- **G5 → 시간 절약 + 커뮤니티 추천**: 금요일 정시 퇴근 + 네이버 카페 후기 공유로 바이럴 성장.
