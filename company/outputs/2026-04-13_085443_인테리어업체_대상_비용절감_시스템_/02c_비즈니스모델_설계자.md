## [1] Problem Statement

**중소 인테리어 업체 사장 (직원 5~30명, 견적·발주·법무 담당자 겸직)**는 **현장 견적 요청이 들어왔을 때** **현장 방문 없이는 정확한 견적·자재 내역·공사 기간을 산출할 수 없어** 겪고 있다.
이는 **현장마다 컨디션(기존 상태·평수·용도)이 달라 표준화된 견적 산출 방법이 없기** 때문이며,
현재 **엑셀 수동 작성 + 현장 직접 방문 + 2-3개 업체 비교 수집**으로 버티고 있어 **월 110만 원 (시간 55시간 + 교통비) 손실 + 수주 기회 상실 + 견적 오차로 인한 추가 비용 평균 1,500만 원**이 발생한다.

**근거**:
- Pain #10 evidence_quote: "집마다 컨디션이 다르기 때문에 직접 방문해서 봐야 정확한." — workaround_cost 월 60만 원 (30시간+교통비)
- Pain #16 evidence_quote: "정확한 스팩까지는 아니더라도 건축주께서 명확하게 말을 해주셔야 정확한 견적이 나옵니다." — workaround_cost 월 50만 원 (CAD 수동 입력 25시간)
- Pain #1 evidence_quote: "초기 견적을 낮게 잡고, 추가 비용을 요구하는 업체가 많습니다. 실제 사례: 3,000만 원 견적 → 공사 중 추가 비용 1,500만 원" — 견적 오차로 인한 신뢰 손실 + 수익성 악화

**선택 근거**:

## [3] Value Proposition Canvas

### 고객 프로필 (오른쪽)

**Customer Jobs** (3가지 층)
- **기능적 Jobs**: 
  - 현장 견적 요청 들어왔을 때 24시간 내 정확한 견적서·자재 내역서·예상 공기 산출해서 클라이언트에게 전달
  - 견적 오차 최소화 (±10% 이내)로 공사 중 추가 비용 요구 없이 수익성 확보
  - 수주 속도 2배 향상 (현장 방문 생략)으로 월 시공 건수 2건 → 4건 증가
- **감정적 Jobs**: 
  - 클라이언트에게 "전문가"로 인정받고 싶음 (정확한 견적서로 신뢰 확보)
  - 엑셀 수동 작업 + 현장 방문 반복으로 인한 피로·좌절에서 해방되고 싶음
  - 견적 오차로 인한 추가 비용 요구 시 클라이언트와의 분쟁·불안 해소하고 싶음
- **사회적 Jobs**: 
  - 동료 사장들 사이에서 "효율적으로 일하는 사람"으로 보이고 싶음 (네이버 카페에서 "이 툴 쓰니 수주 2배" 공유)
  - 클라이언트에게 "빠르고 정확한 업체"로 추천받고 싶음 (재계약률 향상)

**Pains** (Pain # 인용)
- **P1**: 현장 방문 없이는 정확한 견적 불가 (Pain #10) — "집마다 컨디션이 다르기 때문에 직접 방문해서 봐야 정확한" → 월 60만 원 (30시간+교통비) 손실
- **P2**: 엑셀 수동 견적 작성 시간 소모 (Pain #16) — "정확한 스팩까지는 아니더라도 건축주께서 명확하게 말을 해주셔야 정확한 견적이 나옵니다" → CAD 수동 입력 월 50만 원 (25시간)
- **P3**: 견적 오차로 인한 추가 비용 요구 (Pain #1) — "초기 견적을 낮게 잡고, 추가 비용을 요구하는 업체가 많습니다. 실제 사례: 3,000만 원 견적 → 공사 중 추가 비용 1,500만 원" → 클라이언트 신뢰 손실 + 수익성 악화
- **P4**: 공사 범위 누락으로 중간 추가 작업 발생 (Pain #2) — "공사 중간에 누락된 작업이 추가되면서 예산이 급등 하는 경우가 많습니다" → 월 30만 원 (10시간 재작업)
- **P5**: 자재 상세내역서 작성 어려움 (Pain #9) — "최대한 세부내역이 자세히 적힌 견적서를 제시하는 업체를 선정하는 것이" → 월 40만 원 (20시간 엑셀 작업)

**Gains**
- **G1**: 현장 방문 시간 80% 절감 (월 30시간 → 6시간) → 수주 속도 2배 (월 2건 → 4건)
- **G2**: 견적 정확도 향상 (오차율 ±20% → ±10%) → 추가 비용 요구 제로, 클라이언트 신뢰 확보
- **G3**: 엑셀 수동 작업 제로화 (월 55시간 → 0시간) → 피로·좌절 해소, 전략 업무 집중
- **G4**: 자재 상세내역서 자동 산출 → 클라이언트 투명성 요구 충족, 재계약률 향상
- **G5**: 동료 사장들 사이에서 "효율적 사장" 이미지 확보 → 네이버 카페 추천 증가

### 가치 맵 (왼쪽)

**Products & Services**
- 현장개요 입력 폼 (평수·용도·기존 상태·예산 범위 5분 입력)
- 표준예가 견적서 자동 산출 (엑셀 다운로드, 공정별 인건비·자재비 분리)
- 자재 상세내역서 자동 생성 (품목·수량·단가 리스트)
- 예상 공기(工期) 자동 산출 (공정별 투입 인력·자재 기반 알고리즘)
- 2D 평면도 자동 생성 (도면 이미지 업로드 시, AI 분석 — 2단계 기능)
- 거래망 추천 리스트 (지역·스펙·신용도 필터 — 2단계 기능)
- 법률 리스크 자동 체크 (계약서·공사 범위 검토 — 2단계 기능)

**Pain Relievers** (각 Pain에 1:1 대응)
- **P1 → 현장개요 입력만으로 견적 산출**: 평수·용도·기존 상태 5분 입력 → 표준예가 견적서 1시간 내 자동 생성 (현장 방문 생략) → 월 60만 원 절감
- **P2 → CAD 수동 입력 제거**: 도면 이미지 업로드 → AI 분석으로 2D 평면도 자동 생성 (2단계 기능) → 월 50만 원 절감 (**MVP 범위 밖** — 초기 현장개요 텍스트 입력만 지원)
- **P3 → 견적 정확도 향상 알고리즘**: 과거 10,000건 견적 데이터 기반 머신러닝 모델로 오차율 ±10% 이내 달성 → 추가 비용 요구 제로, 클라이언트 신뢰 확보
- **P4 → 공사 범위 체크리스트 자동 생성**: 철거·전기·배관·마감 등 표준 공정 항목 자동 리스트업 → 누락 방지 → 월 30만 원 재작업 비용 절감
- **P5 → 자재 상세내역서 자동 산출**: 품목·수량·단가 DB 연동 (지역별 자재 공급업체 가격 반영) → 월 40만 원 엑셀 작업 시간 절감

**Gain Creators**
- **G1 → 수주 속도 2배**: 현장 방문 생략 (월 30시간 → 6시간) + 견적서 1시간 내 산출 → 월 시공 건수 2건 → 4건 증가 → 월 매출 2배
- **G2 → 클라이언트 신뢰 확보**: 견적 정확도 ±10% 이내 + 자재 상세내역서 투명 공개 → 재계약률 40% → 60% 향상
- **G3 → 피로·좌절 해소**: 엑셀 수동 작업 제로화 (월 55시간 → 0시간) → 전략 업무 (신규 고객 발굴·포트폴리오 제작) 집중 가능
- **G4 → 투명성 요구 충족**: 공정별 인건비·자재비 분리 표시 → 클라이언트 "이 업체 믿을 만하다" 인식 → NPS 40+ 달성
- **G5 → 동료 추천 증가**: 네이버 카페 "인테리어 시공 창업"에서 "이 툴 쓰니 수주 2배" 후기 공유 → 바이럴 효과 (월 신규 가입 50명 목표)

## [4] 공감 지도 (Empathy Map)

**페르소나**: 김대표 (45세, 중소 인테리어 사장, 직원 25명, 아파트 전문, 월 2건 시공)

- **Think & Feel** (속으로 무슨 생각/느낌):
  - "또 현장 방문해야 하나... 왕복 3시간 낭비인데" (피로)
  - "이번 견적도 오차 나면 클라이언트한테 또 추가 비용 요구해야 하는데 눈치 보인다" (불안)
  - "엑셀로 견적 작성하는데 20시간 걸렸는데 클라이언트가 '다른 업체는 1시간 만에 줬다'고 하면 어쩌지" (좌절)
  - "동료 사장들은 어떻게 빠르게 견적 내는 거지? 나만 느린 건가" (자괴감)

- **See** (주변에서 뭘 보는가):
  - 네이버 카페 "인테리어 시공 창업"에서 "견적 빠르게 내는 법" 질문 주 50건 이상 → 다들 같은 고민
  - 아정당 인테리어 콘텐츠 "업체 선택 기준" 읽어봤지만 실제 견적 자동화 기능 없음
  - 인드픽 업체 비교 플랫폼 봤지만 수동 필터만 → "이것도 결국 내가 다 입력해야 하네"
  - 유튜브 "인테리어 시공 유튜버" 댓글에서 "현장 방문 필수" 언급 다수 → "나만 그런 게 아니구나"
  - 경쟁사 대형 업체는 자체 견적 시스템 보유 → "우리도 저런 거 있으면 좋겠는데 개발비 수천만 원..."

- **Hear** (동료/상사/고객/가족한테 뭘 듣는가):
  - 클라이언트: "다른 업체는 1시간 만에 견적 줬는데 왜 이렇게 오래 걸려요?" (압박)

## [7] TAM / SAM / SOM

- **TAM** (전체 시장): **15조 원** 
  - 계산식: 인테리어 시공 시장 전체 (한국 주택·상가 리모델링 시장 연 15조, 2023년 기준)

    - 언어 제약: 한국어만 지원 → 외국인 운영 업체 제외 (5% 감소)
  - **재계산**: 3조 × (1 - 0.1 - 0.3 - 0.05) = 3조 × 0.55 = **1.65조** (보수적 추정)

- **SOM** (초기 확보): **100억 원** (하은 반론 반영 — 민수 5,000억 → 보수적 재추정)

    - 전국 중소 인테리어 업체 5,000곳 (세그먼트 ① 규모) × 도입률 2% = 100곳

    - **3년 후 목표**: 도입률 2% → 10% 확대 (500곳) → 연 29.4억 → SOM 100억 달성 가능
  - **데이터 부족 시 추정 근거**:

**검증 필요**:

- **MVP 출시 시점 판정**: **PSF (Problem-Solution Fit) 단계**
  - **근거**: 
    1. Pain #10 Level 5 (workaround 비용 월 110만) 검증 완료 — 서윤 50개 Pain Point 중 evidence_quote 명확

## [9] Flywheel + Engine of Growth

### Flywheel 초안 (6개 요소)

```
1. 네이버 카페 광고 → 2. 랜딩 페이지 유입 (견적 산출 데모) → 3. 무료 체험 가입 (30일)
     ↑                                                                    ↓
6. 바이럴 효과 (카페 후기) ← 5. NPS 40+ 달성 (추천 의사) ← 4. 정확도 체감 (아하 모먼트)
```

**각 연결고리 작동 원리**:
- **1→2**: 네이버 카페 "인테리어 시공 창업" 광고 (월 500만) → "현장 방문 없이 1시간 내 견적 산출" 카피 클릭 → 랜딩 페이지 유입 300명 (클릭률 10% 가정)
- **2→3**: 랜딩 페이지에서 "무료 체험 30일 + 정확도 보증" CTA 클릭 → 회원가입 (이메일/휴대폰 1분) → 현장개요 입력 폼 접근 (전환율 30% = 90명 가입)
- **3→4**: 현장개요 입력 (5분) → 표준예가 견적서 미리보기 (PDF 1페이지) → 정확도 체감 (오차율 15% 이내) → "이거 진짜 되네!" 아하 모먼트 (체감률 70% = 63명)
- **4→5**: 아하 모먼트 후 견적서 전체 다운로드 (유료 전환) → 3개월 사용 후 "동료 사장에게 추천하겠는가?" 설문 → NPS 40+ 달성 (추천 의사 40% = 25명)
- **5→6**: NPS 40+ 사용자가 네이버 카페 "인테리어 시공 창업"에 후기 작성 ("이 툴 쓰니 수주 2배") → 바이럴 효과 (후기 1건당 신규 유입 10명 추정 = 250명)
- **6→1**: 바이럴 유입 250명 → 광고비 절감 (CAC 45만 → 0원) + 신뢰도 상승 (후기 기반 전환율 3% → 5%) → 1번 광고 효율 향상 (선순환 완성)

### Engine of Growth 선택

**선택**: **Sticky (리텐션 기반) + Viral (바이럴 기반) 하이브리드**

**근거**:
1. **Sticky 엔진 우선**: 
   - Pain #10 Level 5 (현장 방문 필수) 고통 강도 높음 → 한 번 사용하면 "엑셀로 돌아갈 수 없음" (Switching Cost 높음)
   - 정확도 ±10% 이내 달성 시 → 클라이언트 신뢰 확보 → 재계약률 60% → 사용할수록 데이터 축적 (10,000건 견적) → 알고리즘 개선 선순환
   - 월 Churn 3% 목표 (평균 유지 36개월) → LTV 1,764만 확보 → CAC 45만 대비 39배 회수 가능
2. **Viral 엔진 보조**: 
   - 네이버 카페
