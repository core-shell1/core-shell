## [6] Pre-mortem (TOP 5 각각, 5가지 실패 시나리오)

### A. 견적 자동화
- **가치 실패**: "자동 견적은 편리하지만 커스터마이징 불가능해서 실제 프로젝트엔 못 쓴다" → 원인: 표준예가 DB가 특수 공간 (예: 곡선 벽, 고급 자재) 미지원 → 대응: MVP는 "일반 주거용 20~30평"만 타겟팅, 특수 케이스는 2단계 추가
- **전환 실패**: "엑셀 견적에 익숙해서 새 시스템 배우기 귀찮다" → 원인: Switching Cost (기존 엑셀 템플릿 10년 사용) → 대응: "엑셀 업로드 → 자동 변환" 기능 추가, 온보딩 1분 이내
- **도달 실패**: "네이버 카페에서 광고해도 반응 없다" → 원인: 타겟 채널 불일치 (카페 회원 = 정보 공유자, 실제 구매자 아님) → 대응: 집닥 리뷰 섹션에 "엑셀 없이 1분 견적" 댓글 마케팅 + 유튜브 인테리어 시공 채널 협찬
- **단위경제 실패**: CAC 50만원 > LTV 49만원 (첫 달 구독만 하고 이탈) → 원인: 견적 자동화만으로는 Retention 약함 → 대응: 법률·거래망 모듈 번들링 (집중 패키지 49만원 → 89만원 Up-sell)
- **타이밍 실패**: "표준예가 DB 구축에 6개월 걸려서 경쟁사가 먼저 출시" → 원인: 공공 데이터 정제 지연 → 대응: MVP는 "서울·경기 상가 인테리어"만 지원 (전국 확대는 2단계)

### B. 법률 리스크 관리
- **가치 실패**: "리스크 체크는 좋은데 실제 분쟁 발생 안 하면 돈 아깝다" → 원인: "예방" 가치는 즉시 체감 어려움 → 대응: "분쟁 1건 방지 = 1,200만원 절감" 사례 강조 + 첫 3개월 무료 체험 (분쟁 발생 시 즉시 가치 체감)
- **전환 실패**: "변호사 상담이 더 안심된다" → 원인: AI 리스크 체크 신뢰도 부족 → 대응: 법무법인 제휴 인증 마크 + "AI 체크 → 변호사 검토" 2단계 서비스
- **도달 실패**: "분쟁 경험자만 타겟팅하면 시장 좁다" → 원인: TAM 과소평가 → 대응: "예방" 아닌 "계약서 자동 생성" 가치로 전환 (모든 업체 필요)
- **단위경제 실패**: 법률 DB 최신화 비용 월 100만원 > ARPU 19만원 → 원인: 법 개정 시 즉시 반영 필요 → 대응: 법무법인 제휴로 DB 유지 비용 외주화
- **타이밍 실패**: "정부 표준 계약서 보급으로 차별화 약화" → 원인: 정책 변화 → 대응: 표준 계약서 + "맞춤 조항 자동 삽입" 기능으로 차별화

2. **경쟁 빈틈 — 역기획 Layer 5**: 집닥은 "업체가 수작업 엑셀 업로드" (Layer 5 불만 #1) + 아파트멘터리는 B2C만 지원 → **B2B 견적 자동화는 완전 블루오션** + Pain #1, #2, #18, #26 모두 연결
3. **OKR 정합성 + 리안 자산**: KR2 "견적 시간 60% 단축" 직접 기여 + 리안 컴퍼니 기존 SaaS 인프라 활용 (표준예가 DB는 공공 데이터) + 온라인팀 소상공인 채널로 초기 고객 유입 (네이버 카페 5만 명)

**민수와 일치/불일치**:
- **일치** (민수도 A. 견적 자동화 선택, 종합 점수 79점)
- **추가 근거 강화**: 민수는 "월 200만원 절감"만 강조했으나, 실제로는 **Pain #2 (자재 가격 변동 재계산 월 150만원) + Pain #18 (표준예가 업데이트 수동 월 120만원)까지 포함하면 월 470만원 절감 가능** → 가격 49만원 대비 ROI 10배 (첫 달 회수)

## [8] 48시간 검증 실험

### 1. 검증 방법
**선택**: 랜딩 + 사전결제 (코딩 없이 수요 검증)

**이유**:
- 인터뷰는 "Mom Test" 적용 가능하나 B2B는 의사결정 시간 길어서 48시간 내 결론 어려움
- 피지컬 데모는 표준예가 DB 구축 필요 (시간 부족)
- **랜딩 + 사전결제**는 "진짜 돈 내는 고객" 식별 가능 + 48시간 내 전환율 측정 가능

**구체 방법**:
1. 노션 랜딩페이지 제작 (3시간)
   - 헤드라인: "엑셀 견적 10시간 → AI 1분, 월 200만원 절감 보장"
   - 데모 영상: "현장정보 입력 → 표준예가 자동 계산 → PDF 다운로드" 시뮬레이션 (파워포인트 애니메이션)
   - CTA: "베타 테스터 50명 한정, 첫 달 무료 (정가 49만원) → 지금 신청"
2. 네이버 카페 "인테리어 시공 전문가 모임" 게시글 (1시간)
   - 제목: "[무료 체험] 엑셀 견적 10시간 → 1분, 월 200만원 절감 AI 툴 베타 테스터 모집"
   - 본문: Pain #1 evidence_quote 인용 + 데모 영상 링크 + 노션 랜딩 링크
3. 카카오톡 오픈채팅 "인테리어 견적 공유방" 홍보 (30분)
4. 사전결제 설정: 토스페이먼츠 "정기결제 등록" (첫 달 무료, 2개월째 자동 49만원 결제) → **진짜 구매 의사 확인**

### 2. 성공 KPI
**임계값**: 48시간 내 **사전결제 등록 10명 이상**

**산출 근거**:
- 네이버 카페 회원 5만 명 × 게시글 도달률 10% = 5,000명 노출
- 랜딩 클릭률 5% = 250명 방문
- 사전결제 전환율 4% = 10명
- **10명 × 49만원 × 12개월 = 연 5,880만원** (초기 검증 기준 충분)

**실패 시 피벗**: TOP 5의 2순위 **C. 거래망 네트워크**로 전환
- 이유: 종합 점수 72점 (A와 근접) + 블루오션 + 수수료 수익 이중화 가능
- 검증 방법: "협력사 추천 무료 체험 → 거래 성사 시 수수료 3%" 랜딩 테스트
