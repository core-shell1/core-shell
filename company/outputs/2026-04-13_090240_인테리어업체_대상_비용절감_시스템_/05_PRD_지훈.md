# PRD: 인테리어 시공업체 대상 AI 견적 자동화 SaaS

## 제품 개요

**직원 5인 이하 인테리어 시공업체 대표가 현장정보 입력만으로 표준예가 견적서·자재 상세내역서를 1분 내 자동 산출하여 월 470만원 인건비 절감(기존 10시간 → 1분)을 달성하는 B2B SaaS**

---

## 기술 스택

- **FE**: React 18 + TypeScript + TailwindCSS (반응형 웹앱, 모바일 대응)
- **BE**: Node.js (Express) + TypeScript + PostgreSQL
- **DB**: PostgreSQL (관계형, 견적 데이터 ACID 보장 필수) + Redis (캐시, 실시간 가격 업데이트)
- **인프라**: AWS EC2 (t3.medium) + RDS (PostgreSQL) + S3 (PDF 저장) + CloudFront (CDN) / 또는 Naver Cloud Platform (리안 기존 계약)
- **외부 API**: 
  - 자재 가격 API (공급업체 제휴 또는 공공 DB: 대한건설협회 표준시장단가)
  - PDF 생성: PDFKit (Node.js 라이브러리)
  - 인증: Firebase Auth 또는 자체 JWT

## Must NOT (범위 외)

- **법률 자문 제공**: "법률 리스크 체크"는 참고용 정보만 제공. 변호사법 위반 방지. 약관에 명시: "본 서비스는 법적 조언이 아니며, 실제 계약 전 변호사 검토 필수"
- **자재 공급 또는 중개**: 자재 가격 정보만 제공. 직접 판매/구매 중개 불가 (공급업체 이해관계 충돌)
- **시공 품질 보증**: 견적서 정확도만 보장. 시공 결과물 품질은 시공업체 책임
- **3D 렌더링 (v1)**: 2D 평면도만 지원. 3D는 v2 이후 (개발 난이도 높음)
- **국제화**: 한국어만 지원. 해외 진출 없음
- **AI 학습용 데이터 판매**: 사용자 견적 데이터는 절대 제3자에 판매 금지 (신뢰도 필수)

## User Flow

### 시나리오 1: 신규 사용자 - 첫 견적 생성 (무료 체험)

1단계: **회원가입** → 이메일 입력 → 비밀번호 설정 → 회사명/연락처 입력 → 가입 완료 (2분)

2단계: **현장정보 입력** → 평수 선택(10~50평) → 위치 선택(시/도) → 공간타입 선택(거실/침실/주방/욕실 복수선택) → 공정 선택(신축/리모델링) → "견적 생성" 클릭 (1분)

3단계: **표준예가 자동 계산** → 시스템이 국토부 표준품셈 DB 조회 → 공정별 단가 자동 계산 → 합계 산출 (시스템 처리 5초)

4단계: **자재 상세내역서 자동 생성** → 공정별 필요 자재 추출 → 자재 DB 조회 → 단가 × 수량 계산 → 테이블 생성 (시스템 처리 5초)

5단계: **견적서 PDF 생성** → 위 데이터를 템플릿에 자동 삽입 → PDF 다운로드 (시스템 처리 3초)

6단계: **고객에게 제출** → PDF를 이메일/카톡으로 전송 → 고객 반응 대기

**총 소요 시간**: 3분 (기존 10시간 대비 95% 단축)

**심리 트리거**: 
- "와, 1분 만에 견적이 나왔다" → 즉시 가치 체감
- PDF 워터마크 "리안 SaaS 무료 버전" 표시 → "2건째부터는 유료 필요" 인식 → Paywall 트리거

---

### 시나리오 2: 기존 사용자 - 유료 구독 후 반복 사용

1단계: **로그인** → 대시보드 진입 (이전 견적 히스토리 표시)

2단계: **견적 히스토리 조회** → "지난달 A프로젝트 견적 복사" 클릭 → 기존 데이터 자동 로드 (신규 입력 시간 50% 단축)

3단계: **현장정보 수정** → 평수만 변경(25평 → 30평) → "견적 재생성" 클릭 (30초)

4단계: **자재 가격 실시간 반영** → 시스템이 최신 가격 자동 적용 → "가격 변동 10% 이상 시 알림" 설정 활성화

5단계: **사업성 평가서 확인** → 예상 원가 + 마진율 + ROI 자동 계산 → 투자 결정 (2분)

6단계: **PDF 다운로드 및 고객 제출** → 워터마크 없음 (유료 구독자) → 전문성 인상

**총 소요 시간**: 3분 (기존 10시간 대비 95% 단축)

**Sticky 엔진 작동**:
- 견적 히스토리 10건 축적 → "이전 데이터 복사" 기능 활용 → 이탈 시 데이터 손실 리스크 → Churn 감소

---

### 시나리오 3: 자재 가격 변동 대응 (P1 기능)

1단계: **자재 가격 변동 알림** → 푸시 알림 "자재 A 가격 15% 인상" 수신

2단계: **기존 견적 자동 업데이트** → 사용자가 "자동 업데이트" 클릭 → 새 가격 적용 (10초)

3단계: **고객에게 가격 변동 공지** → "자재비 인상으로 견적 재조정" 메시지 + 새 PDF 전송

**기존 Workaround**: 공급업체 전화 → 엑셀 수동 재계산 → 고객 재통보 (월 15시간 소요)

**우리 솔루션**: 자동 알림 + 자동 업데이트 + 자동 PDF 생성 (월 15시간 절감)

## 리스크

### 1. 가치 실패 (하은 반론 기반)

**리스크**: "월 200만원 절감"이 현실적이지 않을 수 있음. 실제 사용자가 체감하는 절감액이 50만원 미만일 가능성.

**근거**: 
- Pain #1 "표준예가 견적서 수작업" 월 200만원은 "주당 10시간 × 시급 5만원" 추정치. 실제로는 시급이 더 낮거나, 견적 작업이 병렬 처리되어 실제 손실이 적을 수 있음.
- Pain #2 "자재 가격 재계산" 월 150만원도 "월 15시간 × 시급 10만원" 추정. 실제로는 자재 가격 변동이 월 2~3회만 발생할 수 있음.

**대응 방안**:
1. **48시간 Pre-sell 검증 필수** (조건 2): 실제 고객 5명 이상에게 "월 200만원 절감 보장" 메시지로 사전결제 유도. 5건 이상 성공 시 가치 검증 완료.
2. **초기 고객 인터뷰** (v1 출시 후 1주): 유료 구독 고객 5명 대상 "실제 절감액 측정" 인터뷰. 평균 절감액 < 50만원 시 가격 인하 (49만원 → 29만원) 또는 기능 추가 (자재 가격 실시간 연동 P1 → P0 승격).
3. **ROI 계산기 제공**: 랜딩 페이지에 "월 견적 건수 × 시간 × 시급" 입력 → 자동 절감액 계산 기능. 사용자가 직접 계산하도록 유도.

---

### 2. 전환 실패 (하은 반론 기반)

**리스크**: B2B SaaS 전환율이 낮음. 무료 체험 후 유료 전환율이 40% 미만일 가능성.

**근거**:
- B2B SaaS 평균 전환율은 3~5% (일반적 벤치마크). 우리가 40% 목표는 과도할 수 있음.
- 엑셀 의존도 높은 고객일수록 "새 시스템 배우기 귀찮다" 심리 강함 (Switching Cost).

**대응 방안**:
1. **온보딩 최소화** (v1): 회원가입 → 현장정보 입력 → 견적 생성까지 **3분 이내**. 추가 설정/학습 없음.
2. **Paywall 트리거 최적화** (v1): 무료 1건 생성 후 2건째 시도 시 "주목 패키지 29만원 구독 또는 건당 5만원 결제" 선택지 제시. 선택지 제공으로 전환율 향상.
3. **무료 기간 연장** (v1.1): 초기 전환율 < 20% 시 무료 기간 1개월 → 2개월로 연장. 사용자가 더 많은 견적을 생성하도록 유도.
4. **번들 할인** (v1.1): "견적 + 자재 + 사업성" 통합 패키지 (집중 49만원)를 강조. 단일 기능 구독보다 가치 체감 높음.
