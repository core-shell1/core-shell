## [1] Problem Statement

**"중소 부동산 개발업체 대표(직원 5~30명, 연 5~15건 토지 검토)는 토지 매입 의사결정 시점에서 인허가 승인 여부를 사전에 알 수 없어 매입 후 거절 시 평균 5억 원 + 6개월 기회비용(총 7~10억)을 손실하는 문제를 겪고 있다. 이는 지자체 인허가 기준이 비공개·비정형이고 과거 승인/거절 패턴 데이터가 산재되어 있기 때문이며, 현재 외부 컨설턴트 고용(월 500만원) 또는 지자체 문의 대기(2주~1개월)로 버티고 있어 의사결정 지연 + 높은 비용 + 여전히 남는 불확실성이라는 손실이 발생한다."**

**근거**:
- Pain #2 evidence_quote: "사업 타당성 개념에 대해 고찰... 재무적 타당성 분석"[5] — 인허가 거절 시 사업 전체 무산, 매입 비용 회수 불가
- Pain #21 evidence_quote: "물건의 종류, 규모, 위치, 물건 가격의 적정성, 토지가격의 적정성"[5] — 분양성 판단 시 인허가 가능성이 선행 조건이나 현재 수작업 검토
- Pain #29 evidence_quote: "토지이용 변화 예측"[3] — 용도지역 판독 어려움으로 인허가 가능성 사전 파악 불가

## [3] Value Proposition Canvas

### 고객 프로필 (오른쪽)

**Customer Jobs** (3가지 층)
- **기능적 Jobs**: 
  - 토지 매입 전 인허가 승인 가능성을 72시간 내 파악하여 Go/No-Go 결정
  - 지자체별 인허가 기준(용도지역, 특례법, 건폐율 등) 자동 확인
  - 과거 유사 토지의 승인/거절 사례 3건 이상 비교 분석
- **감정적 Jobs**: 
  - "이번엔 거절 안 당할 거야"라는 확신 얻기 (과거 5억 손실 트라우마 해소)
  - "컨설턴트 없이도 내가 판단할 수 있다"는 자신감 회복
  - "빠른 의사결정으로 경쟁사보다 먼저 좋은 토지 선점"하는 성취감
- **사회적 Jobs**: 
  - 임원진/투자자에게 "데이터 기반 의사결정"으로 전문성 인정받기
  - 업계 네트워크에서 "최신 프롭테크 활용하는 혁신적 대표"로 인식되기

**Pains** (반드시 서윤 Pain # 인용)
- **P1**: (Pain #2) 인허가 승인 확률을 사전에 알 수 없어 매입 후 거절 시 평균 5억 + 기회비용 6개월 = 총 7~10억 손실
- **P2**: (Pain #2) 외부 컨설턴트 고용 시 월 500만원 비용 + 2주~1개월 대기 시간 → 의사결정 지연으로 좋은 토지 경쟁사에 선점당함
- **P3**: (Pain #29) 용도지역·특례법 판독을 위해 지자체 문의 시 담당자마다 답변 상이 + 평균 2주 대기 → 불확실성 지속
- **P4**: (Pain #21) 분양성 판단 시 토지가격 적정성 검토는 하지만, 인허가 가능성은 "감"에 의존 → 객관적 근거 부족
- **P5**: (Pain #3) 특례법 해당 여부 확인을 위해 변호사 리뷰 의뢰 시 월 300만원 + 법률 용어 해석 어려움 → 비전문가는 이해 불가

**Gains**
- **G1**: 토지 매입 의사결정 시간 2주~1개월 → 48시간으로 단축 → 좋은 토지 선점 확률 3배
- **G2**: 인허가 거절 위험 사전 제거 → 5억 손실 방지 → 연간 1~2건만 막아도 ROI 100배
- **G3**: 컨설턴트 비용 월 500만원 → 69만원 = 86% 절감 → 연간 5,172만원 절약
- **G4**: 과거 유사 사례 3건 자동 제시 → "우리 토지도 승인될 거야"라는 확신 + 임원진 설득 자료 확보
- **G5**: 지자체 담당자 문의 없이 자동 확인 → 담당자 눈치 안 보고 독립적 판단 가능

### 가치 맵 (왼쪽)

**Products & Services**
- 개요서·법률서 PDF 업로드 → 자동 텍스트 추출 (OCR)
- 지자체별 인허가 기준 DB (용도지역, 특례법, 건폐율, 용적률 등) 자동 매칭
- 과거 5년간 서울/경기 지역 인허가 승인/거절 패턴 300건 학습 ML 모델
- 승인 확률 % (예: 78%) + 신뢰도 ★★★★☆ + 유사 사례 3건 자동 제시
- 48시간 내 리포트 생성 (PDF 다운로드 + 이메일 발송)

**Pain Relievers** (각 Pain에 1:1 대응)
- **P1 → 승인 확률 75% 이상 자동 산출**: ML 모델이 과거 300건 학습 → "이 토지는 승인 확률 78%"라고 명시 → 대표가 "78%면 Go" 즉시 판단 가능. 거절 위험 사전 제거 → 5억 손실 방지
- **P2 → 48시간 내 리포트 생성**: 컨설턴트 2주~1개월 대기 → 우리는 업로드 후 48시간 내 자동 산출 → 의사결정 속도 3배. 월 500만원 → 69만원 = 86% 비용 절감
- **P3 → 지자체 기준 자동 매칭**: 지자체 문의 없이 우리 DB에서 용도지역·특례법 자동 확인 → 담당자 답변 상이 문제 해결. 2주 대기 → 즉시 확인
- **P4 → 유사 사례 3건 제시**: 과거 승인/거절된 유사 토지 3건 자동 비교 → "우리 토지는 승인된 A사례와 90% 유사" 객관적 근거 제공 → 임원진 설득 자료
- **P5 → 특례법 자동 판독 (MVP 범위 밖)**: 초기 MVP는 인허가 승인 확률 예측만 제공. 특례법 자동 판독은 Phase 2 (6개월 후) 추가 예정. 현재는 변호사 리뷰 병행 권장

**Gain Creators**
- **G1 → 48시간 리포트**: 업로드 후 48시간 내 자동 산출 → 2주~1개월 → 48시간 = 의사결정 속도 3배
- **G2 → 승인 확률 75% 보장**: 정확도 75% 이상 보장 (실제 승인/거절 결과 대조) → 거절 위험 사전 제거 → 5억 손실 방지
- **G3 → 월 69만원 구독**: 컨설턴트 월 500만원 → 69만원 = 86% 절감 → 연간 5,172만원 절약
- **G4 → 유사 사례 3건**: 과거 승인/거절 사례 자동 제시 → 임원진 설득 자료 + "우리도 승인될 거야" 확신
- **G5 → 독립적 판단**: 지자체 담당자 문의 없이 자동 확인 → 담당자 눈치 안 보고 독립적 의사결정

## [4] 공감 지도 (Empathy Map)

**페르소나**: 김철수 (45세, 수도권 중소 시행사 대표, 직원 20명, 연 12건 토지 검토, 과거 인허가 거절로 5억 손실 경험 1회)

- **Think & Feel**: 
  - "이번 토지도 거절당하면 회사 망한다... 컨설턴트 믿어도 되나?"
  - "500만원씩 매달 나가는데 정확도는 70%라니... 30%는 내 운인가?"
  - "경쟁사는 벌써 계약했다는데 우리는 아직도 검토 중... 이러다 놓치겠네"
  - "임원진한테 '데이터 기반 의사결정'이라고 보고하고 싶은데 근거가 약해"

- **See**: 
  - 경쟁사가 같은 지역 토지를 먼저 계약했다는 뉴스 (네이버 부동산 뉴스)
  - 제너레잇 Z-maps 광고 ("지도 기반 프로젝트 분석") — 클릭했으나 "승인 확률은 안 나오네?"
  - 부동산 스터디 카페에서 "○○ 지역 인허가 거절당했다"는 후기 글 (불안 증폭)
  - 컨설턴트 보고서 — "승인 가능성 높음"이라고만 써있고 % 없음 (신뢰 안 감)

- **Hear**: 
  - 재무팀장: "대표님, 이번 토지 매입하면 현금흐름 6개월 타이트합니다"
  - 아내: "여보, 회사 괜찮아? 요즘 스트레스 많아 보여"
  - 동료 대표 (골프 모임): "나도 작년에 인허가 거절당해서 3억 날렸어... 조심해"
  - 지자체 담당자: "이 토지요? 글쎄요... 특례법 해당 여부는 법무팀에 문의하셔야..."

- **Say & Do**: 
  - 컨설턴트에게 "이번 토지 승인 확률 몇 %예요?" 물어봄 → "높습니다"라는 애매한 답변만 들음
  - 지자체 담당자에게 전화 → 2주 후 회신 약속 → 실제로는 3주 걸림
  - 엑셀에 과거 토지 검토 이력 정리 (승인/거절 여부, 소요 기간) → 패턴 찾으려 하지만 데이터 부족
  - 임원 회의에서 "이번 토지는 컨설턴트가 승인 가능성 높다고 했습니다" 보고 → 임원들 "근거는요?" 질문에 답변 곤란

- **Pain**: 
  - **가장 큰 두려움**: "이번에도 거절당하면 5억 날리고 회사 문 닫아야 할 수도..."
  - **좌절**: "컨설턴트한테 500만원씩 주는데 왜 정확한 답을 못 주지? 내가 뭘 더 해야 하나?"

- **Gain**: 
  - **"이것만 해결되면" 이상적 상태**: 
    - "토지 개요서만 업로드하면 48시간 내 '승인 확률 78%' 딱 나오고, 유사 사례 3건 보여주면... 나도 바로 결정할 수 있을 텐데"
    - "컨설턴트 500만원 안 쓰고 월 69만원으로 해결되면 연간 5천만원 아끼는 거잖아. 그 돈으로 토지 한 건 더 사지"
    - "임원진한테 '이 토지는 과거 A사례와 90% 유사하고 승인 확률 78%입니다' 보고하면 다들 납득할 거야"

## [9] Flywheel + Engine of Growth

### Flywheel 초안 (6개 요소)

```
1. 사용자가 토지 1건 업로드
     ↓
2. 48시간 내 승인 확률 78% + 유사 사례 3건 리포트 받음
     ↓
3. "정확하네!" 만족 → 2건째 분석 요청 (유료 전환)
     ↓
4. 월 10건 분석 → 과거 데이터 축적 (우리 DB 강화)
     ↓
5. ML 모델 정확도 향상 (75% → 80%) → 더 정확한 예측
     ↓
6. 만족한 고객이 동료 대표에게 추천 ("이거 진짜 좋아")
     ↑
1. 추천받은 신규 사용자가 토지 1건 업로드 (선순환)
```

**각 연결고리 작동 원리 1줄**:
- **1→2**: 사용자가 개요서 PDF 업로드 → 우리 시스템이 OCR + ML 모델 → 48시간 내 리포트 자동 생성 → 이메일 발송
- **2→3**: 리포트에서 "승인 확률 78% + 유사 사례 3건" 확인 → "컨설턴트보다 빠르고 정확하네!" 만족 → 2건째 분석 시도 (유료 전환 트리거)
- **3→4**: 유료 고객이 월 평균 10건 분석 → 우리 DB에 10건 × 고객 50명 = 월 500건 데이터 축적 → ML 모델 학습 데이터 증가
- **4→5**: 과거 데이터 300건 → 800건 → 1,300건 증가 → ML 모델 재학습 → 정확도 75% → 80% 향상 → 더 정확한 예측 제공
- **5→6**: 정확도 80% 달성 → 사용자가 "이거 진짜 정확해!" 만족 → 동료 대표 (골프 모임, 세미나)에게 "너도 써봐" 추천 → 바이럴 계수 1.2
- **6→1**: 추천받은 신규 사용자가 14일 무료 체험 가입 → 토지 1건 업로드 → 선순환 시작

### Engine of Growth 선택

**선택**: **Sticky (리텐션 기반)**

**근거**: 
- B2B SaaS 특성상 사용자가 한 번 익숙해지면 Switching Cost 높음 (엑셀 → SaaS 전환 시 학습 곡선)
- 인허가 승인 확률 예측은 **반복적 니즈** (연 10~20건 토지 검토) → 매월 사용
- 우리 가치 제안 "48시간 내 리포트 + 정확도 75% 보장"은 **대체재 없음** (컨설턴트는 2주 소요 + 비용 높음) → 사용할수록 의존도 증가

**가속 트리거**: 
- **ML 모델 정확도 향상** (75% → 80% → 85%) — 사용자가 많을수록 데이터 축적 → 정확도 향상 → 만족도 증가 → Churn 감소
- 구체적: 고객 50명 × 월 10건 = 월 500건 데이터 축적 → 3개월마다 ML 모델 재학습 → 정확도 +5%p 향상

**해자 (Moat) 가능성**: 
- **데이터 해자** (Data Moat): 과거 인허가 승인/거절 데이터 300건 → 800건 → 1,300건 축적 → 경쟁사는 데이터 없음 → 6개월 격차
- **파트너 해자** (Partner Moat): 법무법인 3곳과 독점 계약 (API 연동) → 경쟁사는 로펌 섭외 어려움 (법률 도메인 진입장벽)
- **네트워크 효과 (약함)**: 사용자가 많을수록 유사 사례 DB 풍부 → 예측 정확도 향상 → 신규 사용자 유입 증가 (단, B2B 특성상 네트워크 효과는 제한적)

**해자 없는 경우 대응**: 
- 만약 경쟁사 (제너레잇 등)가 데이터 + 파트너 빠르게 확보 시 → **실행 속도로 대체**
- 구체적: 우리는 6개월 내 MVP 출시 + 베타 고객 15개사 확보 → 경쟁사는 기획~개발 1년 소요 → 6개월 선점 효과
- 선점 효과로 고객 50개사 확보 → 데이터 500건 축적 → 경쟁사가 따라잡으려면 추가 6개월 필요 → 총 1년 격차 유지
