### 카페 네이버 플레이스 리뷰 늘리는 방법 지도 상위 노출

### 리뷰 늘리는 실전 5가지 방법 (키자드 사례 기반)[1]
- **리뷰 이벤트 열기**: 고객에게 플레이스 리뷰 작성 시 **소정 할인(10% off)**이나 **사은품(음료 1잔 무료)** 제공. 예: "리뷰 쓰면 다음 커피 무료!" 포스터 부착으로 1주일 만에 20개 리뷰 증가 사례[2][3].
- **자연스러운 요청**: 결제 시 QR코드로 바로 리뷰 유도. "리뷰 남겨주시면 감사 인사 드려요" 말 걸기. 매장 내 태블릿 설치로 즉시 작성률 30% UP[2].
- **키워드 최적화 리뷰 작성법**: 리뷰 본문에 **가게 이름 + '맛집', '추천' 키워드** 넣기. 위치 정보(주소) 삽입으로 노출 오류 방지, 검색 상위 노출률 높임[4].
- **혜택 연계 프로모션**: 리뷰 작성자 추첨 **기프트카드(1만 원)** 증정. SNS 공유 시 추가 이벤트로 바이럴 효과, 한 카페에서 50개 리뷰 폭증 사례[5].
- **고객 참여 독려**: 방문 후 문자/SNS로 "리뷰 부탁드려요" 메시지 발송. 이벤트 기간 2주로 제한해 집중 유도[1].

### 지도 상위 노출 팁 (리뷰 수 직결)

### **촬영 구도 실전 팁 (초보자 즉시 적용)**
- **창가 자연광 자리 선점**: 햇빛이 디저트에 직접 들어오게 배치. 그림자 피하고 부드러운 빛으로 감성 연출. 예: 유리창 구조 확인 후 디저트 앞으로 이동 → 빛 몰입 사진 30% 더 좋아보임.[1]
- **아웃포커스 부각**: 스마트폰 인물모드 F값 낮춰 배경 흐리게 (1.1~2배 줌). 피사체 가까이(10cm 이내), 배경 멀리 → 메뉴가 2배 돋보임. 사례: 라면/파스타 사진에서 배경 날아가 식욕 ↑.[1][2]
- **45~60도 높이 구도 (가장 무난)**: 눈높이 각도로 디저트 전체 담기. 높이 있는 케이크는 위에서, 평평한 건 옆에서. 여러 메뉴 함께 찍을 때 정돈된 느낌.[1]
- **공간 구도 믹스**: 정면(벽/선반 수직으로 안정적), 사선(테이블 선 소실점으로 공간감). 클로즈업 소품(컵/꽃) 추가로 디테일 강조 → 인스타 스토리 풍성.[1]
