## 📩 DM #2: 안정 단골형 카페 (월매출 3000-4000만원)
**제목: "사장님, 네이버 리뷰 50개가 '잠자는 자산'인 이유"**

```

✓ 단골 고객 많은데 신규 유입 2년째 정체
✓ 네이버 리뷰 50개인데 월 예약 5건뿐
✓ 인스타 팔로워 800명인데 주문 연결 안 됨

→ 전형적인 **'자산 활용 실패'** 케이스입니다.

[실제 사례: 청담동 '빈스커피']
• 기존: 월매출 3,400만원 (2년 정체)
• 네이버 리뷰 68개, 별점 4.6, 인스타 팔로워 870명
• 문제: 자산은 있는데 '전환 설계' 부재

→ [저희 투입 6주 후]
• 월매출 4,780만원 (+40.6%)

• 인스타 주문 전환율 1.2% → 8.7%

**저희가 바꾼 3가지 (기존 자산 100% 활용):**

1️⃣ **네이버 리뷰 68개 → 예약 유도 장치로 재설계**

[기존] 리뷰 답글: "감사합니다 :)"
[변경] 리뷰 답글 템플릿 3종 운영
- A형: "다음엔 시그니처 ○○ 꼭 드셔보세요! 예약링크→[URL]"
- B형: "재방문 감사 쿠폰 (QR코드 첨부, 7일 한정)"
- C형: "사진 속 메뉴, 인스타 스토리에 올려주시면 음료 1잔 무료"

→ 결과:
• 리뷰 답글 클릭률 0% → 31%
• 재방문 쿠폰 사용률 54% (월 평균 18명)
• 예약 전환: 리뷰 1개당 평균 0.6건 발생

2️⃣ **인스타 팔로워 870명 → 구매 고객으로 전환**

[기존] 게시물: 예쁜 음료 사진 + 감성 캡션
[변경] '구매 동선' 3단계 설계
- 스토리: 매일 오전 9시 "오늘의 추천 2종" (긴급성)
  → 링크: 네이버 예약 또는 전화 주문 선택지
- 릴스: 신메뉴 제조 과정 15초 (ASMR) + 마지막 2초 "프로필 주문"
- 피드: 월 2회 '팔로워 전용 할인' (스토리 캡처 필수)

→ 결과:
• 스토리 링크 클릭률 8% → 23%
• 팔로워 주문 전환율 1.2% → 8.7%
• 월 인스타 발생 매출 52만원 → 310만원

3️⃣ **네이버 플레이스 '예약 장벽' 제거**

[문제 발견] 예약 클릭 후 이탈률 78%
→ 이유: 예약 폼 9개 항목 (이름/연락처/인원/날짜/시간/메뉴/요청사항/약관 동의/결제 방식)

[해결] 필수 항목 4개로 축소 (이름/연락처/날짜/인원)
→ 메뉴는 방문 후 선택 가능하도록 문구 추가
→ 예약 확정 후 카톡 자동 발송 (매장 위치/주차 정보)

→ 결과:
• 예약 이탈률 78% → 34%
• 예약→방문 전환율 61% → 89%
