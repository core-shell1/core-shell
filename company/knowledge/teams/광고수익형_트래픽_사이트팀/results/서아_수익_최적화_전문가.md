## 📈 PHASE 1: 런칭 직후 (0-10K 세션) — AdSense 기본 세팅

### **1-1. 광고 배치 초기 설정 (정책 위반 0% 원칙)**

| 위치 | 광고 형식 | 배치 근거 | 예상 효과 |
|------|----------|----------|-----------|
| **도구 페이지 상단 (QR생성기, 인보이스 등)** | Responsive Display (ATF) | 사용자가 도구 로딩 대기 중 viewability 80%+ | CTR 2.5%, 체류시간 5분+ 활용 |
| **가이드 글 40% 스크롤 지점** | In-Feed Native | "Instagram 콘텐츠 캘린더 만들기" 3단계 직후 삽입 | 몰입도 최고 구간, CTR 3%+ |
| **템플릿 다운로드 직전** | Multiplex (관련 콘텐츠형) | "인보이스 템플릿 15종" 미리보기 후 배치 | 전환 직전 고의도, RPM 40%↑ |
| **사이드바 (데스크탑)** | Vertical Rectangle | 비교형 글(스케줄러 비교 등)에서 지속 노출 | Fill rate 85%, 안정적 수익 |

**절대 금지 배치**:
- 도구 실행 버튼 위 광고 (오클릭 유도 → 정책 위반)
- 다운로드 버튼 바로 옆 광고 (invalid click 위험)
- 모바일 화면 50% 이상 덮는 Anchor 광고 (UX 저해 → CPC 2배↑ 역효과)

### **1-2. AdSense Auto Ads 설정 (ML 최적화 활용)**

```
우선 적용: 
✅ In-page Ads (자동 배치, 3개 제한)
✅ Anchor Ads (모바일만, 페이지 하단)
✅ Vignette Ads (OFF) — 도구 사용 중 전면광고는 이탈률 60%↑

제외:
❌ Overlay Ads (도구 인터페이스 방해)
❌ Side Rails (수동 배치가 더 효율적)
```

**예상 RPM**: $2-5 (한국 트래픽 50% 가정, Tier-1 50%)

## 💰 PHASE 3: 수익화 고도화 (50K+ 세션) — Mediavine 준비 + 스폰서십

### **3-1. Mediavine 전환 준비 체크리스트**

**50K 세션 달성 예상 시점**: 정우의 S급 키워드 10개 TOP3 진입 시 (런칭 후 4-6개월)

**전환 전 필수 작업** (4주 프로세스):
1. **주간 1-2**: Core Web Vitals 최적화
   - LCP 2.5초 이하 (현재 도구 페이지 JS 번들 최적화)
   - CLS 0.1 이하 (광고 레이아웃 시프트 제거)
   - → **RPM 37%↑ 효과** (2024 모바일 데이터)

2. **주간 3**: 콘텐츠 품질 감사
   - 하윤 확인: 300개 글 중 Thin Content (500단어 미만) 제거
   - 중복 콘텐츠 0% (Copyscape 검증)
   - → Mediavine 승인률 95%+

3. **주간 4**: 트래픽 증명 자료 준비
   - GA4 스크린샷 (세션, 국가별 분포, 페이지뷰 TOP20)
   - AdSense/Ezoic 수익 리포트 (월 $2,500+ 증명)
   - → 전용 매니저 배정 확률 80%

**Mediavine 후 예상 수익**:
- **RPM $15-25** (Ezoic $12 대비 1.5배↑)
- **월 수익 $3,000-5,000** (60K 세션 기준, 미국 40% 가정)
