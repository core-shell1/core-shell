#### **도구 #1: 무제한 QR코드 생성기**
```
[검색 근거]
- 키워드: "free qr code generator for small business"
- 월 검색량: 8,900 | KD: 18 | CPC: $1.2
- 경쟁사 약점: 기존 상위 결과 90%가 5개 생성 후 유료 전환 압박, 로고 삽입 유료화

[UX 와이어프레임]
1. 입력부 (상단 30%)
   - URL/텍스트 입력창 (실시간 미리보기)
   - 색상 선택 (브랜드 컬러 6개 프리셋)
   - 로고 업로드 (중앙 배치, 무료)
   
2. 생성부 (중단 40%)
   - [QR코드 생성] 버튼 (클릭 시 0.3초 내 생성)
   - 다운로드: PNG/SVG/PDF 3종
   - 스캔 추적 링크 자동 생성 (Google Analytics 연동)

3. 관련 콘텐츠 (하단 20%)
   - "소상공인 QR 마케팅 7가지 활용법" 링크
   - 클릭률 30% 목표

4. 도구 추천 (최하단 10%)
   - "명함 디자인 생성기" / "인스타그램 해시태그 추출기" 카드

[프로그래매틱 확장]
- 변수: 업종(카페/식당/미용실/병원) × QR 용도(메뉴/예약/리뷰)
- 자동 생성 페이지: 12개
  예: "카페 메뉴판 QR코드 무료 제작" (월 검색 430)

[내부링크 설계]
- 도구 사용 후 → "QR코드 디자인 팁 5가지" 글 (체류 +3분)
- 관련 도구 → "전단지 템플릿 생성기" (세션당 페이지뷰 +1.5)

[광고 배치]
- 위치: QR코드 다운로드 버튼 하단 (사용 완료 후)
- 형식: 300×250 디스플레이 광고 1개
- 예상 CTR: 2.5% (업계 평균 1.8%)

[Core Web Vitals 목표]
- LCP: 1.2초 (SVG 경량화 + CDN)
- FID: 50ms (vanilla JS, 라이브러리 최소화)
- CLS: 0.05 (광고 영역 사전 확보)
```

**UltraProduct 개발 요구사항**:
```javascript
// 기능 명세
- QR 생성: qrcode.js 라이브러리
- 로고 오버레이: Canvas API (중앙 20% 크기)
- 추적 링크: bit.ly API 연동 (클릭 수 대시보드)
- 다운로드: blob + FileSaver.js
- 반응형: 모바일 터치 최적화 (입력창 zoom 방지)
```

#### **도구 #2: 업종별 인보이스 자동 계산기**
```
[검색 근거]
- 키워드: "best free invoice template for freelancers"
- 월 검색량: 6,200 | KD: 22
- 경쟁사 약점: Excel 다운로드만 제공, 브라우저 내 편집 불가, 업종별 항목 커스터마이징 없음

[UX 와이어프레임]
1. 업종 선택 (상단)

   - 선택 시 항목 자동 변경 (예: 디자인 → "시안 수정 횟수" 항목 자동 추가)

2. 입력부 (중단)
   - 품목명/수량/단가 입력 테이블 (행 추가 버튼)
   - 부가세 자동 계산 (10% 토글)
   - 총액 실시간 표시 (큰 폰트, 우측 고정)

3. 생성부 (하단)
   - [PDF 다운로드] / [이메일 발송] 버튼
   - 미리보기: A4 사이즈 실시간 렌더링

4. 관련 콘텐츠
   - "프리랜서 세금 절세 5가지 방법" (클릭률 25% 목표)

[프로그래매틱 확장]
- 변수: 업종 15개 × 통화(KRW/USD/EUR)
- 자동 생성: 45개 페이지
  예: "디자이너 인보이스 템플릿 무료 다운로드" (월 검색 280)

[내부링크]
- 사용 후 → "견적서 작성 가이드" 글
- 추천 도구 → "영수증 생성기"

[광고]
- 위치: PDF 다운로드 완료 후 페이지 전환 시 전면 광고 1회 (Skip 가능)
- 예상 RPM: $8 (고가치 B2B 트래픽)

[Core Web Vitals]
- LCP: 1.5초 (PDF 생성 Web Worker 백그라운드 처리)
- 테이블 렌더링: 가상 스크롤 (100행 이상 시)
```

**개발 요구사항**:
```javascript
// 핵심 기능

- 이메일 발송: EmailJS API (무료 200건/월)

- 업종별 프리셋: JSON 데이터 15개 (동적 로드)

- 결과 페이지 → "인스타그램 광고 $50 운영법" 글
- 추천 도구 → "페이스북 광고 ROI 계산기"

[광고]
- 위치: 그래프 하단 (결과 확인 후)
- 형식: 네이티브 광고 (마케팅 툴 추천)

[Core Web Vitals]
- LCP: 1.8초 (Chart.js lazy load)
- 인터랙션: 입력 시 실시간 계산 (<100ms)
```
