### 📊 **타겟 발굴 프로세스 (리안이 직접 실행한 방법)**

**1단계: 네이버 지도 검색 (30분 소요)**
```
검색어: "양주 미용실", "의정부 카페"
필터: 리뷰 낮은 순 정렬 → 50개 미만만 체크
수집 항목: 업체명, 리뷰 수, 평점, 주소
```

**2단계: 인스타그램 수작업 검색 (40분 소요)**
```
검색어: "#양주미용실", "#의정부카페", "[업체명]"
체크리스트:
☑ 팔로워 500명 미만
☑ 최근 게시물 좋아요 10개 미만
☑ 게시물 20건 미만
☑ 스토리 업데이트 주 1회 미만
```

**3단계: 블로그 검색 (20분 소요)**
```
네이버 블로그 검색: "[업체명] 후기"
체크: 
- 포스팅 10건 미만 = 취약
- 최근 6개월 업데이트 없음 = 고위험
```

**4단계: 점수 계산 (엑셀 자동화)**
```
네이버 리뷰: 50개 이상=10점, 30-49=7점, 10-29=4점, 10 미만=0점
인스타 팔로워: 1000+ =10점, 500-999=7점, 100-499=3점, 100 미만=0점
블로그: 10건 이상=10점, 5-9건=5점, 1-4건=2점, 0건=0점
총점 18점 이상 = 최우선 타겟
```
