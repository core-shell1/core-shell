## 1️⃣ 박탐정 — 양주/의정부 미용실·카페 잠재고객 리스트업

### 타겟 기준 (돈 낼 수 있는 업체 = 월 광고비 50만원 이상)

```
필수 조건 (4가지 모두 충족):
✅ 네이버플레이스 리뷰 100개 이상 (= 월 방문 300명 이상 추정)
✅ 인스타그램 팔로워 500명 이상 + 최근 1개월 게시물 5개 이상
✅ 블로그 운영 중 (직접 포스팅 = 시간 쏟고 있음)
✅ 네이버 광고 집행 흔적 (검색 시 상단 노출 또는 플레이스 광고)

제외 조건:
❌ 대형 프랜차이즈 (본사 통제)
❌ 오픈 6개월 미만 (예산 부족)
❌ 리뷰 평점 3.5 이하 (서비스 문제 우선)
```

### 리스트업 실행 프로세스 (2시간)

**Step 1: 네이버 지도 크롤링** (30분)
```
검색어: "양주 미용실", "의정부 미용실", "양주 카페", "의정부 카페"

Octoparse 설정:
1. 네이버 지도 검색 결과 1-50위
2. 수집 항목: 상호명/주소/전화/리뷰수/평점/블로그 링크
3. 필터: 리뷰 100개 이상만

예상 결과: 미용실 23개, 카페 31개
```

**Step 2: 인스타그램 검색** (40분)
```
각 업체명으로 인스타 검색:
- 계정 있음 → 팔로워/최근 게시물 확인
- 500명 이상 + 월 5개 이상 포스팅 = O
- 미달 = 리스트 제외

도구: Phantombuster (Instagram Profile Scraper)
```

**Step 3: 광고 집행 여부 확인** (30분)
```
네이버 검색:
- "양주 미용실", "의정부 카페" 검색
- 상단 "광고" 표시 업체 체크
- 플레이스 지도 상단 노출 = 광고 가능성 80%

→ 광고 중인 업체 = 마케팅 예산 있음
```

**Step 4: 우선순위 스코어링** (20분)

| 항목 | 배점 | 확인 방법 |
|------|------|-----------|
| 리뷰 200개 이상 | +30점 | 플레이스 |
| 광고 집행 중 | +25점 | 검색 결과 |
| 인스타 팔로워 1000명+ | +20점 | 프로필 |
| 블로그 직접 운영 | +15점 | 포스팅 작성자 확인 |
| 콘텐츠 품질 낮음 (사진 흐림/구도 엉망) | +10점 | 육안 평가 |

**→ 70점 이상만 최종 리스트 (목표 20개)**
