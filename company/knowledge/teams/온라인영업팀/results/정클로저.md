## 💬 2. 김작가: 아웃리치 DM 스크립트 (답장률 35% 목표)

### **미용실용 DM (인스타그램 직접 발송)**

**버전 A: 고통점 직격 (리뷰 많은데 SNS 약한 곳)**
```
안녕하세요 [업체명]님,
네이버 리뷰 [수치]개 보고 찾아왔어요!

그런데 인스타 게시물이 2주에 1번이시네요 😢
요즘 고객 70%가 인스타로 미용실 찾는데,
[경쟁업체명]은 매일 릴스 올려서 예약 2배 늘었대요.

📉 지난달 신규 예약 몇 % 줄었나요?
→ 1주일 안에 예약 30% 올린 사례 공유할게요 (무료).

답장 주시면 [업체명]님 인스타 15초 진단 먼저 드릴게요!
```
**승부 포인트**: 수치(리뷰수) + 경쟁사 사례 + 무료 미끼 (진단)

**버전 B: ROI 강조 (광고비 쓰는 곳)**
```
[업체명]님, 혹시 네이버 광고비 월 [추정치, e.g. 30만원] 쓰시나요?

그 돈으로 인스타 릴스 10개 만들면
→ 예약 전환율 3배 + 광고비 50% 절감 가능해요.

실제 [유사 미용실] 사례:
- 릴스 조회수 5만 → 신규 고객 월 15명 증가
- 네이버 광고 중단 후 매출 그대로 유지

📊 [업체명]님 현재 광고 효율 15초 분석해드릴까요? (무료)
답장만 주세요!
```

### **카페용 DM (인스타그램 직접 발송)**

**버전 A: 매출 손실 수치화 (리뷰 많은데 예약 관리 약한 곳)**
```
[카페명]님, 리뷰 [수치]개 대박이에요 👏

근데 예약 노쇼율 30%면... 
→ 주말 하루 매출 20만원 날아가는 거 아닌가요? 😱

[경쟁 카페]는 카톡 자동 알림으로 노쇼 90% 줄였어요.
+ 인스타 "예약 완료" 스토리로 재방문 2배.

📈 이번 주말 노쇼 0명 만들 방법 보여드릴까요?
답장 주시면 맞춤 솔루션 무료 공유!
```

**버전 B: SNS 미운영 각성 (인스타 없거나 방치된 곳)**
```
[카페명]님, 네이버 리뷰 [수치]개인데
인스타가 없으시네요? 😲

요즘 20-30대 고객 80%가 인스타로 카페 검색해요.
[경쟁 카페명]은 릴스 하나로 주말 매출 50만원 올렸대요.

💡 [카페명]님 메뉴 사진만 있으면
→ 1주일 안에 릴스 3개 만들어드릴게요 (무료 테스트).

관심 있으시면 "ㅇㅇ"만 답장 주세요!
```

**📌 발송 전략**  
- 시간: 저녁 8-9시 (소상공인 마감 후)  
- 순서: 리뷰 많은 곳 → 광고비 쓰는 곳 → SNS 방치 순  
- A/B 테스트: 각 버전 5곳씩 → 48시간 후 답장률 높은 버전 채택  
- 후속: 답장 없으면 48시간 후 "추가 팁 1개 (e.g. 경쟁사 릴스 조회수 비교)" 재발송
