## 📍 PART 1: 강남/명동 샵 온보딩 (지훈)

### 1-1. 영업 대상 선정 (4월 10일까지)

**타겟 프로필**
```
업종: 헤어/네일/피부 중소형 (1-3인)
위치: 강남역·명동역 도보 5분 이내
특징: 네이버 외국인 리뷰 0-2개 (언어 장벽 명확)
현재 상황: 월 매출 500-1,500만 원
```

**10곳 리스트 작성 양식**
| 순번 | 샵명 | 주소 | 업종 | 외국인 리뷰 수 | 전화번호 | 방문 예정일 |
|------|------|------|------|---------------|----------|------------|
| 1 | [샵A] | 강남역 3번 출구 50m | 네일 | 0개 | 010-xxxx | 4/12 14:00 |
| ... | | | | | | |

**리스트 수집 방법**
1. 네이버 지도 "강남역 네일샵" 검색 → 외국인 리뷰 필터링
2. 인스타그램 위치 태그 #강남네일 → DM 가능 계정 추출
3. 카카오맵 "명동 헤어샵" → 전화번호 공개 매장

### 1-2. 첫 방문 피칭 스크립트 (3분 버전)

**오프닝 (30초)**
```
"안녕하세요, Kello 지훈입니다. 
네이버에서 [샵명] 검색해보니 외국인 리뷰가 0개더라고요.
명동은 지금 외국인 손님 90%인데 안타까워서 연락드렸어요."
```

**고통 공감 (1분)**
```
"요즘 외국인 손님 오시면:
- 영어 메뉴판 없어서 설명 어렵고
- 예약 받으려면 DM 영어로 답장 막막하고
- 결제는 카드만 되는데 설명 못 하고
→ 그냥 돌아가시죠?"
```

**솔루션 제시 (1분)**
```
"Kello 앱 하나면:
1. 16개 언어 자동 번역 메뉴 (올리브영 방식)
2. 예약은 외국인이 앱에서 직접 → 샵은 확인만
3. 결제도 앱 안에서 끝 → 샵은 현금 받기만

**5월 무료 파일럿**: 
- 수수료 0원
- 외국인 손님 1명이라도 오면 성공
- 안 오면 그냥 끝 (계약 없음)"
```

**클로징 (30초)**
```
"지금 명동 [경쟁샵명] 이미 시작했어요.
5월 한 달만 써보시고 손님 안 오면 그만두셔도 돼요.
지금 바로 QR 찍고 메뉴 사진 3장만 주시면 
내일부터 외국인한테 노출 시켜드릴게요."
```
