국내 건설(인테리어 포함 실내건축공사업) 하도급 거래 시 **신용평가등급**을 핵심으로 재무·실적·법적 준수 평가. **적격심사세부기준(지방자치단체입찰시낙찰자결정기준 별표5)** 적용: 추정가격 4억~10억 미만 공사 기준[3].

| 평가 항목 | 세부 기준 | 점수/감점 수치 | 출처 |

### **국내 건설업 등록 데이터 공공 API 활용 방법**
**국토교통부 건설업 등록정보 API** (ice.go.kr/ais) 핵심. 무료 공공데이터 포털(data.go.kr) 연동. **등록번호**로 신용·실적 조회 → 하도급 거래 전 자동 평가.

#### **활용 단계 및 API 스펙**
1. **접근**: data.go.kr 가입 → "건설업 등록정보" 검색 → 국토교통부 API 신청(일일 1,000건 제한).
2. **주요 엔드포인트**:
   ```
   GET https://apis.data.go.kr/1230000/constructionEngRegiInfo/constructionEngRegiInfoList
   ?serviceKey={인증키}
   &registNm={업체명 or 등록번호}
   &pageNo=1&numOfRows=10
   ```
   - 응답: JSON/XML (업체명, 등록번호, 업종, 자본금, 대표자, 등록일, 실적, 면허취소 여부).
3. **인테리어 특화 쿼리**: `bizType=실내건축공사업` 필터 → 신용등급 연계(NICE API 콜백).
4. **자동화 프레임워크**:
   - Python 예시:
     ```python
     import requests
     url = "https://apis.data.go.kr/1230000/constructionEngRegiInfo/constructionEngRegiInfoList"
     params = {'serviceKey': 'YOUR_KEY', 'registNo': '등록번호', 'bizType': '실내건축'}
     response = requests.get(url, params=params).json()
     # 신용평가: response['body'][0]['capital'] > 10억 체크
     ```

### 인테리어 공종별 표준 공기 산출 모델 (면적·공종 수·인력 투입 기반 CPM/PERT 경량화 적용)
### 인테리어 공종별 표준 공기 산출 모델 (CPM/PERT 경량화)

**핵심 모델: 면적(A, ㎡) × 공종 단위공기율 + 공종 수(K) × 병목 인력 투입 지연(일) + 비작업일(기상/준비 10-20%) = 총 공기(Y, 일). CPM 경량화: 주공정(Critical Path)만 선별(인테리어 70% 공종 무시), PERT: 낙관/최악/최대확률 평균(β=1/6)으로 ±15% 변동 반영.**[1][2][5][7]

#### 1. 공종별 표준 단위공기율 (㎡당 일수, 층당 보정 × G(지상층수))
인테리어 리모델링 기준(297㎡ 1층 사례: 총 202일, 준비45+정리30 포함).[5][7]
| 공종 | 단위공기율(㎡당 일) | 층당 보정(일) | 공비 비율(%) | 인력 투입(인/㎡) | 사례(100㎡) |
|------|---------------------|---------------|--------------|------------------|-------------|
| 도장 | 0.05-0.07 | 3-5 | 3-4 | 0.02 | 6일 |
| 타일 | 0.08-0.10 | 5-8 | 4-5 | 0.03 | 10일 |
| 도배 | 0.04-0.06 | 2-4 | 2 | 0.01 | 5일 |
| 바닥(마루) | 0.06-0.08 | 3-5 | 3 | 0.02 | 7일 |
| 가구 | 0.10-0.12 (㎡ 무관, 고정) | 7-10 | 7-8 | 0.04 | 11일 |
| 조명 | 0.03-0.05 | 3-6 | 3 | 0.01 | 4일 |
| 위생기구 | 0.07-0.09 (㎡ 무관) | 6-8 | 5 | 0.03 | 8일 |
| **합계 (주공정 5종)** | **0.40-0.55** | **30-45** | **27-30** | **0.16** | **51일 + 병목** |[3][5]

- **계산 예: 500㎡, 3층 타일+바닥+가구 주공정**  
  Y = (500×0.09 타일 + 500×0.07 바닥) + (3종×5일 병목) + 15% 비작업 = 88 + 15 + 15 = **118일**.[2][5]

#### 2. CPM/PERT 경량화 프레임워크 (총 공기 30% 단축)
- **CPM 핵심: 주공정만 추출** (인테리어 80% 병렬 가능, 타일→바닥→가구 선형만 계산. Float 무시).[1][8]
  - 입력: 공종별 작업량(A×율) → 네트워크 다이어그램 → 최장 경로(Y).
  - 공식: Y = max(∑ 주공정 i) + K×2(공종 간 지연).[5]
- **PERT 변동 반영**: β=1/6, σ=√[(b-a)^2/6]. 낙관(a)=80%, 최대확률(m)=100%, 최악(b)=130%.[8]
  | 시나리오 | 공기(일, 500㎡) | 확률 |
  |----------|------------------|------|
  | 낙관 | 95 | 20% |
  | 평균 | 118 | 60% |
  | 최악 | 150 | 20% |
  | **PERT 기대값** | **119** | - |

- **인력 투입 보정**: 총인력 0.16인/㎡ 초과 시 +10% 지연. 주52시간(6일/주) 적용, 동절기 층당+1일.[4][8]

#### 3. 실전 적용 사례 (공공 리모델링 297㎡ 1층)
- 기본: 공종별 작업량 133일
