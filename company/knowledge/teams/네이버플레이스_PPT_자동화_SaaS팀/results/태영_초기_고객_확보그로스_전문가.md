## 🎯 채널별 고객 확보 전략 (50명 = 4개 채널 배분)

### **채널 1: 인스타그램 릴스 → 15명 (30%)**

**논리**: 마케팅 대행사 실무자 80%가 인스타 일 1회 이상 사용[3]. 릴스 알고리즘 상 "before/after" 콘텐츠 참여율 5배↑[1]

#### 실행 프레임워크 (4주 사이클)

```
Week 1: 릴스 10개 제작 (A/B 테스트)
  ├─ 타입A (5개): "네이버플레이스→PPT 5분 완성" 실제 화면 녹화 (15초)
  │   └─ Hook(3초): "클라이언트 보고서 2시간→5분?" 
  │   └─ Demo(10초): URL 입력→PPT 다운로드 타임랩스
  │   └─ CTA(2초): "프로필 링크 무료 체험" 
  ├─ 타입B (5개): 고객 인터뷰 형식 (30초)
  │   └─ "OO대행사 김대표님 후기" + 실제 PPT 결과물 슬라이드 3장
  └─ 목표: 타입당 조회 5,000회, 프로필 클릭율 8% (400명)

Week 2-3: 최고 성과 릴스 ₩500K 광고 부스팅
  ├─ 타겟팅: 25-45세, 관심사 "디지털마케팅", "창업", "소상공인"
  ├─ 예산 배분: ₩25K/일 × 20일
  └─ 목표: 리드폼 제출 60명 (CPL ₩8,333)

Week 4: 리드 Nurturing
  ├─ DM 자동응답: "무료 PPT 1건 제작 쿠폰" 즉시 발송
  ├─ 24시간 내 수동 팔로우업: "어떤 업종이신가요?" 질문
  └─ 전환 목표: 60명 리드 → 15명 유료 (25% 전환율)
```

#### 콘텐츠 템플릿 (복사 즉시 사용)

**릴스 스크립트 예시 - 타입A**
```
[0-3초] 화면: 복잡한 엑셀 데이터 스크린샷
자막: "네이버플레이스 리뷰 300개... 보고서 어떻게 만들죠?"
음성: (긴급한 톤) "클라이언트 미팅 2시간 남았는데"

[3-13초] 화면: 
- 우리 SaaS 웹사이트 URL 입력창 (0.5초)
- "분석 중..." 로딩 바 (2초, 2배속)
- 완성된 PPT 자동 재생 (7초)
자막: "URL 하나로 끝" → "평점·리뷰·키워드 자동 분석" → "PPT 5분 완성"

[13-15초] 화면: "지금 무료 체험" 버튼 클릭 애니메이션
자막: "프로필 링크 클릭 👆"
음성: "오늘 하루 1건 무료!"
```
