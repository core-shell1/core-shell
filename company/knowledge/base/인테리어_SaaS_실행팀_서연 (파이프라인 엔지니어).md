### 건설 견적 자동화 알고리즘 설계 (면적 기반 자재 수량 산출 공식 및 오차율 기준)
### **면적 기반 자재 수량 산출 공식**
건설 견적 자동화에서 **면적(Area)**을 핵심 입력으로 자재 수량 산출은 **기본 공식: 수량 = 면적 × 단위당 소비율 × 보정계수**로 설계. AI는 컴퓨터 비전으로 도면 분석 후 적용[3].

#### **주요 자재별 실전 공식 (㎡ 기준, 표준 내역 기반)**
| 자재 유형 | 공식 | 단위 소비율 예시 | 보정계수 적용 사례 |
|-----------|------|------------------|-------------------|
| **콘크리트 타설** | \( Q = A \times 0.15 \sim 0.20 \) (m³/㎡) | 슬래브 0.15m³, 보강 시 +20% | 높이별 ×1.05 (3층 이상), AI 과거 데이터로 자동 조정[1][2] |
| **철근** | \( Q = A \times 80 \sim 120 \) (kg/㎡) | 바닥판 100kg, 기둥 밀집 +30% | 형상계수 ×1.1 (곡선), 딥러닝 패턴 인식으로 중복 제거[3] |
| **방수/도장** | \( Q = A \times 1.05 \sim 1.20 \) (㎡/㎡) | 롤링 1.1배, 모서리 +10% | 불규칙면 ×1.15, 컴퓨터 비전 불일치 39% 포착[3] |
| **타일/석재** | \( Q = A \times 1.10 \sim 1.15 \) (㎡/㎡) | 절단 손실 10%, 벽체 1.12배 | 폐기율 ×1.05, 실시간 시장 변동 연동[3] |
| **배관/전선** | \( Q = A \times L_f \) (m/㎡, L_f=길이 팩터 2~5m) | 일반 3m/㎡, 다층 +20% | 지역 인건비 ×1.0~1.3 (우편번호 기반)[3] |

- **보정계수 프레임워크**: \( C = C_h \times C_s \times C_r \) (높이/계절/지역). AI NLP로 표준 내역+과거 사례 분석, 단가 추천[1][2].
- **자동화 알고리즘 플로우**:
  1. 도면 입력 → CV 추출 (치수/패턴 인식, Togal/Buildxact 사례)[3].
  2. 역사 DB 교차 (공공 단가+유사 공사, 85-90% 정확도)[1][3].
  3. BOQ 생성 (중복 오류 0%, 75% 시간 단축)[1][3].

### 실전 프레임워크: 3단계 파싱 워크플로
1. **입력 파싱**: `ezdxf.readfile('drawing.dxf')` 또는 `fitz.open('plan.pdf')` → entities/drawings 추출[1][4].
2. **면적/공간 calc**: Closed poly 확인 (`is_closed=True`) → `shoelace_formula` 또는 lib 내장 (`area = sum(x*y1 - x1*y)/2`). 공간 클러스터링: DBSCAN (scikit-learn, eps=0.1m).
3. **출력/검증**: JSON `{ "room1": {"area": 25.5m², "bounds": [x1,y1,x2,y2]} }`. 정확도 98% (테스트: 50 도면, 오차 <1%)[4].

**벤치마크 사례**: 100室 평면도 PDF → ezdxf+PyMuPDF 콤보: 총 면적 5000m², 120 공간 2분 처리. FreeCAD 단독 3분 (3D 확장)[1][2].  
**추천**: CAD 중심=ezdxf+FreeCAD, PDF 중심=PyMuPDF+PDFBox. 코드 스타터: `pip install ezdxf PyMuPDF`. 2024 업데이트: VLM/GNN 보강 (Mamba 등)으로 20% 정확 향상[6].

### 사업성평가서 자동 생성 템플릿 구조 (ROI 산출, 공사비 회수 기간 계산 로직)
### 사업성평가서 자동 생성 템플릿 구조

**핵심 구조: 5단계 프레임워크 (기획→분석→재무→위험→결론)로 엑셀/템플릿 기반 자동화. 입력 셀(사업명/투자액/수익률)만 채우면 ROI/회수기간 자동 산출.** 세계 최고 수준(Agentic AI+엑셀 타당성분석) 적용: NotebookLM 시장분석→Claude 재무초안→젠스파크 포맷팅 연계[3][5][8].

#### 1. 템플릿 입력 섹션 (표준화 기반 자동화 토대)[1][2]
- **필수 입력 변수** (노란색 셀, 사용자 입력):
  | 변수 | 예시 수치 | 설명 |
  |------|----------|------|
  | 초기 공사비 (CAPEX) | 10억 원 | 총 투자액[8] |
  | 연간 운영비 (OPEX) | 2억 원 | 유지/인력 비용 |
  | 연간 예상 수익 | 4억 원 | 매출/이익[8] |
  | 할인율 (WACC) | 8% | 자본비용 |
  | 사업 기간 | 10년 | 회수 기간 계산용 |
- **Tips**: 주소/토지 입력만으로 부동산 변수(건폐율/용적률) 자동 끌어옴 (랜드업 사례)[6].

#### 2. ROI 산출 로직 (현금흐름 자동 계산, 엑셀 수식)[8]
**ROI = (순이익 / 투자액) × 100. 고급: NPV/IRR 병행.**
- **엑셀 수식 예시** (현금흐름표 자동 생성):
  ```
  // B10 셀: 연간 순현금흐름 = 수익 - OPEX - 감가상각
  =C5 - C6 - (C4/10)  // C5:수익, C6:OPEX, C4:CAPEX/기간

  // B20: ROI (단순)
  =(SUM(B10:B19)/C4)*100

  // B21: NPV (할인현금흐름)
  =NPV(C7, B10:B19) - C4  // C7:할인율

  // B22: IRR (내부수익률)
  =IRR(B10:B19, C7)
  ```
- **결과 해석 기준** (타당성 판단):
  | 지표 | 우수 | 보통 | 불가 |
  |------|------|------|------|
  | ROI | >20% | 10-20% | <10% |
  | NPV | >0 | ≈0 | <0 |
  | IRR | >WACC+5% | >WACC | <WACC[8] |

#### 3. 공사비 회수 기간 계산 로직 (Payback Period)[8]
**누적현금흐름이 초기투자 회수까지. 정적/동적 2종.**
- **엑셀 수식** (D열: 누적현금흐름):
  ```
  // D10: 누적 = 이전누적 + 당기현금흐름
  =IF(ROW()=10, B10, D9 + B10)

  // E25: 정적 회수기간 (년)
  =MATCH(-C4, D10:D19, 0) + (C4 + D(MATCH(-C4,D10:D19,0))) / B(MATCH(-C4,D10:D19,0)+1)

  // E26: 동적 회수기간 (할인 적용)
  =MATCH(-NPV(C7,B10:B19), OFFSET(D10,0,0,10,1), 0) + (C4 + NPV(...))/...
  ```
- **사례**: 공사비 10억, 연수익 3억 → 회수기간 4.5년 (누적 10억 도달 시점)[8].
- **기준**: <5년=고위험 사업 우수, >7년=재검토.

#### 4. 전체 템플릿 자동 생성 워크플로우 (AI 연계)[3][5]
1. **기획**: 아이디어 입력 → 문제/시장분석 (NotebookLM 인사이트 추출).

### **활용 프레임워크 (실전 코드 예시)**
1. **API 연동 (Python requests)**:
   ```python
   import requests
   url = "https://apis.data.go.kr/1230000/PriceInfoService/getPriceInfoList"  # 나라장터 예시[2]
   params = {'authKey': 'YOUR_KEY', 'itemClsNo': '인테리어_문틀', 'pageNo': 1}
   resp = requests.get(url, params=params).json()
   price = resp['response']['body']['items'][0]['price']  # 실시간 가격 추출
   print(price)  # 출력: 150000 (예시)
   ```
   - 트래픽: 1초 간격 sleep() 적용, 운영 시 활용사례 등록[2].

2. **크롤링 (Selenium, 자재로 대상)**:
   ```python
   from selenium import webdriver
   driver = webdriver.Chrome()
   driver.get('https://zzro.kr/search?q=ABS+도어')  # 검색[1]
   prices = driver.find_elements_by_class('price-class')  # CSS 셀렉터 확인
   for p in prices: print(p.text)  # 실시간: 120,000원 등
   driver.quit()
   ```
   - 주기: 5분 cronjob, robots.txt 준수, 헤더 User-Agent 랜덤화.
