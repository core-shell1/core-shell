## 1. 프로젝트 개요

### 서비스 정의
한국을 방문한 외국인 관광객이 현지 친구처럼 대화하며 여행할 수 있는 AI 기반 맞춤형 여행 가이드 서비스.

### 핵심 가치 제안
> "복잡한 검색 없이, GPS 하나로 지금 내 주변 진짜 로컬 맛집을 바로 만난다."

### 주요 타깃
- **주 타깃:** 30대 여성 외국인 관광객
- **여행 형태:** 자유여행, 1~2인 소그룹
- **특성:** 감성적 경험 중시, SNS 공유 선호, 안전하고 편안한 분위기 중요

## 3. 핵심 기능 요구사항

### [최우선] F-001 GPS 기반 근처 맛집 자동 추천

**설명:** 사용자의 현재 GPS 위치를 기반으로 반경 내 맛집을 AI가 자동으로 추천한다.

**상세 요구사항:**
- 앱 진입 시 GPS 권한 요청 → 허용 시 자동으로 현재 위치 감지
- 위치 감지 완료 즉시 주변 맛집 3곳 자동 표시 (사용자 입력 불필요)
- 추천 기준: 거리(도보 기준) + 현재 영업 여부 + 타깃(30대 여성) 선호도
- 각 추천 결과에 포함할 정보:
  - 식당명 (영문 + 한국어)
  - 거리 (도보 몇 분)
  - 카테고리 및 대표 메뉴
  - 가격대 ($ / $$ / $$$ / $$$$)
  - 영어 메뉴 여부
  - 분위기 태그 (예: cozy, trendy, quiet, good for solo)
  - 별점
  - AI 추천 이유 (1~2문장, 맥락 포함)
- 위치 새로고침 버튼으로 재탐색 가능
- GPS 거부 시 → 동네명 텍스트 직접 입력으로 대체

**입력:** GPS 좌표 (위도/경도) 또는 텍스트 위치  
**출력:** 맛집 추천 카드 3개  
**응답 시간:** 3초 이내

---

### F-002 자연어 취향 입력 추천

**설명:** 사용자가 자유롭게 원하는 것을 말하면 AI가 맥락을 이해하고 추천한다.

**예시 입력:**
- "I want something spicy but not too much"
- "A cozy cafe for just me"
- "Traditional Korean food that foreigners usually love"

**요구사항:**
- 영어 자연어 입력 처리
- 이전 대화 맥락 유지 (세션 내 히스토리 기반)
- 후속 질문 처리: "a bit cheaper", "something closer", "show me more"

---

### F-003 식당 상세 정보

**설명:** 추천 카드 클릭 시 상세 정보 표시.

**포함 정보:**
- 전체 주소 (영문)
- 영업시간 (요일별)
- 영어 메뉴 여부 / 사진 메뉴 여부
- 가격대 설명
- 대표 메뉴 3개 + 설명 (영어)
- Google Maps 연결 링크
- 분위기 상세 태그
- "좋은 점 / 주의할 점" AI 요약

---

### F-004 분위기/상황 필터

**설명:** 추천 결과를 상황에 맞게 필터링한다.

**필터 항목:**
- 인원: Solo / Couple / Small Group
- 분위기: Cozy / Trendy / Traditional / Quiet / Lively
- 식이제한: Vegetarian / Vegan / Halal / Gluten-free
- 가격대: $ / $$ / $$$ / $$$$
- 영어 메뉴 필수 여부

---

### F-005 대화 맥락 유지

**설명:** 같은 여행 세션 내에서 이전 대화를 기억하고 연속 추천이 가능하다.

**요구사항:**
- 세션 ID 기반 대화 히스토리 저장
- "아까 추천해준 곳 말고", "비슷하지만 더 조용한 곳" 처리
- 세션 만료: 24시간

## 4. 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| **응답 속도** | GPS 감지 → 추천 결과 표시: 3초 이내 |
| **모바일 최적화** | 스마트폰 세로 화면 기준 UI 설계 (PWA 지원) |
| **언어** | 서비스 언어: 영어 (추후 일본어/중국어 확장) |
| **가용성** | 99% 이상 (여행 중 중단 시 치명적) |
| **보안** | GPS 데이터 서버 저장 금지 (요청 시 처리 후 폐기) |
| **접근성** | 글자 크기 최소 16px, 고대비 색상 |

## 5. 기술 스택 (현재 기준)

| 영역 | 현재 | 목표 |
|------|------|------|
| 백엔드 | FastAPI (Python) | 유지 + 위치 검색 로직 추가 |
| AI | Claude Haiku | Haiku (일반 추천) + Sonnet (복잡 쿼리) |
| 데이터 | 정적 JSON 20개 | DB 전환 + 100개 이상 |
| 위치 처리 | 텍스트 파라미터 | GPS 좌표 → 반경 검색 (Haversine 공식) |
| 프론트엔드 | 정적 HTML | 모바일 반응형 PWA |
| 대화 상태 | Stateless | 세션 기반 히스토리 |
| 배포 | 로컬 | Fly.io / Render |
